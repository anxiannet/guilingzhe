## 二、标准版卡牌复制数

```text
通用装备：每种4张。
其他非唯一卡牌：每种2张。
具有“唯一”属性的卡牌：每种1张。
```

“其他非唯一卡牌”包括：

```text
游侠。
节点装备。
游侠装备。
一次性物品。
其他未标记“唯一”的卡牌。
```

唯一属性：

```text
“唯一”优先于卡牌类型的基础复制数。
任何具有“唯一”属性的卡牌，每种只放入1张。
节点卡默认具有“唯一”属性，因此每座节点各1张。
所有卡牌必须在卡表中明确登记是否具有“唯一”属性。
```

当前核定：

```text
蜂鸟：通用装备，4张。
蜻蜓无人机：节点装备，非唯一，2张。
驱离枪：游侠装备，非唯一，2张。
破解储存卡：一次性物品，非唯一，2张。
正版储存卡：一次性物品，非唯一，2张。
蜂鸟巢：节点卡，唯一，1张。
维相枢纽：节点卡，唯一，1张。
```

## 三、核心胜利与节点规则

```text
控制同一系统域的3座在线节点，立即获得最高权限并获胜。
每座节点最多有3个部署实体。
同一节点不能同时存在多个玩家控制的部署实体。
无主节点不能存在部署实体或被携带装备。
```

计入节点3个部署位：

```text
游侠。
独立部署的节点装备。
独立部署的通用装备。
```

不占节点部署位：

```text
由游侠携带的游侠装备。
由游侠携带的通用装备。
```

## 四、装备分类

### 节点装备

```text
只能独立部署到节点。
占用1个节点部署位。
不能由游侠携带。
拥有独立ATK与HP。
```

### 游侠装备

```text
不能直接部署到节点。
必须装备给己方游侠。
不占节点部署位。
每个游侠最多携带1件装备。
游侠装备不是独立实体。
其ATK与HP通常分别增加携带者的ATK与HP。
HP为0或卡面明确不提供HP加成的装备除外。
```

### 通用装备

```text
可以独立部署，也可以由游侠携带。
独立部署时占用1个节点部署位，使用自己的ATK与HP。
由游侠携带时不占部署位，其ATK与HP通常分别增加游侠ATK与HP。
```

## 五、携带装备合并结算

```text
每个游侠最多携带1件游侠装备或通用装备。
携带装备随游侠移动、攻击和进入目标节点。
携带装备不单独支付移动或攻击费用。
```

属性公式：

```text
最终ATK = 游侠ATK + 携带装备ATK。
最终HP = 游侠HP + 携带装备HP。
HP为0或卡面明确不增加HP的装备除外。
```

战斗规则：

```text
游侠与携带装备视为1个战斗单位。
只选择游侠作为攻方或守方。
携带装备不单独攻击、不单独造成伤害，也不单独承受普通战斗伤害。
若受到的伤害大于等于最终HP，游侠被消灭，携带装备同时进入公共弃牌区。
```

装备单独处理：

```text
卡牌或节点技能仍可单独选择被携带装备进行返回、销毁、夺取或其他处理。
携带装备离场后，立即重新计算游侠ATK与HP。
若重新计算后，当前伤害大于等于新的最终HP，游侠立即被消灭。
```

## 六、携带与转换

```text
游侠被消灭时，携带装备同时进入公共弃牌区，除非技能另有说明。
游侠返回手牌时，携带装备同时返回其控制者手牌。
若携带装备被效果单独销毁，游侠保留，并立即重新计算ATK与HP。
```

通用装备同节点转换：

```text
独立部署的通用装备可改由同节点内未携带装备的游侠携带。
被游侠携带的通用装备可卸下并独立部署，但节点必须有空余部署位。
通用装备可在同节点内转交给另一名未携带装备的己方游侠。
同节点内转换不支付电荷。
转换不视为打出、部署、移动或重新上场，不触发技能。
转换后立即重新计算相关游侠的ATK与HP。
```

## 七、回合与普通移动

行动阶段可按任意顺序执行：

```text
打出游侠。
打出节点装备。
打出游侠装备。
打出通用装备。
打出一次性物品。
移动己方部署实体。
改变同节点内通用装备的部署方式。
攻击节点。
使用节点技能。
主动结束行动阶段。
```

普通移动：

```text
移动1个己方部署实体，当前支付1电荷。
只能移动到自己控制的另一座未满节点。
游侠携带的装备随游侠移动，不额外付费、不占目标节点部署位。
同一部署实体每回合可以普通移动1次并攻击1次，顺序自由。
```

## 九、驱离枪

```text
编号：G-S03。
卡名：驱离枪 / Displacement Gun。
装备分类：游侠装备。
唯一：否。
复制数：2。
费用：3。
ATK 2 / HP 0。
技能名：定向驱离。
```

技能：

```text
此装备给游侠装备时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

规则边界：

```text
驱离枪不能直接部署到节点。
必须装备给己方未携带装备的游侠。
为携带者提供+2 ATK。
HP为0，因此不增加游侠HP。
没有合法目标时仍可装备，但不执行定向驱离。
同节点内转交或重新调整驱离枪不会重新触发技能。
```
