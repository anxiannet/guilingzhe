## 二、中文名称规范

```text
名称必须简单、直观，符合中文语境。
玩家看到名称后，应能直接判断它是什么，或大致理解它的用途。
名称不应依赖额外解释才能成立。
优先使用常见、明确、可识别的中文词组。
卡牌名称通常控制在2至5个汉字。
```

```text
命名先说明对象是什么，再体现核心功能或结构，最后才考虑企业风格与世界观修饰。
若名称必须经过一段解释才能理解，应重新命名。
```

## 四、装备分类与功能

```text
节点装备：只能独立部署，占用1个节点部署位。
游侠装备：只能由游侠携带，不占节点部署位。
通用装备：可以独立部署，也可以由游侠携带。
每个游侠最多携带1件装备。
```

携带装备：

```text
最终ATK = 游侠ATK + 携带装备ATK。
最终HP = 游侠HP + 携带装备HP。
HP为0或卡面明确不增加HP的装备除外。
```

技能与功能：

```text
技能具有独立名称、触发条件或结算动作。
功能是装备持续提供的基础作用，不设置技能名。
不得为了卡面包装而把基础功能强行命名为技能。
```

## 五、核心节点与移动规则

```text
控制同一系统域的3座在线节点，立即获得最高权限并获胜。
每座节点最多有3个部署实体。
普通移动1个己方部署实体当前支付1电荷。
每个部署实体每回合最多普通移动1次。
刚上场的部署实体本回合不能普通移动。
```
