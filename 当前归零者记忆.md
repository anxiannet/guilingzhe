## 二、核心胜利与节点规则

```text
控制同一系统域的3座在线节点，立即获得最高权限并获胜。
每座节点最多有3个部署实体。
同一节点不能同时存在多个玩家控制的部署实体。
无主节点不能存在部署实体或被携带装备。
```

计入节点3个部署位的对象：

```text
游侠。
独立部署的节点装备。
独立部署的通用装备。
```

不占节点部署位的对象：

```text
由游侠携带的游侠装备。
由游侠携带的通用装备。
```

## 三、装备分类

### 节点装备

```text
只能独立部署到节点。
占用1个节点部署位。
不能由游侠携带。
拥有独立ATK与HP。
```

### 游侠装备

```text
不能直接部署到节点。
必须装备给己方游侠。
不占节点部署位。
每个游侠最多携带1件装备。
游侠装备不是独立实体。
其ATK与HP通常分别增加携带者的ATK与HP。
HP为0或卡面明确不提供HP加成的装备除外。
```

### 通用装备

```text
可以独立部署，也可以由游侠携带。
独立部署时占用1个节点部署位，使用自己的ATK与HP。
由游侠携带时不占部署位，其ATK与HP通常分别增加游侠ATK与HP。
卡面明确不提供某项加成时除外。
```

## 四、携带与转换

```text
每个游侠最多携带1件游侠装备或通用装备。
携带装备随游侠移动、攻击和进入目标节点。
携带装备不单独支付移动或攻击费用。
携带装备的ATK加入游侠ATK。
携带装备的HP加入游侠HP。
HP为0或卡面明确不提供HP加成的装备不增加游侠HP。
游侠被消灭时，携带装备同时进入公共弃牌区，除非技能另有说明。
游侠返回手牌时，携带装备同时返回其控制者手牌。
```

通用装备同节点转换：

```text
独立部署的通用装备可改由同节点内未携带装备的游侠携带。
被游侠携带的通用装备可卸下并独立部署，但节点必须有空余部署位。
通用装备可在同节点内转交给另一名未携带装备的己方游侠。
同节点内转换不支付电荷。
转换不视为打出、部署、移动或重新上场，不触发技能。
转换后立即重新计算相关游侠的ATK与HP。
```

## 五、回合与普通移动

行动阶段可按任意顺序执行：

```text
打出游侠。
打出节点装备。
打出游侠装备。
打出通用装备。
打出一次性物品。
移动己方部署实体。
改变同节点内通用装备的部署方式。
攻击节点。
使用节点技能。
主动结束行动阶段。
```

普通移动：

```text
移动1个己方部署实体，支付与单个实体攻击相同的电荷，当前为1电荷。
只能移动到自己控制的另一座未满节点。
游侠携带的装备随游侠移动，不额外付费、不占目标节点部署位。
同一部署实体每回合可以普通移动1次并攻击1次，顺序自由。
```

## 六、蜂鸟

```text
编号：G-S01。
卡名：蜂鸟 / Hummingbird。
类型：装备。
装备分类：通用装备。
子类型：无人机。
费用：4。
ATK 3 / HP 1。
```

两种状态：

```text
独立部署：占1个节点部署位，作为ATK 3 / HP 1的装备实体。
游侠携带：不占节点部署位，为携带者提供+3 ATK与+1 HP。
```

---

## 七、驱离枪

```text
编号：G-S03。
卡名：驱离枪 / Displacement Gun。
类型：装备。
装备分类：游侠装备。
子类型：武器。
费用：4。
ATK 2 / HP 0。
技能名：强制驱离。
```

技能：

```text
此装备给游侠装备时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

规则边界：

```text
驱离枪不能直接部署到节点。
必须装备给己方未携带装备的游侠。
不占节点部署位。
为携带者提供+2 ATK。
驱离枪HP为0，因此不增加游侠HP。
没有合法目标时仍可装备，但不执行强制驱离。
```
