## 一、项目定位

```text
归零者 / ZERO ACCESS
```

定位：

```text
一款关于系统弃民、越权装备与城市权限夺取的独立战术卡牌项目。
```

核心口号：

```text
从零权限，到根权限。
From Zero Access to Root Access.
```

核心幻想：

```text
系统不给你权限，你就夺取 Root Access。
```

## 四、对战与多人核心规则

```text
双人 / 多人游玩时不使用节点牌库。
双人 / 多人游玩时不从节点卡中随机抽取节点。
权限节点作为场上固定争夺对象存在。
游侠不再拥有“查看节点牌库 / 调整节点牌库 / 重洗节点牌库”的技能。
原本与节点牌库相关的效果，转移到固定权限节点自身的节点技能体系中。
```

使用方式：

```text
2 到 4 人共用游玩：所有玩家共用同一副标准手牌牌库与同一个手牌弃牌区。
对战卡组游玩：每名玩家可以各自使用一副标准手牌牌库与对应手牌弃牌区。
```

统一区位处理：

```text
抽牌：从当前模式对应的手牌牌库抽。
弃牌：放入当前模式对应的手牌弃牌区。
处理 / 清理：置入来源手牌弃牌区。
吹回：回到该游侠的控制者手牌。
放回牌库：放回来源手牌牌库。
隔离：放入隔离区，不改变控制关系；隔离结束按技能返回。
支援游侠：不是实体牌，不进入任何玩家手牌、手牌牌库或手牌弃牌区。
```

禁用旧词：

```text
公共牌库
弃牌堆
节点牌库
你的牌组
你的牌库
你的手牌牌库
```

## 六、权限节点三类技能结构

每个系统域固定 3 座权限节点。

每个系统域内，3 座权限节点分别承担：

```text
侦察类
进攻类
防守类
```

分类说明：

```text
侦察类：查看、筛选、预判手牌或信息。
进攻类：强化接管、推进争夺、提升进攻游侠星级。
防守类：保护己方节点、恢复游侠、阻止或削弱敌方接管。
```

## 八、游侠等级与装备设定

```text
1星游侠：只有匕首类冷兵器和简易工具，几乎无护甲。
2星游侠：拥有手枪、自制枪械或土改武器，弹药和稳定性不足。
3星游侠：拥有残缺或低配制式装备，品质不高，弹药不足。
4星游侠：拥有全套制式装备，头盔、防弹衣齐全，弹药充足，装备精良。
5星游侠：拥有轻型动力机甲或高度集成外骨骼装备，接近小队核心战力。
```

说明：

```text
游侠星级代表当前装备状态，不作为游侠中文代号命名依据。
游侠插画必须同时参考技能、人物气质 / 代号和当前星级装备状态。
```

详细设定来源：

```text
14-游侠等级与装备设定.md
```
