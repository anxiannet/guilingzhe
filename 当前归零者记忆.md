## 三、当前总规则核心

胜利条件：

```text
控制同一系统域的3座在线节点，立即获得最高权限并获胜。
```

节点与实体：

```text
每座节点最多有3个实体。
同一节点不能同时存在多个玩家控制的实体。
无主节点上不能存在任何实体。
控制节点的必要条件是该节点上至少有1个己方实体。
```

权限电荷：

```text
每回合获得等同于当前回合数的权限电荷，最高为10。
回合结束时未使用电荷清空。
攻击节点时，每派出1个己方实体，当前支付1电荷。
普通移动1个己方实体支付与单个实体攻击相同的电荷，当前为1电荷。
```

## 四、当前回合流程

```text
1. 回合开始。
2. 获得权限电荷。
3. 抽1张牌。
4. 行动阶段。
5. 结束阶段。
```

行动阶段可按任意顺序执行：

```text
打出游侠。
打出装备。
打出一次性物品。
移动己方实体。
攻击节点。
使用节点技能。
主动结束行动阶段。
```

节点技能口径：

```text
不再设置独立节点技能窗口。
节点技能可以在行动阶段中的任意合法时点使用。
每名玩家每回合最多使用1次节点技能。
发动时必须控制该节点，且该节点必须在线。
节点技能结算后，只要仍有电荷，玩家可以继续执行其他行动。
```

## 五、普通移动己方实体

普通移动是行动阶段的通用行动，不是卡牌技能，也不是节点技能。

移动条件：

```text
只能移动自己控制的游侠或装备。
只能移动到自己控制的另一座节点。
目标节点可以在线或离线，但必须少于3个实体。
不能移动到无主节点或其他玩家控制的节点。
刚上场的实体本回合不能普通移动。
每个实体每回合最多普通移动1次。
```

移动与攻击关系：

```text
普通移动和攻击节点是两项独立行动。
同一实体在同一回合可以普通移动1次，并参与攻击节点1次。
可以先移动后攻击，也可以先攻击后移动。
两项行动都必须分别支付费用，并在执行时满足合法条件。
不存在“攻击后不能移动”或“移动后才能攻击”的单向限制。
```

结算边界：

```text
普通移动不视为部署，不触发入场技能。
普通移动不改变实体控制者。
若来源节点因移动而清空，立即结算节点回流。
普通移动完成后立即检查节点归属与最高权限。
```

## 十、移动效果分层

```text
普通移动：支付攻击等价电荷，在自己的节点之间移动己方实体。
驱离枪：移动1个对手实体到该对手控制的另一节点。
维相枢纽：移动任意实体到任意合法节点，并可在传入己方节点时夺取控制权。
```

技能动作池口径：

```text
普通移动属于总规则行动，不属于手牌技能基础动作。
普通移动的存在不会自动允许游侠或物品技能任意移动实体。
武器装备只通过“强制驱离”窄例外移动对手实体。
节点技能仍可使用更宽的移动与控制权重映射动作。
```
