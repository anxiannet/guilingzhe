## 六、维度空间关联标准版卡牌

### 空白储存卡

```text
编号：G-E01。
卡名：空白储存卡 / Blank Storage Card。
类型：一次性物品。
子类型：维度封存介质。
复制数：1张。
费用：5。
技能名：二维压缩。
技能：选择场上1个游侠或装备，将其加入你的手牌。
制造商：维相科技。
```

规则边界：

```text
可以选择己方或对手实体。
效果不是复制、消灭、销毁或普通传送。
目标进入发动者手牌，表示调用权限被重新绑定。
```

### 相位转运卡

```text
编号：G-E02。
卡名：相位转运卡 / Phase Transfer Card。
类型：一次性物品。
子类型：维度转运介质。
复制数：1张。
费用：4。
技能名：空间传送。
技能：选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点，包括离线节点。
制造商：维相科技。
```

规则边界：

```text
可以选择任意玩家的游侠或装备。
目标节点必须合法容纳该实体。
移动不视为部署，不触发入场技能。
相位转运卡不改变目标控制者。
若移动到无主节点，目标原控制者获得该节点控制权。
原节点清空时立即结算节点回流。
```

动作池口径：

```text
单体空间移动作为一次性物品专属例外进入22-标准版基础动作池.md。
游侠与装备的入场技能不能使用空间移动。
多目标移动与控制权重映射仍只允许作为节点技能。
```

## 七、维度空间核心节点：维相枢纽

```text
编号：N-S02。
节点名：维相枢纽 / Phase Nexus。
系统域：维度空间。
技能名：全域重映射。
费用：10电荷。
技能：选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点，包括离线节点。若移动到你控制的节点，你获得该实体的控制权。
```

规则边界：

```text
可以选择任意玩家控制的游侠或装备。
目标节点可以在线或离线。
目标节点不得超过3个实体。
若目标移动到发动者控制节点，目标在进入后改由发动者控制，因此不形成混合控制者节点。
若目标移动到其他玩家控制节点，控制权不改变，且目标必须与该节点现有实体属于同一控制者。
若目标移动到无主节点，目标原控制者获得该节点控制权。
全域重映射视为移动，不视为部署，不触发入场技能。
若原节点清空，立即结算节点回流。
维相枢纽自身必须在线并由发动者控制。
```

设计分层：

```text
相位转运卡：4费，单体移动，不改变控制权。
维相枢纽：10费，单体移动，并可在传入己方节点时夺取控制权。
```

## 九、核心规则口径

```text
每座节点最多有3个实体。
一个节点不能同时存在多个玩家控制的实体。
在线节点可以发动节点技能并计入最高权限胜利条件。
离线节点不能发动节点技能，也不计入最高权限胜利条件。
每名玩家每回合最多发动1次节点技能。
```

正式胜利条件：

```text
控制同一系统域的3座在线节点，立即获得最高权限并获胜。
```
