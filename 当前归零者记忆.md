## 三、权限电荷资源系统

正式资源名：

```text
权限电荷 / Access Charge
```

卡面短称：

```text
电荷
```

世界观定义：

```text
权限电荷，是归零者小队在当前回合临时劫持到的短时权限窗口。
```

资源用途：

```text
打出游侠需要消耗权限电荷。
打出越权模块需要消耗权限电荷。
发动权限节点技能需要消耗权限电荷。
```

当前测试规则：

```text
每回合开始时，当前玩家获得等同于当前回合数的权限电荷。
本回合电荷 = 当前回合数，最高为10。
第10回合起，每回合获得10电荷。
回合结束时，未使用的权限电荷全部清空。
没有足够电荷时，不能打出对应卡牌，也不能发动对应技能。
```

费用口径：

```text
游侠打出费用默认等于当前星级。
越权模块费用按模块强度单独标注。
权限节点技能默认花费1电荷。
游侠入场效果包含在该游侠的打出费用中，不额外支付技能费用。
```

节奏目标：

```text
第1回合只能做低成本动作。
第3回合开始可以稳定打出3星游侠。
第5回合开始可以直接打出5星游侠。
第10回合后进入满权限对抗阶段。
```

## 五、对战与多人核心规则

```text
双人 / 多人游玩时不使用节点牌库。
双人 / 多人游玩时不从节点卡中随机抽取节点。
权限节点作为场上固定争夺对象存在。
游侠不再拥有“查看节点牌库 / 调整节点牌库 / 重洗节点牌库”的技能。
原本与节点牌库相关的效果，转移到固定权限节点自身的节点技能体系中。
```

使用方式：

```text
2 到 4 人共用游玩：所有玩家共用同一副标准手牌牌库与同一个手牌弃牌区。
对战卡组游玩：每名玩家可以各自使用一副标准手牌牌库与对应手牌弃牌区。
```

统一区位处理：

```text
抽牌：从当前模式对应的手牌牌库抽。
弃牌：放入当前模式对应的手牌弃牌区。
处理 / 清理：置入来源手牌弃牌区。
吹回：回到该游侠的控制者手牌。
放回牌库：放回来源手牌牌库。
隔离：放入隔离区，不改变控制关系；隔离结束按技能返回。
支援游侠：不是实体牌，不进入任何玩家手牌、手牌牌库或手牌弃牌区。
```

禁用旧词：

你的牌库
你的手牌牌库

## 七、权限节点三类技能结构

每个系统域固定 3 座权限节点。

每个系统域内，3 座权限节点分别承担：

```text
侦察类
进攻类
防守类
```

分类说明：

```text
侦察类：查看、筛选、预判手牌或信息。
进攻类：强化接管、推进争夺、提升进攻游侠星级。
防守类：保护己方节点、恢复游侠、阻止或削弱敌方接管。
```

## 九、游侠等级与装备设定

```text
1星游侠：只有匕首类冷兵器和简易工具，几乎无护甲。打出费用1电荷。
2星游侠：拥有手枪、自制枪械或土改武器，弹药和稳定性不足。打出费用2电荷。
3星游侠：拥有残缺或低配制式装备，品质不高，弹药不足。打出费用3电荷。
4星游侠：拥有全套制式装备，头盔、防弹衣齐全，弹药充足，装备精良。打出费用4电荷。
5星游侠：拥有轻型动力机甲或高度集成外骨骼装备，接近小队核心战力。打出费用5电荷。
```

说明：

```text
游侠星级代表当前装备状态，不作为游侠中文代号命名依据。
游侠插画必须同时参考技能、人物气质 / 代号和当前星级装备状态。
```
