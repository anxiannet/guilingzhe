## 五、维度空间关联标准版卡牌

### 破解储存卡

```text
编号：G-E01。
卡名：破解储存卡 / Cracked Storage Card。
类型：一次性物品。
子类型：破解维度封存介质。
复制数：1张。
费用：5。
技能名：越权压缩。
技能：选择场上1个游侠或装备，将其加入你的手牌。
制造商：维相科技。
```

规则边界：

```text
可以选择己方或对手实体。
目标以二维封存体状态进入发动者手牌。
目标调用权限被重绑给发动者。
该效果不是复制、消灭、销毁或普通传送。
原节点清空时立即结算节点回流。
```

### 相位转运卡

```text
编号：G-E02。
卡名：相位转运卡 / Phase Transfer Card。
类型：一次性物品。
子类型：维度转运介质。
复制数：1张。
费用：4。
技能名：空间传送。
技能：选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点，包括离线节点。
制造商：维相科技。
```

规则边界：

```text
不改变目标控制者。
移动不视为部署，不触发入场技能。
若移动到无主节点，目标原控制者获得该节点控制权。
原节点清空时立即结算节点回流。
```

### 正版储存卡

```text
编号：G-E03。
卡名：正版储存卡 / Licensed Storage Card。
类型：一次性物品。
子类型：合法维度封存介质。
复制数：1张。
费用：3。
技能名：二维压缩。
技能：选择场上1个游侠或装备，将其返回其控制者手牌。
制造商：维相科技。
```

规则边界：

```text
可以选择己方或对手实体。
目标必须返回其当前控制者手牌。
返回手牌不是移动、消灭或销毁。
目标重新打出时必须重新支付费用。
原节点清空时立即结算节点回流。
```

版本区别：

```text
正版储存卡：合法二维压缩，目标返回原控制者手牌。
破解储存卡：越权压缩，目标进入发动者手牌。
```

## 六、维度空间核心节点：维相枢纽

```text
编号：N-S02。
节点名：维相枢纽 / Phase Nexus。
系统域：维度空间。
技能名：全域重映射。
费用：10电荷。
技能：选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点，包括离线节点。若移动到你控制的节点，你获得该实体的控制权。
```

规则边界：

```text
目标节点不得超过3个实体。
传入发动者控制节点时，目标改由发动者控制。
传入其他节点时不改变控制权。
移动不视为部署，不触发入场技能。
原节点清空时立即结算节点回流。
```

## 七、设计分层

```text
正版储存卡：3费，返回原控制者手牌。
相位转运卡：4费，移动到其他节点，不改变控制权。
破解储存卡：5费，加入发动者手牌并重绑调用权。
维相枢纽：10费，移动并可夺取控制权。
```

---

## 八、核心规则口径

```text
每座节点最多有3个实体。
一个节点不能同时存在多个玩家控制的实体。
在线节点可以发动节点技能并计入最高权限胜利条件。
离线节点不能发动节点技能，也不计入最高权限胜利条件。
控制同一系统域的3座在线节点，立即获得最高权限并获胜。
```
