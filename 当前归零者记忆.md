## 一、项目定位

```text
归零者 / ZERO ACCESS
```

定位：

```text
一款关于系统弃民、越权装备与城市权限夺取的独立战术卡牌项目。
```

核心口号：

```text
从零权限，到根权限。
From Zero Access to Root Access.
```

玩家身份：

```text
玩家扮演一支零权限小队 / 归零者小队的首领。
小队成员是被城市权限系统删除的人。
目标不是重新申请权限，而是夺取 Root Access。
```

核心幻想：

```text
系统不给你权限，你就夺取 Root Access。
```

## 五、首版系统域与权限节点

首版固定 4 个系统域，每个系统域 3 个权限节点。

| 系统域 | 节点1 | 节点2 | 节点3 |
|---|---|---|---|
| 门禁域 Access Domain | 高塔门禁 Tower Gate | 电梯栈 Elevator Stack | 封锁线闸门 Lockdown Gate |
| 医疗域 Medical Domain | 纳米诊所 Nano Clinic | 复苏舱 Revival Pod | 生物档案库 Bio Archive |
| 无人机域 Drone Domain | 蜂鸟巢 Hummingbird Hive | 空域中继 Sky Relay | 目标网格 Target Grid |
| 信用域 Credit Domain | 黑钱包 Black Wallet | 信用铸机 Credit Mint | 债务库 Debt Vault |

`身份域 Identity Domain` 暂不进入首版，作为扩展候选。

```text
在你的回合中，你可以使用1次你已控制权限节点的节点技能。

没有合法目标时，不能发动对应节点技能。
```

## 七、当前视觉母设定

归零者卡牌不是普通印刷插画卡，而是：

```text
高科技二维封存体卡牌。
```

核心定义：

```text
《归零者》中的卡牌，是将执行者、越权模块与权限节点压缩封存后的二维权限载体。
调用一张卡，不是“打出一张牌”，而是“释放一个被封存的战术实体”。
```

视觉关键词：

```text
高科技封存载体
二维压缩
透明数据膜
卡内压缩空间
空间折叠边缘
深色金属卡框
半透明技能面板
收藏卡质感
冷白青蓝 Root Access 光
```

禁止方向：

```text
不要普通赛博朋克霓虹街头。
不要普通黑客键盘敲代码感。
不要普通雇佣兵枪战。
不要大面积白底说明卡。
不要脏旧废土感。
不要过度刮痕和复杂花纹。
不要让卡面像教学说明页。
不要让 AI 生成文字直接进入印刷终稿。
```

5. 底边附近 Logo
