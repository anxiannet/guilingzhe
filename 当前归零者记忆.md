## 一、项目定位

```text
归零者 / ZERO ACCESS
```

定位：

```text
一款关于系统弃民、越权装备与城市权限夺取的独立战术卡牌项目。
```

核心口号：

```text
从零权限，到根权限。
From Zero Access to Root Access.
```

玩家身份：

```text
玩家扮演一支零权限小队 / 归零者小队的首领。
小队成员是被城市权限系统删除的人。
目标不是重新申请权限，而是夺取 Root Access。
```

核心幻想：

```text
系统不给你权限，你就夺取 Root Access。
```

## 五、首版系统域与权限节点

首版固定 4 个系统域，每个系统域 3 个权限节点。

| 系统域 | 节点1 | 节点2 | 节点3 |
|---|---|---|---|
| 门禁域 Access Domain | 高塔门禁 Tower Gate | 电梯栈 Elevator Stack | 封锁线闸门 Lockdown Gate |
| 医疗域 Medical Domain | 纳米诊所 Nano Clinic | 复苏舱 Revival Pod | 生物档案库 Bio Archive |
| 无人机域 Drone Domain | 蜂鸟巢 Hummingbird Hive | 空域中继 Sky Relay | 目标网格 Target Grid |
| 信用域 Credit Domain | 黑钱包 Black Wallet | 信用铸机 Credit Mint | 债务库 Debt Vault |

`身份域 Identity Domain` 暂不进入首版，作为扩展候选。

权限节点封存卡口径：

```text
12 张权限节点共用同一套权限节点外框。
权限节点卡号不是后台管理编号，而是世界观内真实存在的封存卡存储序列。
系统域识别不是辅助说明，而是该存储卡的正式域标签。
系统域通过域标签、域代号、域色光条、域图标、插画主题和节点位址区分。
卡号同时作为版本管理入口。
卡号只有卡面上的一个。
版本记录归属到卡号下面，但不拼进卡号。
蜂鸟巢唯一卡号为 N-007 / DN-D1。
```

## 六、归零者专属规则状态

当前规则状态：

```text
归零者当前保留“权限节点技能”作为测试规则。
后续是否成为正式独立规则，需要继续试玩验证。
```

权限节点技能规则：

```text
在你的回合中，你可以使用1次你已控制权限节点的节点技能。
节点技能不占用主要行动。
每名玩家每回合最多使用1次节点技能。
即使你控制多个权限节点，每回合也只能选择其中1座权限节点发动技能。
中央区无主权限节点不能发动节点技能。
没有合法目标时，不能发动对应节点技能。
```

节点技能卡面写法：

```text
“控制此节点”作为规则前提写入规则说明，不再作为每张权限节点卡的固定前缀。
权限节点卡面只写具体效果。
```

示例：

```text
查看节点牌库顶1张，可置顶或置底。
```

## 七、当前视觉母设定

归零者卡牌不是普通印刷插画卡，而是：

```text
高科技二维封存体卡牌。
```

核心定义：

```text
《归零者》中的卡牌，是将执行者、越权模块与权限节点压缩封存后的二维权限载体。
调用一张卡，不是“打出一张牌”，而是“释放一个被封存的战术实体”。
```

视觉关键词：

```text
高科技封存载体
二维压缩
透明数据膜
卡内压缩空间
空间折叠边缘
深色金属卡框
半透明技能面板
收藏卡质感
真实存储卡
唯一卡号
系统域标签
冷白青蓝 Root Access 光
```

禁止方向：

```text
不要普通赛博朋克霓虹街头。
不要普通黑客键盘敲代码感。
不要普通雇佣兵枪战。
不要大面积白底说明卡。
不要脏旧废土感。
不要过度刮痕和复杂花纹。
不要让卡面像教学说明页。
不要让 AI 生成文字直接进入印刷终稿。
不要把封存卡号做成后台制作备注感。
不要把系统域标签做成旁白说明感。
不要制造第二套卡号。
不要把版本号拼进卡号。
```

## 十、当前制卡工作流

后续不再直接生成完整成卡来赌文字与细节。

推荐流程：

```text
1. 使用蜂鸟巢无文字插图候选 01。
2. 参考历史成品卡与版式区域标注。
3. 重新处理卡框与插画窗口。
4. 用设计工具合成卡框、插图和人工文字。
5. 技能文案、唯一卡号、系统域标签、Logo 全部人工重排。
6. 导出蜂鸟巢中文版样卡 v1。
7. 在审核记录中记录卡号、系统域标签、语言版本、样卡版本、插图版本和卡框版本。
```

AI 不应负责最终文字：

```text
技能文案
N-007 / DN-D1
无人机域
归零者 / ZERO ACCESS
```

这些必须在设计工具中人工排版。
