## 二、模式定位

```text
混战 = 全局战争。
对战 = 单个节点的局部战争。
```

### 混战｜全局战争

```text
人数：2至4人。
所有玩家共用公共手牌牌库与公共弃牌区。
战场由多座节点组成。
使用节点卡、节点牌库、中央节点区与节点技能。
目标是控制同一系统域的3座节点，获得 Root Access。
```

### 对战｜局部战争

```text
每名玩家使用自己构筑的独立牌组。
每名玩家从自己的牌库抽牌。
战场固定为1座唯一节点。
唯一节点不是节点卡，没有节点技能。
不使用节点牌库、中央节点区或 Root Access 胜利条件。
节点装备可以加入独立牌组并部署到唯一节点。
```

局部战争的组牌规则、唯一节点容量、战斗流程与胜利条件另行登记。

## 三、卡牌类型术语

```text
一级卡牌类型：游侠、物品卡、节点。
物品卡分为：装备、消耗品。
原术语“一次性物品”已废止，统一改称“消耗品”。
```

消耗品规则口径：

```text
打出后立即结算效果。
结算后进入弃牌区，除非卡面另有说明。
不作为场上实体存在。
只显示费用，不显示ATK与HP。
代码内部类型标识统一使用 consumable。
```

## 四、标准版手牌动作池

标准版手牌有效动作当前为9种：

```text
1. 获得手牌牌库。
2. 获得公共弃牌区。
3. 调整手牌。
4. 调整手牌牌库。
5. 调整公共弃牌区。
6. 调整游侠。
7. 调整装备。
8. 消除游侠。
9. 消除装备。
```

标准版手牌技能禁止混牌：

```text
不得取得其他玩家手牌。
不得取得其他玩家场上游侠或装备的控制权。
不得将其他玩家卡牌加入自己的手牌、牌库或控制区。
```

扩展包手牌动作：

```text
获得其他玩家手牌。
获得场上游侠。
获得场上装备。
消除手牌。
消除手牌牌库。
```

查看属于信息步骤，不计入9种动作。

## 五、节点技能边界

节点卡只用于混战的全局战争。

```text
标准版节点技能可以包含混牌或控制权变化效果。
节点技能使用独立节点动作池。
节点技能不计入标准版手牌动作覆盖率。
```

对战是单个节点的局部战争，不使用节点卡或节点技能，因此不受节点混牌技能影响。

## 六、标准版卡表口径

```text
游侠、装备和消耗品原则上应兼容全局战争与局部战争。
节点卡只用于全局战争。
节点装备可用于两种模式。
候选卡与扩展包卡必须单独标记。
```

当前标准版核定：

```text
信使：2张。
鹰眼：2张。
蜂鸟：4张。
驱离枪：2张。
轻动力甲：2张。
防御地堡：2张。
基因备份卡：2张。
正版储存卡：2张。
力量激素：2张。
耐力激素：2张。
强化激素：2张。
蜂鸟巢：1张。
维相枢纽：1张。
指挥中心：1张。
```

当前兼容审核：

```text
驱离枪依赖“另一座节点”，在局部战争中无法正常结算。
轻动力甲依赖普通跨节点移动，在局部战争中功能可能失效。
这两张卡需要模式兼容处理。
```

## 七、节点与胜利条件

全局战争胜利句：

```text
控制同一系统域的3座节点，立即获得 Root Access 并获胜。
```

局部战争不使用 Root Access 胜利条件。

维相枢纽和指挥中心均为标准版节点，仅用于全局战争。

---

## 八、装备分类与模式适用

```text
节点装备：只能独立部署。
游侠装备：只能由游侠携带。
通用装备：可以独立部署，也可以由游侠携带。
每个游侠最多携带1件装备。
```

模式适用：

```text
全局战争：节点装备部署到玩家控制的节点。
局部战争：节点装备部署到唯一节点。
局部战争中，通用装备可独立部署到唯一节点，也可由游侠携带。
```
