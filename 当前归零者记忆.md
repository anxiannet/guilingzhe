## 二、模式定位

```text
混战 = 全局战争。
对战 = 单个节点的局部战争。
```

### 混战｜全局战争

```text
人数：2至4人。
所有玩家共用公共手牌牌库与公共弃牌区。
战场由多座节点组成。
使用节点卡、节点牌库、中央节点区与节点技能。
目标是控制同一系统域的3座节点，获得 Root Access。
```

### 对战｜局部战争

```text
每名玩家使用自己构筑的独立牌组。
每名玩家从自己的牌库抽牌。
战场固定为1座唯一节点。
唯一节点不是节点卡，没有节点技能。
不使用节点牌库、中央节点区或 Root Access 胜利条件。
设施可以加入独立牌组并部署到唯一节点。
```

局部战争的组牌规则、唯一节点容量、战斗流程与胜利条件另行登记。

## 三、卡牌类型术语

```text
一级卡牌类型：游侠、装备、设施、消耗品、节点。
装备：只能由游侠携带的实体卡牌。
设施：只能独立部署到节点的实体卡牌。
消耗品：结算后进入弃牌区，不作为场上实体存在。
节点：只用于全局战争的战场卡牌。
```

术语废止：

```text
“游侠装备”废止，统一称“装备”。
“节点装备”废止，统一称“设施”。
“通用装备”概念废止。
“一次性物品”废止，统一称“消耗品”。
```

双类型规则：

```text
个别卡牌可以同时具有“装备／设施”双类型。
双类型卡打出时，选择作为装备由游侠携带，或作为设施部署到节点。
进入场上后，按当前部署方式视为装备或设施。
蜂鸟是当前标准版装备／设施双类型卡。
```

消耗品规则口径：

```text
打出后立即结算效果。
结算后进入弃牌区，除非卡面另有说明。
不作为场上实体存在。
只显示费用，不显示ATK与HP。
代码内部类型标识统一使用 consumable。
```

## 四、标准版手牌动作池

标准版手牌有效动作当前为9种：

```text
1. 获得手牌牌库。
2. 获得公共弃牌区。
3. 调整手牌。
4. 调整手牌牌库。
5. 调整公共弃牌区。
6. 调整游侠。
7. 调整装备／设施。
8. 消除游侠。
9. 消除装备／设施。
```

标准版手牌技能禁止混牌：

```text
不得取得其他玩家手牌。
不得取得其他玩家场上游侠、装备或设施的控制权。
不得将其他玩家卡牌加入自己的手牌、牌库或控制区。
```

扩展包手牌动作：

```text
获得其他玩家手牌。
获得场上游侠。
获得场上装备／设施。
消除手牌。
消除手牌牌库。
```

查看属于信息步骤，不计入9种动作。

## 五、节点技能边界

节点卡只用于混战的全局战争。

```text
标准版节点技能可以包含混牌或控制权变化效果。
节点技能使用独立节点动作池。
节点技能不计入标准版手牌动作覆盖率。
```

对战是单个节点的局部战争，不使用节点卡或节点技能，因此不受节点混牌技能影响。

## 六、标准版卡表口径

```text
游侠、装备、设施和消耗品原则上必须兼容全局战争与局部战争。
节点卡只用于全局战争。
设施可用于两种模式。
候选卡与扩展包卡必须单独标记。
依赖另一座节点或跨节点移动、无法在局部战争中正常生效的手牌卡，归入扩展包。
测试版游侠按“逐张提议、逐张审核、通过后迁移”的流程进入标准版。
```

## 八、节点与胜利条件

全局战争胜利句：

```text
控制同一系统域的3座节点，立即获得 Root Access 并获胜。
```

局部战争不使用 Root Access 胜利条件。

维相枢纽和指挥中心均为标准版节点，仅用于全局战争。

## 九、装备与设施

```text
装备：只能由游侠携带。
设施：只能独立部署到节点。
每个游侠最多携带1件装备。
全局战争：设施部署到玩家控制的节点。
局部战争：设施部署到唯一节点。
```

蜂鸟是特殊双类型卡：

```text
类型：装备／设施。
打出时选择一种部署方式。
作为装备时，由游侠携带。
作为设施时，独立部署到节点。
进入场上后，按当前部署方式结算相关技能和动作。
```

项目不再使用“通用装备”概念。
