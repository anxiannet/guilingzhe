## 三、核心胜利与节点规则

```text
控制同一系统域的3座在线节点，立即获得最高权限并获胜。
每座节点最多有3个部署实体。
```

计入节点部署位：

```text
游侠。
独立部署的节点装备。
独立部署的通用装备。
```

不占节点部署位：

```text
由游侠携带的游侠装备。
由游侠携带的通用装备。
```

---

## 四、装备分类

```text
节点装备：只能独立部署，占用1个节点部署位。
游侠装备：只能由游侠携带，不占节点部署位。
通用装备：可以独立部署，也可以由游侠携带。
每个游侠最多携带1件装备。
```

携带装备：

```text
最终ATK = 游侠ATK + 携带装备ATK。
最终HP = 游侠HP + 携带装备HP。
HP为0或卡面明确不增加HP的装备除外。
```

战斗中，游侠与携带装备视为1个战斗单位；装备不单独攻击或承受普通战斗伤害。

---

## 五、通用装备转换与普通移动

```text
通用装备可在同一节点内免费切换独立部署与游侠携带状态。
转换不视为打出、部署、移动或重新上场，不触发技能。
移动1个己方部署实体当前支付1电荷。
游侠携带的装备随游侠移动，不额外付费。
```

## 七、阈锋军械

```text
中文名：阈锋军械。
英文名：THRESHOLD ARMAMENTS。
简称：TA。
类型：武器装备制造公司。
```

企业定位：

```text
专门制造游侠武器、节点武装与战术装备。
擅长把外部授权技术封装、整合并武器化。
负责武器平台设计、枪体结构、能源与火控系统、生产、装配和销售。
```

命名含义：

```text
“阈”代表权限阈值、系统边界与武器启动条件。
“锋”代表将技术转化为直接战术输出的武器化能力。
```

## 八、驱离枪

```text
编号：G-S03。
卡名：驱离枪 / Displacement Gun。
装备分类：游侠装备。
唯一：否。
复制数：2。
费用：3。
ATK 2 / HP 0。
技能名：定向驱离。
制造商：阈锋军械 / TA。
技术支持：维相科技 / DPT。
```

技能：

```text
此装备给游侠装备时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

规则边界：

```text
驱离枪不能直接部署到节点。
为携带者提供+2 ATK，不增加HP。
没有合法目标时仍可装备，但不执行定向驱离。
同节点内转交或重新调整驱离枪不会重新触发技能。
```

企业分工：

```text
阈锋军械负责驱离枪的武器化设计、枪体制造、能源与火控系统、装配和销售。
维相科技提供维相转换技术、定向坐标偏移模块、授权接口和安全标准。
维相科技不是驱离枪制造商。
```
