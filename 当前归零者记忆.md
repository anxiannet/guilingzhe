## 三、双人对战核心规则

```text
双人对战时不使用节点牌库。
双人对战时不从节点卡中随机抽取节点。
权限节点作为场上固定争夺对象存在。
游侠不再拥有“查看节点牌库 / 调整节点牌库 / 重洗节点牌库”的技能。
原本与节点牌库相关的效果，转移到固定权限节点自身的节点技能体系中。
```

手牌牌库定义：

```text
手牌牌库 = 游侠牌 + 越权模块牌组成的抽牌牌库。
双人对战时，每名玩家各自使用一副手牌牌库。
每名玩家各自拥有手牌、手牌牌库、手牌弃牌区、已打出游侠与已使用模块。
每张游侠牌与越权模块牌始终归属于其拥有者。
任何技能都不能改变卡牌拥有者。
```

统一归属：

```text
抽牌：从你的手牌牌库抽。
弃牌：放入你的手牌弃牌区。
处理 / 清理：置入其拥有者手牌弃牌区。
吹回：回到其拥有者手牌。
放回牌库：放回其拥有者手牌牌库。
隔离：放入其拥有者隔离区，不改变拥有者。
支援游侠：不是实体牌，不进入任何玩家手牌、手牌牌库或手牌弃牌区。
```

禁用旧词：

```text
公共牌库
弃牌堆
节点牌库
```

## 五、游侠等级与装备设定

```text
1星游侠：只有匕首类冷兵器和简易工具，几乎无护甲。
2星游侠：拥有手枪、自制枪械或土改武器，弹药和稳定性不足。
3星游侠：拥有残缺或低配制式装备，品质不高，弹药不足。
4星游侠：拥有全套制式装备，头盔、防弹衣齐全，弹药充足，装备精良。
5星游侠：拥有轻型动力机甲或高度集成外骨骼装备，接近小队核心战力。
```

说明：

```text
游侠星级代表当前装备状态，不作为游侠中文代号命名依据。
游侠插画必须同时参考技能、人物气质 / 代号和当前星级装备状态。
```

## 九、当前视觉母设定

归零者卡牌不是普通印刷插画卡，而是：

```text
高科技二维封存体卡牌。
```

核心定义：

```text
《归零者》中的卡牌，是将游侠、越权模块与权限节点压缩封存后的二维权限载体。
调用一张卡，不是“打出一张牌”，而是“释放一个被封存的战术实体”。
```

插画空间层次硬标准：

```text
插画必须具备明确空间层次与立体感。
画面应尽量建立特写、近景、中景、远景中的多层结构。
通过遮挡、透视、体块、景深、封存边缘和空间折叠关系，表现“高科技二维封存体”的卡内压缩空间感。
```
