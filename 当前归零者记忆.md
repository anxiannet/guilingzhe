## 四、一级卡牌类型与牌库

一级卡牌类型：

```text
游侠。
物品卡。
节点。
```

物品卡定义：

```text
物品卡 / Gear，是所有非游侠手牌卡的统称。
装备，是可以作为可部署实体进入节点的物品卡。
一次性物品，只结算效果，不留在节点上。
```

当前牌库模式：

```text
2到4人共用1个手牌牌库。
玩家共用1个公共弃牌区。
玩家共用1个节点牌库。
中央节点区通常保持3座无主节点。
双人对战独立牌库规则尚未设定。
```

旧称：

```text
旧称“越权模块”废止，不再作为卡面主类型。
“权限节点”简化为“节点”。
```

维相转换技术 / Dimensional Phase Conversion。

现实，只是一种展开状态。

## 八、空白储存卡

```text
编号：G-E01。
卡名：空白储存卡 / Blank Storage Card。
类型：一次性物品。
子类型：维度封存介质。
复制数：1张。
费用：5。
制造商：维相科技。
内部型号：DPT-P0。
技能名：二维压缩。
技能：选择场上1个游侠或装备，将其加入你的手牌。
```

规则解释：

```text
可以选择己方或对手控制的游侠或装备。
目标离开原节点，以二维封存体状态进入发动者手牌。
该效果不是复制、消灭、销毁或普通传送。
若原节点因此没有任何实体，立即结算节点回流。
```

## 九、维度空间核心节点：维相枢纽

节点口径：

```text
编号：N-S02。
节点名：维相枢纽。
英文名：Phase Nexus。
系统域：维度空间。
技能名：空间传送。
费用：10电荷。
技能：选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点，包括离线节点。
```

规则边界：

```text
可以选择任意玩家控制的游侠或装备。
目标节点可以是在线节点或离线节点。
目标节点不得超过3个实体。
移动后不能使同一节点出现多个玩家控制的实体。
若目标移动到无主节点，该实体控制者立即获得该节点控制权。
空间传送视为移动，不视为部署，不触发入场技能。
空间传送不改变目标实体的控制者。
若原节点因目标离开而没有任何实体，立即结算节点回流。
维相枢纽自身必须在线并由发动者控制，才能发动节点技能。
离线节点仍不能发动节点技能，也不计入最高权限胜利条件。
```

世界观定位：

```text
维相枢纽是城市级空间运输与实体调度网络的核心节点。
它通过相位锁定、坐标解除和重新展开，把实体移动到其他合法节点。
它不是普通传送门，也不是维相科技总部。
```

## 十、资源、战斗与胜利口径

资源：

```text
每回合电荷 = min(当前回合数, 10)。
每回合不限制打出手牌张数。
每回合不限制攻击节点次数。
每打出1张游侠或物品卡，立即支付其费用。
每派出1个实体攻击节点，立即支付1电荷。
每名玩家每回合最多发动1次节点技能。
刚上场的游侠或装备，本回合不能参与攻击节点。
```

节点规则：

```text
每座节点最多有3个实体。
一个节点上不能同时存在多个玩家控制的实体。
无主节点上通常不能存在实体；若维相枢纽把实体移动到无主节点，则按该技能专门规则获得节点控制权。
在线节点可以发动节点技能并计入最高权限胜利条件。
离线节点不能发动节点技能，也不计入最高权限胜利条件。
```

正式胜利条件：

```text
控制同一系统域的3座在线节点，立即获得最高权限并获胜。
```

英文：

```text
Control all 3 online Nodes in one System Domain to gain Root Access and win.
```
