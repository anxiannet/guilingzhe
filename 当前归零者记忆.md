## 三、当前总规则核心

胜利条件：

```text
控制同一系统域的3座在线节点，立即获得最高权限并获胜。
```

节点与实体：

```text
每座节点最多有3个实体。
同一节点不能同时存在多个玩家控制的实体。
无主节点上不能存在任何实体。
控制节点的必要条件是该节点上至少有1个己方实体。
```

权限电荷：

```text
每回合获得等同于当前回合数的权限电荷，最高为10。
回合结束时未使用电荷清空。
攻击节点时，每派出1个己方实体，当前支付1电荷。
```

## 四、普通移动己方实体

普通移动是行动阶段的通用行动，不是卡牌技能，也不是节点技能。

当前规则：

```text
玩家可以支付与派出1个实体攻击节点相同的电荷，移动1个自己的游侠或装备。
当前攻击单个实体的费用为1电荷，因此普通移动1个实体当前支付1电荷。
如果以后调整单个攻方实体的攻击费用，普通移动费用同步调整。
```

移动条件：

```text
只能移动自己控制的游侠或装备。
只能移动到自己控制的另一座节点。
目标节点可以在线或离线，但必须少于3个实体。
不能移动到无主节点或其他玩家控制的节点。
刚上场的实体本回合不能普通移动。
本回合已经攻击过的实体不能普通移动。
每个实体每回合最多进行1次普通移动。
```

结算边界：

```text
普通移动不视为部署，不触发入场技能。
普通移动不改变实体控制者。
完成普通移动的实体，本回合仍可参与1次攻击节点。
若来源节点因移动而清空，立即结算节点回流。
普通移动完成后立即检查节点归属与最高权限。
```

设计目的：

```text
允许玩家付出与攻击相同的单体行动成本，在己方节点之间重新调配驻守与攻击位置。
普通移动不能用于低费占领无主节点，也不能侵入敌方节点。
```

## 九、移动效果分层

```text
普通移动：支付攻击等价电荷，在自己的节点之间移动己方实体。
驱离枪：移动1个对手实体到该对手控制的另一节点。
维相枢纽：移动任意实体到任意合法节点，并可在传入己方节点时夺取控制权。
```

技能动作池口径：

```text
普通移动属于总规则行动，不属于手牌技能基础动作。
普通移动的存在不会自动允许游侠或物品技能任意移动实体。
武器装备只通过“强制驱离”窄例外移动对手实体。
节点技能仍可使用更宽的移动与控制权重映射动作。
```
