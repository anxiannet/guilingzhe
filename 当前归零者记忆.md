## 二、中文名称规范

```text
名称必须简单、直观，符合中文语境。
名称还必须符合《归零者》的时代背景、技术水平和世界观逻辑。
玩家看到名称后，应能直接判断它是什么，或大致理解它的用途。
名称不应依赖额外解释才能成立。
优先使用常见、明确、可识别的中文词组。
卡牌名称通常控制在2至5个汉字。
```

命名先说明对象是什么，再体现核心功能或结构，然后检查是否符合未来城市权限系统、零权限者、高科技封存载体及相关企业技术体系，最后才考虑风格修饰。

每个新名称必须同时通过：

```text
中文语境审核。
对象识别审核。
功能直觉审核。
世界观审核。
时代感审核。
```

禁止：

```text
生造难懂词。
翻译腔。
只追求气势的抽象名称。
与世界观技术体系无关的普通现代软件名。
虽然功能准确，但不像《归零者》世界中真实存在的人物、设备、物品或节点名称。
```

若名称必须经过一段解释才能理解，或与项目时代背景不符，应重新命名。

## 三、卡牌类型术语

```text
一级卡牌类型：游侠、物品卡、节点。
物品卡分为：装备、消耗品。
原术语“一次性物品”已废止，统一改称“消耗品”。
```

消耗品规则口径：

```text
消耗品打出后立即结算效果。
结算后进入公共弃牌区，除非卡面另有说明。
消耗品不作为场上实体存在。
消耗品只显示费用，不显示ATK与HP。
代码内部类型标识统一使用 consumable，不再使用 one_shot。
```

## 四、标准版支持模式

标准版不是仅限双人公共牌库模式。

当前标准版支持：

```text
2至4人公共牌库混战。
玩家各自构筑独立牌组进行对战。
```

公共牌库混战：

```text
所有玩家从同一公共手牌牌库抽牌。
所有玩家共用公共弃牌区。
跨玩家获得卡牌或改变控制权不会产生实体卡所有权混淆。
```

独立牌组对战：

```text
每名玩家使用自己构筑的独立牌组。
卡牌所有权由组牌时的牌组归属决定。
获得控制权不改变卡牌所有权。
其他玩家的卡牌进入你的手牌或控制区时，必须保持可识别。
卡牌返回牌库时，默认返回其所有者的牌库。
游戏结束后，所有卡牌归还原所有者。
```

## 五、标准版基础动作池

标准版有效动作当前为12种：

```text
1. 获得其他玩家手牌。
2. 获得手牌牌库。
3. 获得公共弃牌区。
4. 获得场上游侠。
5. 获得场上装备。
6. 调整手牌。
7. 调整手牌牌库。
8. 调整公共弃牌区。
9. 调整游侠。
10. 调整装备。
11. 消除游侠。
12. 消除装备。
```

标准版模式边界：

```text
获得其他玩家手牌、获得场上游侠、获得场上装备属于标准版动作。
在独立牌组模式下，这些动作必须遵守卡牌所有权与赛后归还规则。
查看是信息步骤，不计入12种有效动作。
消耗品是卡牌类型限制，不是独立基础动作对象。
```

扩展包动作：

```text
消除手牌。
消除手牌牌库。
```

原因：

```text
两者都会使有限牌库中的卡牌更快进入弃牌区。
不进入标准版动作覆盖率。
```

已删除动作：

```text
获得消耗品。
调整消耗品。
消除公共弃牌区。
消除消耗品。
```

## 七、节点状态与胜利条件

```text
节点在线、离线机制已移除。
节点不再记录在线或离线状态。
节点技能只要求玩家控制对应节点并支付费用，不再检查在线状态。
游侠移动、装备重新部署和技能移动不再引用在线或离线节点。
```

当前胜利句：

```text
控制同一系统域的3座节点，立即获得 Root Access 并获胜。
```

维相枢纽：

```text
技能名：全域调度。
费用：10。
技能：选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点。若移动到你控制的节点，你获得该实体的控制权。
属于标准版节点技能。
独立牌组模式下，获得控制权不改变卡牌所有权。
```

指挥中心：

```text
技能名：紧急征用。
费用：10。
属于标准版2至4人混战及独立牌组对战可用节点。
独立牌组模式下，被征用装备仍保留原所有权。
```

## 八、卡面三维信息

```text
三维统一指：费用、ATK、HP。
游侠与可部署装备必须在正式卡面显示完整三维。
三维数字必须独立可读，并使用同类型固定位置。
不得使用星级替代ATK。
消耗品只显示费用，不显示ATK与HP，也不显示“不适用”占位。
节点只显示节点技能费用，不套用游侠与装备的ATK、HP三维。
```

游侠正式卡面信息：

```text
卡名、费用、ATK、HP、技能描述、编号、Logo。
```

可部署装备正式卡面信息：

```text
卡名、费用、ATK、HP、技能／功能／规则限制文本、编号、Logo。
```

消耗品正式卡面信息：

```text
卡名、费用、技能描述、编号、Logo。
```

节点正式卡面信息：

```text
卡名、系统域标签、节点技能费用、节点技能描述、卡号／节点位址、Logo。
```

## 九、装备分类与移动

```text
节点装备：只能独立部署，占用1个节点部署位。
游侠装备：只能由游侠携带，不占节点部署位。
通用装备：可以独立部署，也可以由游侠携带。
每个游侠最多携带1件装备。
```

```text
游侠执行普通移动，当前支付1电荷。
节点装备执行重新部署，费用与游侠普通移动相同。
重新部署不是移动、打出、部署或重新上场，不触发装备技能。
```
