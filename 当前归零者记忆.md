## 二、中文名称规范

```text
名称必须简单、直观，符合中文语境。
玩家看到名称后，应能直接判断它是什么，或大致理解它的用途。
名称不应依赖额外解释才能成立。
优先使用常见、明确、可识别的中文词组。
卡牌名称通常控制在2至5个汉字。
```

```text
命名先说明对象是什么，再体现核心功能或结构，最后才考虑企业风格与世界观修饰。
若名称必须经过一段解释才能理解，应重新命名。
```

## 四、装备分类与移动

```text
节点装备：只能独立部署，占用1个节点部署位。
游侠装备：只能由游侠携带，不占节点部署位。
通用装备：可以独立部署，也可以由游侠携带。
每个游侠最多携带1件装备。
```

```text
游侠执行普通移动，当前支付1电荷。
节点装备执行重新部署，费用与游侠普通移动相同。
重新部署不是移动、打出、部署或重新上场，不触发装备技能。
```

## 十一、三种战术激素

统一属性：

```text
类型：一次性物品。
子类型：激素。
复制数：每种2张。
费用：每种1。
制造企业：纳米科技公司。
研发部门：生物基因部。
```

统一规则：

```text
激素只能对己方游侠使用。
每个游侠每回合最多使用1张激素。
激素效果持续到当前回合结束。
```

```text
力量激素：本回合获得+2 ATK。
耐力激素：本回合获得+2 HP。
强化激素：本回合获得+1 ATK和+1 HP。
```
