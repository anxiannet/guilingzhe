## 一、项目定位

```text
归零者 / ZERO ACCESS
```

定位：

```text
一款关于系统弃民、物品装备与城市节点夺取的独立战术卡牌项目。
```

核心口号：

```text
从零权限，到最高权限。
From Zero Access to Root Access.
```

核心幻想：

```text
系统不给你权限，你就夺取最高权限。
```

核心胜利术语：

```text
Root Access 的中文术语统一为“最高权限”。
中文规则和中文世界观优先写“最高权限”。
中英对照首次出现时可写“最高权限 / Root Access”。
```

## 八、资源与行动口径

```text
每回合获得权限电荷：本回合电荷 = min(当前回合数, 10)。
每回合不限制打出手牌张数。
每回合不限制攻击节点次数。
每打出1张游侠或物品卡，立即支付其费用。
每派出1个实体攻击节点，立即支付1电荷。
节点技能每回合最多使用1次，消耗电荷。
节点技能只能在节点技能窗口使用。
刚上场的游侠或装备，本回合不能参与攻击节点。
```

费用锚点：

```text
10费是终局锚点费用。
10费效果用于加快收束、集中战力或帮助玩家完成胜利条件。
10费效果不应自动跳过胜利条件检查。
```

## 九、ATK / HP 数值锚点

```text
旧星级不作为规则字段，只作为 ATK 的视觉锚点。
ATK 是主动突破能力。
HP 是防御与驻守能力。
费用是卡牌总价值。
生命值可以在玩家说明中理解为防御力，但正式字段仍写 HP / 生命值。
```

ATK 锚点：

```text
ATK 0：不能主动攻击节点，偏侦察、工具、后勤、无人设备、信标、数据设备。
ATK 1：轻量攻击，视觉参考旧1星游侠感，匕首、撬棍、短刀、旧工具、临时护具。
ATK 2：标准攻击，视觉参考旧2星至3星游侠感，手枪、自制枪械、土改武器、低配制式装备。
ATK 3：强攻击，视觉参考旧4星至5星游侠感，完整战术装备、稳定火力、重火力、外骨骼部件。
```

HP 锚点：

```text
HP 1：脆弱，无护甲或临时护具。
HP 2：基础防御，厚外套、拼接护具、局部护甲。
HP 3：稳定驻守，低配制式护甲、战术背心、头盔。
HP 4：强防御，完整防弹衣、重型护甲、盾牌。
HP 5：重型防御 / 工事，重型装甲、固定工事、防御地堡、外骨骼防护结构。
```

## 十、攻击节点与胜利条件

攻击节点条件：

```text
攻击方可以从自己控制的任意节点中派出任意数量的己方游侠或装备作为攻方。
攻方不能包含本回合刚上场的实体。
攻方不能包含本回合已攻击过的实体。
攻方不能包含 ATK 为0的实体。
每派出1个攻方实体，支付1电荷。
攻击目标必须是1座由其他玩家控制的节点。
无主节点不能成为攻击目标；无主节点只能通过部署获得节点控制权。
被保护节点不能成为攻击目标。
```

正式胜利条件：

```text
控制同一系统域的3座在线节点，立即获得最高权限并获胜。
```

英文：

```text
Control all 3 online Nodes in one System Domain to gain Root Access and win.
```
