## 二、中文名称规范

```text
名称必须简单、直观，符合中文语境。
玩家看到名称后，应能直接判断它是什么，或大致理解它的用途。
名称不应依赖额外解释才能成立。
优先使用常见、明确、可识别的中文词组。
卡牌名称通常控制在2至5个汉字。
```

命名优先级：

```text
1. 先说明对象是什么。
2. 再体现最核心的功能、结构或识别特征。
3. 最后才考虑企业风格、技术气质和世界观修饰。
```

判断标准：

```text
遮住卡牌说明，只看名称，玩家仍应能对卡牌类型或主要作用形成正确预期。
若名称必须经过一段解释才能理解，应重新命名。
```

禁止：

```text
只强调气势、等级或抽象概念，却不体现对象类型。
生造难懂词、过度文艺词或翻译腔组合。
为了企业品牌感，强行在所有产品名称中加入企业专属字词。
中文名称读起来不像中文日常产品、设施或装备名称。
```

当前影响：

```text
轻型动力装甲尚未正式命名。
此前提出的“越阈轻甲”不自动定稿，需要按本规范重新讨论。
```

## 四、核心胜利与节点规则

```text
控制同一系统域的3座在线节点，立即获得最高权限并获胜。
每座节点最多有3个部署实体。
```

计入节点部署位：

```text
游侠。
独立部署的节点装备。
独立部署的通用装备。
```

不占节点部署位：

```text
由游侠携带的游侠装备。
由游侠携带的通用装备。
```

---

## 五、装备分类

```text
节点装备：只能独立部署，占用1个节点部署位。
游侠装备：只能由游侠携带，不占节点部署位。
通用装备：可以独立部署，也可以由游侠携带。
每个游侠最多携带1件装备。
```

携带装备：

```text
最终ATK = 游侠ATK + 携带装备ATK。
最终HP = 游侠HP + 携带装备HP。
HP为0或卡面明确不增加HP的装备除外。
```

战斗中，游侠与携带装备视为1个战斗单位；装备不单独攻击或承受普通战斗伤害。

## 六、通用装备转换与普通移动

```text
通用装备可在同一节点内免费切换独立部署与游侠携带状态。
转换不视为打出、部署、移动或重新上场，不触发技能。
移动1个己方部署实体当前支付1电荷。
游侠携带的装备随游侠移动，不额外付费。
```

## 十一、驱离枪

```text
编号：G-S03。
卡名：驱离枪 / Displacement Gun。
装备分类：游侠装备。
唯一：否。
复制数：2。
费用：3。
ATK 2 / HP 0。
技能名：定向驱离。
制造商：阈锋军械 / TA。
技术支持：维相科技 / DPT。
```

技能：

```text
此装备给游侠装备时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

规则边界：

```text
驱离枪不能直接部署到节点。
为携带者提供+2 ATK，不增加HP。
没有合法目标时仍可装备，但不执行定向驱离。
同节点内转交或重新调整驱离枪不会重新触发技能。
```
