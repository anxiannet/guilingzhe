# 归零者｜游侠代号命名规范

> 文件定位：本文件用于记录 `归零者 / ZERO ACCESS` 游侠卡名的中文代号命名规范。
>
> 本文件是游侠中文代号命名的主要依据，用于同步 `04-卡表.md`、`12-公共牌编号与版本记录.md` 和 `assets/version-records/operators/`。

---

## 一、核心结论

```text
游侠名称不参考英文名称。
游侠名称不参考星级。
游侠名称不绑定装备等级。
游侠名称根据技能气质、人物外号、装备绰号和行动代号综合命名。
```

关键判断：

```text
技能是命名依据，但卡名不是技能说明。
卡名要像小队外号、街头绰号、装备绰号或行动代号。
```

---

## 二、为什么不按星级命名

```text
游侠星级代表当前装备状态，而不是永久身份。
同一名游侠可能因为装备变化提升或降低星级。
因此游侠名称不能绑定“底层、核心、高危、Root级、重装、堡垒”等固定等级印象。
```

允许：

```text
大兵
扳手
黄毛
毒牙
独眼
鹰眼
算盘
喇叭
```

不建议：

```text
一星侦察员
底层游侠
核心密钥官
Root级破壁者
四星重装兵
五级机甲手
```

---

## 三、为什么不按英文名翻译

```text
中文卡名不是英文卡名的翻译稿。
英文名可以保留英文桌游习惯。
中文名必须像中文玩家会喊出来的代号。
```

禁止：

```text
为了 Trace Runner 硬译成跑线侦察员。
为了 Root Breaker 硬译成根权限破壁者。
为了 Access Recoil 硬译成权限吹回者。
```

允许：

```text
英文名继续保留功能感或科幻感。
中文名改成更自然的外号与代号。
```

## 四、命名来源类型

游侠代号可以来自以下来源。

### 1. 人物外号

适合表现街头身份、小队称呼、玩家桌上喊名。

```text
大兵
黄毛
独眼
老炮
捞仔
中间人
扒手
赌徒
```

### 2. 装备 / 工具绰号

适合表现技能动作或角色惯用工具。

```text
扳手
喇叭
算盘
铁钩
弹簧
雾弹
```

### 3. 动物 / 感官代号

适合表现侦察、速度、危险、狡黠等气质。

```text
鹰眼
猎豹
白狐
毒牙
乌鸦
```

### 4. 行动气质代号

适合表现回收、反制、延续、清理等抽象技能。

```text
回魂
续火
逆鳞
沉石
清道夫
```

## 五、命名流程

每张游侠命名按以下顺序处理：

```text
1. 读取技能效果。
2. 提炼技能行为：查看、抽弃、偷看、喊名、压底、洗牌、处理、吹回、回库、拉人、送点、抢点、清场。
3. 判断技能气质：侦察、周转、偷窃、呼应、压制、扰乱、点杀、反弹、复归、牵线、突进、扫尾。
4. 选择适合的人物外号、装备绰号、动物代号或行动代号。
5. 检查是否像玩家会直接喊出来的名字。
6. 检查是否没有绑定英文名、星级和装备等级。
```

---

## 六、合格标准

合格游侠代号应满足：

```text
短。
有识别度。
有街头感。
有行动气质。
能被玩家直接喊出来。
能从技能气质上解释。
不依赖英文名。
不依赖星级。
不依赖装备等级。
不像规则说明。
不像职业说明。
不像翻译腔。
```

---

## 七、禁止方向

```text
不要所有卡都动物化。
不要所有卡都叫“XX客”。
不要所有卡都叫“XX手”。
不要把技能直接写成卡名。
不要用星级、装备等级或强弱印象命名。
不要为了英文名对应而硬译。
不要使用论文术语、软件术语或说明书式名字。
不要使用过长的名词短语。
不要把游侠命名成正规军军衔或官方探员职位。
```

错误示例：

```text
探点客
倒手客
洗库客
送点客
一星侦察员
Root级破壁者
反向路由操作员
访问反冲执行者
```

正确方向示例：

```text
鹰眼
黄毛
独眼
扒手
喇叭
沉石
雾弹
赌徒
毒牙
弹簧
算盘
回魂
扳手
大兵
门神
猎豹
中间人
清道夫
```
