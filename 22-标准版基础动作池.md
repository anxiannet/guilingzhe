# 归零者｜标准版基础动作池

> 文件定位：本文件记录《归零者 / ZERO ACCESS》标准版中游侠与物品卡技能使用的基础动作、作用对象及合法组合，并区分总规则行动、标准版手牌技能与扩展包内容的边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、当前结论

```text
底层基础动作统一为4类：
1. 查看。
2. 获得。
3. 调整。
4. 消除。
```

对应“增、删、改、查”：

```text
增：获得。
删：消除。
改：调整。
查：查看。
```

抽牌、抽取、给予、移动、返回、置顶、置底、排序、重洗、消灭、销毁和弃置，不再作为独立的底层基础动作，而作为4类基础动作的具体执行方式或卡面表达。

---

## 二、作用对象判定标准

```text
作用对象必须能够被“查看、获得、调整、消除”中的至少一个动作直接处理。
```

不能被四个基础动作直接处理的内容，不算作用对象，只能作为：

```text
执行者。
接收者。
控制者。
条件。
归属标签。
区域说明。
状态限制。
选择方式。
```

因此，下列内容不属于作用对象：

```text
玩家。
你。
对手。
任意玩家。
控制者。
所有者。
发动者。
接收者。
```

下列内容也不属于独立作用对象：

```text
己方。
对手。
任意。
随机。
最多。
全部。
同一节点。
同一系统域。
在线。
离线。
独立部署。
被携带。
```

这些分别用于描述归属、数量、位置、状态或选择方式。

## 三、技能结构

每个技能由以下要素组成：

```text
基础动作
+ 作用对象
+ 对象归属
+ 数量
+ 来源区域
+ 目标区域或结果
+ 选择方式
+ 时机与限制
```

示例：

```text
从1名对手的手牌中随机获得1张牌。
```

拆解：

```text
基础动作：获得。
作用对象：手牌中的卡牌。
对象归属：对手。
数量：1张。
来源区域：对手手牌。
目标区域：你的手牌。
选择方式：随机。
接收者：你。
```

玩家不是作用对象，只用于限定谁的手牌、谁获得卡牌、谁控制实体或谁执行技能。

## 四、四类基础动作定义

### 查看

```text
获取作用对象的信息，但不改变其位置、区域、顺序、状态、归属或控制关系。
```

常见卡面表达：

```text
查看手牌。
查看手牌牌库顶。
查看公共弃牌区。
查看节点。
```

### 获得

```text
使作用对象进入指定接收者的持有范围、控制范围或资源范围。
```

常见卡面表达：

```text
抽牌。
从手牌中随机抽取卡牌。
从公共弃牌区取回卡牌。
获得控制权。
获得权限电荷。
```

规则口径：

```text
“抽牌”属于从手牌牌库顶获得卡牌。
“抽取”属于按随机方式获得目标。
“给予”不作为独立动作；从接收者角度统一视为获得。
```

### 调整

```text
改变作用对象的位置、所在区域、顺序、状态、部署方式或控制关系，但不使其进入公共弃牌区。
```

常见卡面表达：

```text
移动。
重新部署。
返回手牌。
返回牌库。
置顶。
置底。
排序。
重洗。
转移。
上线。
离线。
改变装备部署方式。
```

规则口径：

```text
移动属于位置调整。
返回属于区域调整。
置顶、置底和排序属于顺序调整。
重洗属于随机顺序调整。
控制权转移属于控制关系调整。
```

### 消除

```text
使作用对象离开当前有效区域并进入公共弃牌区。
```

按对象使用不同卡面词：

```text
游侠：消灭。
装备：销毁。
手牌中的卡牌：弃置。
底层基础动作统一为“消除”。
```

游侠被消除时，其携带装备同时进入公共弃牌区，除非技能另有说明。

## 五、固定作用对象序号

### 1. 卡牌

```text
定位：上位对象。
```

用于描述：

```text
手牌中的卡牌。
手牌牌库中的卡牌。
公共弃牌区中的卡牌。
手牌牌库顶的卡牌。
```

卡牌可被查看、获得、调整或消除。

但“卡牌”本身过于宽泛，不直接作为标准版技能对象使用。实际技能必须附带区域、类型或状态限定。

### 2. 手牌

```text
定位：区域或卡牌集合对象。
版本：标准版手牌技能对象。
```

手牌整体可以被查看或调整；其中的卡牌可以被获得、调整或消除。

### 3. 手牌牌库

```text
定位：区域或卡牌集合对象。
版本：标准版手牌技能对象。
```

手牌牌库可以被查看或调整，例如排序、置顶、置底与重洗；其中的卡牌可以被获得。

### 4. 公共弃牌区

```text
定位：区域或卡牌集合对象。
版本：标准版手牌技能对象。
```

公共弃牌区可以被查看或调整；其中的卡牌可以被获得或调整。

### 5. 游侠

```text
定位：场上实体对象。
版本：标准版手牌技能对象。
```

游侠可以被查看、调整或消除。

若技能获得或调整游侠的控制权，直接作用对象为9号“控制权”。

### 6. 装备

```text
定位：场上实体对象。
版本：标准版手牌技能对象。
```

装备可以被查看、调整或消除。

若技能获得或调整装备的控制权，直接作用对象为9号“控制权”。

以下仅是装备类型或状态限定，不另列一级作用对象：

```text
节点装备。
游侠装备。
通用装备。
独立部署。
被游侠携带。
```

### 7. 一次性物品

```text
定位：卡牌子类型对象。
版本：标准版手牌技能对象。
```

一次性物品可作为手牌或公共弃牌区中的具体卡牌类型，被查看、获得、调整或消除。

### 8. 节点

```text
定位：场上规则实体对象。
版本：扩展包对象。
```

节点可以被查看或调整，包括在线、离线、保护、回流等状态或区域变化。

若技能获得或调整节点控制权，直接作用对象为9号“控制权”。

节点通常不被消除。

### 9. 控制权

```text
定位：关系对象。
版本：扩展包对象。
当前限制：只能由总规则改变。
```

控制权可以被获得或调整，具体包括：

```text
游侠控制权。
装备控制权。
节点控制权。
```

当前不开放给标准版手牌技能，也不作为常规卡牌技能对象使用。现阶段控制权只依据部署、攻击占领、节点清空及其他总规则发生变化。

已有卡牌或节点技能若明确改变控制权，必须作为单独例外登记，不能反向开放控制权基础组合。

### 10. 权限电荷

```text
定位：资源对象。
版本：扩展包对象。
```

权限电荷可以被获得或调整数量。

支付电荷属于总规则中的资源支出，不属于消除。权限电荷不会进入公共弃牌区，因此不能使用“消除权限电荷”的口径。

## 六、版本分层

```text
1号：上位对象，不直接作为标准版技能对象。
2至7号：标准版手牌技能对象。
8至10号：扩展包对象。
9号：目前只能由总规则改变。
```

对象进入标准版或扩展包，只表示可以进入对应版本的“动作 × 对象”组合审核，不代表查看、获得、调整、消除四种动作全部自动开放。

## 七、标准版手牌技能的当前合法范围

### 查看

当前允许：

```text
查看1名对手的手牌。
查看手牌牌库顶若干张。
```

### 获得

当前允许：

```text
指定玩家从手牌牌库顶获得卡牌。
从1名对手的手牌中随机获得卡牌。
```

以下仍需单独审核后才能进入普通手牌技能：

```text
从公共弃牌区获得卡牌。
```

以下属于扩展包对象，不进入标准版手牌技能：

```text
获得场上实体控制权。
获得节点控制权。
获得权限电荷。
```

### 调整

当前允许：

```text
调整手牌牌库顶的顺序。
将手牌牌库顶的牌置顶或置底。
重洗手牌牌库。
将场上游侠或装备返回手牌。
将场上游侠或装备返回手牌牌库。
```

当前明确例外：

```text
驱离枪可以定向调整1个对手游侠或装备所在的节点。
```

普通移动己方部署实体属于总规则行动，不因“调整”成为基础动作而自动开放为所有手牌技能。

### 消除

当前允许：

```text
消除1个对手场上游侠。
消除1个对手场上装备。
```

卡面分别写作：

```text
消灭1个对手场上游侠。
销毁1个对手场上装备。
```

## 八、装备与位置调整边界

```text
节点装备可以作为独立部署实体重新部署。
游侠装备不能脱离携带者单独普通移动。
通用装备独立部署时可以重新部署；被游侠携带时随游侠移动。
同节点内改变通用装备的独立部署或携带状态，属于部署方式调整，不支付普通移动费用。
```

### 普通移动与重新部署

```text
属于行动阶段通用行动。
只能处理己方部署实体。
目标只能是自己控制的另一座合法节点。
普通移动和重新部署不是卡牌技能。
```

### 定向驱离

```text
属于“调整 × 对手场上实体”的明确例外。
只能选择对手控制的游侠或装备。
目标节点必须由目标控制者控制。
不能改变目标控制权。
不能处理己方实体。
```

驱离枪当前文本：

```text
此装备给游侠装备时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

---

## 九、返回与携带装备

```text
返回手牌属于区域调整，不属于消除。
将1个场上的游侠或装备返回其控制者手牌时，目标不会进入公共弃牌区。
```

涉及携带装备时：

```text
若效果选择游侠，游侠携带的装备默认随游侠返回手牌，除非效果明确只处理游侠。
若效果选择被携带装备，可以只将该装备返回手牌。
```

## 十、当前明确禁止或未开放的普通手牌技能组合

```text
公开对手手牌。
隔离卡牌或实体。
禁用卡牌、技能或节点。
直接改变卡牌费用。
直接改变生命值。
直接造成伤害。
直接治疗。
获得或调整控制权。
直接获得权限电荷。
消除己方场上实体。
任意单体空间转运。
多目标空间移动。
```

## 十二、当前状态

```text
底层基础动作已统一为4类：查看、获得、调整、消除。
一级作用对象序号已固定为1至10。
标准版手牌技能对象为2至7号。
扩展包对象为8至10号。
9号控制权目前只能由总规则改变。
动作统计必须同时标明作用对象，不能只统计卡面动词。
卡面仍可使用抽牌、返回、移动、重洗、消灭、销毁等自然语言。
后续卡表审核应将旧“基础动作拆解”统一映射到四类动作与固定对象序号。
```
