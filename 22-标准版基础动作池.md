# 归零者｜标准版基础动作池

> 文件定位：本文件记录《归零者 / ZERO ACCESS》标准版中游侠与物品卡技能使用的基础动作、作用对象及合法组合，并区分总规则行动、标准版手牌技能与节点技能边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、当前结论

底层基础动作统一为4类：

```text
1. 查看。
2. 获得。
3. 调整。
4. 消除。
```

对应“增、删、改、查”：

```text
增：获得。
删：消除。
改：调整。
查：查看。
```

标准版手牌技能只统计“获得、调整、消除”3类有效动作，与6个标准版作用对象形成18种基本动作。

查看是合法底层动作，但不计入18种标准版有效手牌基本动作。

## 二、标准版作用对象

| 对象编号 | 作用对象 | 定位 |
|---:|---|---|
| 2 | 手牌 | 区域或卡牌集合 |
| 3 | 手牌牌库 | 区域或卡牌集合 |
| 4 | 公共弃牌区 | 区域或卡牌集合 |
| 5 | 游侠 | 场上实体 |
| 6 | 装备 | 场上实体 |
| 7 | 一次性物品 | 手牌或弃牌区中的卡牌子类型 |

以下不是独立作用对象：

```text
玩家、你、其他玩家、控制者、所有者、发动者、接收者。
己方、其他玩家、任意、随机、最多、全部。
同一节点、同一系统域、独立部署、被携带。
```

节点不再具有在线或离线状态，因此“在线”“离线”不得再作为对象状态、条件、结果或选择限制写入当前规则与卡牌文本。

## 三、技能结构

每个技能由以下要素组成：

```text
基本动作
+ 作用对象
+ 对象归属
+ 数量
+ 来源区域
+ 目标区域或结果
+ 选择方式
+ 时机与限制
```

示例：

```text
从1名其他玩家的手牌中随机获得1张牌。
```

拆解：

```text
基础动作：获得。
作用对象：手牌。
对象归属：其他玩家。
数量：1张。
来源区域：其他玩家手牌。
目标区域：你的手牌。
选择方式：随机。
接收者：你。
```

## 四、四类底层动作定义

### 查看

获取作用对象的信息，但不改变其位置、区域、顺序、状态、归属或控制关系。

常见表达：

```text
查看手牌。
查看手牌牌库顶。
查看公共弃牌区。
查看节点信息。
```

### 获得

使作用对象中的卡牌、实体或资源进入指定接收者的持有范围、控制范围或资源范围。

常见表达：

```text
抽牌。
从指定手牌中随机抽取卡牌。
从公共弃牌区取回卡牌。
使指定对象进入接收者手牌。
```

规则口径：

```text
抽牌属于从手牌牌库获得卡牌。
抽取属于按随机方式获得目标。
给予不作为独立动作；从接收者角度视为获得。
```

### 调整

改变作用对象的位置、区域、顺序、状态、部署方式或内部结构，但不使其进入公共弃牌区。

常见表达：

```text
移动。
重新部署。
返回手牌。
返回牌库。
置顶。
置底。
排序。
重洗。
转移。
改变装备部署方式。
```

### 消除

使作用对象中的卡牌或实体离开当前有效区域并进入公共弃牌区。

按对象使用不同卡面词：

```text
游侠：消灭。
装备：销毁。
手牌中的卡牌：弃置。
一次性物品：弃置或销毁。
```

游侠被消除时，其携带装备同时进入公共弃牌区，除非技能另有说明。

## 六、标准版手牌技能生成规则

```text
标准版所有游侠技能，只能由18种基本动作中的一种进行变体，或由多种基本动作组合形成。
标准版所有物品卡技能，同样遵守该规则。
卡面自然语言可以不同，但必须完整映射回基本动作。
不得新增第19种平行动作。
```

允许变体：

```text
对象归属、数量、来源区域、目标区域、选择方式、触发时机、作用范围、支付条件、使用限制。
```

不允许：

```text
把执行者、接收者、控制者或条件误写为作用对象。
用新术语包装未登记的底层动作。
以已废止的在线／离线状态制造技能变体。
```

## 七、节点与扩展对象边界

节点是场上规则实体，不属于标准版18种手牌作用对象。

节点技能可以拥有独立系统能力，但必须在卡表中明确登记，并符合城市基础设施定位。

当前节点技能可以包括：

```text
查看节点信息。
保护节点。
移动游侠或装备。
临时调整游侠或装备。
从公共弃牌区回收卡牌。
系统级实体调度。
明确登记的控制权例外。
```

明确废止：

```text
使节点在线。
使节点离线。
节点在线／离线状态变化。
以在线节点数量计算Root Access。
```

控制权仍不是标准版普通手牌技能对象。已有节点技能若改变控制权，必须作为单独例外登记。

权限电荷属于资源对象。支付电荷是总规则资源支出，不属于消除。

## 八、装备与位置调整边界

```text
节点装备可以作为独立部署实体重新部署。
游侠装备不能脱离携带者单独普通移动。
通用装备独立部署时可以重新部署；被游侠携带时随游侠移动。
同节点内改变通用装备的独立部署或携带状态，属于部署方式调整。
```

普通移动与重新部署属于行动阶段通用行动，不是卡牌技能。

定向驱离属于：

```text
10. 调整游侠。
11. 调整装备。
```

目标只能移动到其控制者控制且可容纳该实体的另一座节点，不再引用在线或离线状态。

---

## 九、返回与携带装备

```text
返回手牌属于区域调整，不属于消除。
将场上的游侠或装备返回其控制者手牌时，目标不会进入公共弃牌区。
若效果选择游侠，游侠携带的装备默认随游侠返回手牌，除非效果明确只处理游侠。
若效果选择被携带装备，可以只将该装备返回手牌。
```

## 十一、当前状态

```text
底层基础动作：查看、获得、调整、消除。
标准版有效手牌动作：获得、调整、消除。
标准版作用对象：手牌、手牌牌库、公共弃牌区、游侠、装备、一次性物品。
标准版手牌技能空间：18种基本动作。
节点在线、离线机制已移除。
后续不得生成或保留依赖该状态的技能、测试字段或卡面文案。
```
