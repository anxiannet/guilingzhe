# 归零者｜标准版基础动作池

> 文件定位：本文件记录《归零者 / ZERO ACCESS》标准版手牌技能的基础动作，并区分全局战争节点技能与扩展包动作。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、模式定位

```text
混战 = 全局战争。
对战 = 单个节点的局部战争。
```

### 全局战争

```text
2至4人使用公共手牌牌库。
战场包含多座节点。
使用节点卡、节点牌库、中央节点区和节点技能。
目标是争夺系统域并获得 Root Access。
```

### 局部战争

```text
玩家各自使用独立牌组。
战场固定为1座唯一节点。
唯一节点不是节点卡，没有节点技能。
不使用节点牌库或中央节点区。
节点装备可以加入牌组并部署到唯一节点。
```

标准版手牌技能不得造成混牌。全局战争中的标准版节点技能可以造成跨玩家控制权变化。

## 二、底层动作

```text
查看。
获得。
调整。
消除。
```

查看属于信息步骤，不计入标准版手牌动作覆盖率。

消耗品是卡牌类型限制，不是独立动作对象。

## 三、标准版手牌9种有效动作

### 获得类

```text
1. 获得手牌牌库。
2. 获得公共弃牌区。
```

### 调整类

```text
3. 调整手牌。
4. 调整手牌牌库。
5. 调整公共弃牌区。
6. 调整游侠。
7. 调整装备。
```

### 消除类

```text
8. 消除游侠。
9. 消除装备。
```

## 四、动作定义

### 1. 获得手牌牌库

```text
全局战争：从公共手牌牌库获得卡牌。
局部战争：从自己的牌库获得卡牌。
```

### 2. 获得公共弃牌区

将公共弃牌区中的合法目标加入手牌。

```text
局部战争中默认只能获得自己所有的卡牌。
标准版手牌技能不得取得其他玩家所有的卡牌。
```

### 3. 调整手牌

改变自己手牌中卡牌的位置，或将其移至非弃牌区域。

```text
不得取得其他玩家手牌。
不得将手牌直接置入弃牌区。
```

### 4. 调整手牌牌库

改变牌库中的顺序或结构，但卡牌仍留在牌库中。

```text
包括置顶、置底、排序和重洗。
```

### 5. 调整公共弃牌区

将弃牌区中的卡牌移至非手牌区域，但不加入任何玩家手牌。

### 6. 调整游侠

改变游侠的位置、状态、数值或所在区域，但不将其置入弃牌区，也不改变控制权。

```text
全局战争可以包含跨节点移动。
局部战争只有唯一节点，不存在跨节点移动。
```

### 7. 调整装备

改变装备的位置、状态、部署方式、数值或所在区域，但不将其置入弃牌区，也不改变控制权。

```text
局部战争中，节点装备和独立部署的通用装备均部署到唯一节点。
```

### 8. 消除游侠

将场上游侠置入弃牌区。卡面使用“消灭”。

### 9. 消除装备

将场上装备置入弃牌区。卡面使用“销毁”。

## 五、扩展包手牌动作

```text
获得其他玩家手牌。
获得场上游侠。
获得场上装备。
消除手牌。
消除手牌牌库。
```

前三项会造成手牌技能混牌；后两项会加速有限牌库中的卡牌进入弃牌区。

扩展包动作不占用标准版手牌9种编号，也不计入标准版动作覆盖率。

---

## 六、全局战争节点技能

节点卡只用于全局战争。

节点技能可以：

```text
取得其他玩家游侠或装备的控制权。
将其他玩家控制的实体移动到发动者控制的节点。
在卡面明确时，将其他玩家卡牌加入指定玩家手牌。
```

节点技能使用独立节点动作池，不计入标准版手牌动作覆盖率。

局部战争不使用节点卡，因此不会触发节点技能或节点混牌效果。

---

## 七、查看的定位

```text
查看只获取隐藏信息，不改变游戏状态。
查看不是标准版手牌基础动作。
查看后排序、置顶或置底，映射为4. 调整手牌牌库。
查看公共弃牌区、场上卡牌与公开节点信息属于玩家基础信息权限。
```

---

## 八、模式兼容

标准版手牌卡原则上应兼容全局战争与局部战争。

依赖以下内容的手牌卡必须单独审核：

```text
另一座节点。
跨节点移动。
节点控制权。
节点回流。
系统域。
节点技能。
```

当前重点兼容审核：

```text
驱离枪。
轻动力甲。
```
