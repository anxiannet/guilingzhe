# 归零者｜标准版基础动作池

> 文件定位：本文件记录《归零者 / ZERO ACCESS》标准版中游侠与物品卡技能使用的基础动作、作用对象及合法组合，并区分总规则行动、标准版手牌技能、扩展包动作与节点技能边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、当前结论

底层基础动作统一为4类：

```text
1. 查看。
2. 获得。
3. 调整。
4. 消除。
```

对应“增、删、改、查”：

```text
增：获得。
删：消除。
改：调整。
查：查看。
```

标准版手牌技能当前登记13种有效基本动作。

查看是合法底层动作，但不计入13种标准版有效手牌基本动作。

13种动作不是由“动作 × 对象”机械排列产生，而是只保留具有独立规则结果且适合标准版牌库规模的组合。

“消除手牌牌库”具有独立规则意义，但会直接减少公共牌库牌数，当前指定为扩展包动作，不进入标准版13种动作池。

---

## 二、规则对象与限制条件

当前具有独立规则意义的区域或实体对象：

```text
其他玩家手牌。
手牌牌库。
公共弃牌区。
场上游侠。
场上装备。
```

“消耗品”是卡牌类型限制，不是独立基础动作对象。

例如：

```text
从公共弃牌区获得1张消耗品。
```

拆解为：

```text
基础动作：3. 获得公共弃牌区。
卡牌类型限制：消耗品。
```

以下也不是独立作用对象：

```text
玩家、你、其他玩家、控制者、所有者、发动者、接收者。
己方、任意、随机、最多、全部。
同一节点、同一系统域、独立部署、被携带。
游侠、装备或消耗品等卡牌类型筛选条件，在区域动作中只作为目标限制。
```

节点不再具有在线或离线状态，因此“在线”“离线”不得作为对象状态、条件、结果或选择限制。

## 三、技能结构

每个技能由以下要素组成：

```text
基本动作
+ 来源区域或场上对象
+ 对象归属
+ 卡牌类型限制
+ 数量
+ 目标区域或结果
+ 选择方式
+ 时机与限制
```

示例：

```text
从1名其他玩家的手牌中随机获得1张牌。
```

拆解：

```text
基础动作：1. 获得其他玩家手牌。
来源：其他玩家手牌。
数量：1张。
目标区域：你的手牌。
选择方式：随机。
```

## 四、四类底层动作定义

### 查看

获取信息，但不改变卡牌或实体的位置、区域、顺序、状态、归属或控制关系。

常见表达：

```text
查看手牌。
查看手牌牌库顶。
查看公共弃牌区。
查看节点信息。
```

### 获得

使卡牌或场上实体进入指定玩家的手牌或控制范围。

规则口径：

```text
抽牌属于从手牌牌库获得卡牌。
从其他玩家手牌取得卡牌属于获得其他玩家手牌。
从公共弃牌区取回卡牌属于获得公共弃牌区。
取得场上游侠或装备属于获得场上游侠或获得场上装备。
卡牌类型筛选不构成新的基础动作。
```

### 调整

改变卡牌或实体的位置、区域、顺序、状态、部署方式或内部结构，但不使其进入公共弃牌区。

常见表达：

```text
移动。
重新部署。
返回手牌。
返回牌库。
置顶。
置底。
排序。
重洗。
转移。
交换手牌。
改变装备部署方式。
```

### 消除

使手牌中的卡牌或场上实体进入公共弃牌区。

按对象使用不同卡面词：

```text
游侠：消灭。
装备：销毁。
手牌中的卡牌：弃置。
```

游侠被消除时，其携带装备同时进入公共弃牌区，除非技能另有说明。

公共弃牌区中的卡牌已经处于消除结果区域，不能再次执行“消除公共弃牌区”。

将手牌牌库中的卡牌直接置入公共弃牌区属于扩展包动作，不属于标准版消除动作。

## 五、标准版13种手牌基本动作

### 获得类

```text
1. 获得其他玩家手牌。

3. 获得公共弃牌区。
4. 获得场上游侠。
5. 获得场上装备。
```

### 调整类

```text
6. 调整手牌。

8. 调整公共弃牌区。
9. 调整游侠。
10. 调整装备。
```

### 消除类

```text
11. 消除手牌。
12. 消除游侠。
13. 消除装备。
```

## 六、标准版动作定义与边界

### 1. 获得其他玩家手牌

将其他玩家手牌中的卡牌加入你的手牌。

```text
卡面示例：从1名其他玩家的手牌中随机获得1张牌。
```

### 2. 获得手牌牌库

从公共手牌牌库获得卡牌。

```text
卡面示例：从手牌牌库顶抽1张牌。
```

### 3. 获得公共弃牌区

将公共弃牌区中的合法目标加入手牌。

```text
卡牌必须明确目标类型。
取回游侠必须具有复苏、备份或重构等合法解释。
取得装备可解释为回收或修复。
取得消耗品仍映射为本动作，消耗品只是目标限制。
```

### 4. 获得场上游侠

取得场上其他玩家游侠的控制权，或将其加入指定玩家手牌。

```text
卡面必须明确是获得控制权还是加入手牌。
不得只写“获得1个游侠”。
```

### 5. 获得场上装备

取得场上其他玩家装备的控制权，或将其加入指定玩家手牌。

```text
卡面必须明确是否可以选择独立部署装备或被携带装备。
卡面必须明确获得控制权还是加入手牌。
```

### 6. 调整手牌

改变手牌中卡牌的归属，或将其移至非弃牌区域。

```text
交换、给予、放回牌库均可映射为调整手牌。
弃置手牌不属于本动作，属于11. 消除手牌。
```

### 7. 调整手牌牌库

改变手牌牌库中的顺序或结构，但卡牌仍留在牌库中。

```text
置顶、置底、排序、重洗均属于本动作。
```

### 8. 调整公共弃牌区

将公共弃牌区中的卡牌移至非手牌区域，但不加入任何玩家手牌。

```text
典型结果：置于手牌牌库顶、置于牌库底、洗回牌库。
单纯调整弃牌区顺序没有规则意义。
```

### 9. 调整游侠

改变场上游侠的位置、状态、数值或所在区域，但不将其置入公共弃牌区。

```text
包括移动、返回手牌、返回牌库、临时强化或削弱。
```

### 10. 调整装备

改变场上装备的位置、状态、部署方式、数值或所在区域，但不将其置入公共弃牌区。

```text
包括移动、重新部署、返回手牌、返回牌库和改变携带关系。
```

### 11. 消除手牌

将手牌中的卡牌置入公共弃牌区。

```text
卡面优先使用“弃置”。
消耗品可作为目标类型限制，但仍映射为本动作。
```

### 12. 消除游侠

将场上游侠置入公共弃牌区。

```text
卡面使用“消灭”。
```

### 13. 消除装备

将场上装备置入公共弃牌区。

```text
卡面使用“销毁”。
```

## 七、扩展包动作

### 消除手牌牌库

将手牌牌库中的卡牌直接置入公共弃牌区。

```text
该动作具有独立规则意义。
该动作会直接减少公共手牌牌库牌数。
由于标准版牌库规模有限，不进入标准版动作池。
仅允许扩展包卡牌或扩展规则使用。
使用前必须补齐牌库耗尽、空牌库抽牌和弃牌区是否重洗等规则。
```

扩展包技能映射统一写为：

```text
扩展动作：消除手牌牌库。
```

不得占用标准版13种动作编号。

## 八、已删除动作

以下旧动作已删除：

```text
获得消耗品。
调整消耗品。
消除公共弃牌区。
消除消耗品。
```

删除原因：

```text
获得消耗品、调整消耗品、消除消耗品中的“消耗品”只是卡牌类型限制，应映射到对应来源区域动作。
公共弃牌区已经是消除结果区域，没有合法的再次消除结果；若新增移出游戏区，会引入额外区域和规则负担。
```

旧名称同步废止：

```text
获得手牌 → 获得其他玩家手牌。
获得游侠 → 获得场上游侠。
获得装备 → 获得场上装备。
```

## 九、标准版手牌技能生成规则

```text
标准版所有游侠与物品卡技能，只能由13种基本动作中的一种进行变体，或由多种基本动作组合形成。
卡面自然语言可以不同，但必须完整映射回基本动作。
不得新增第14种标准版平行动作，除非先修改本文件并完成规则审核。
卡牌类型限制不单独建立基础动作。
扩展包动作不得计入标准版动作覆盖率。
```

允许变体：

```text
对象归属、数量、来源区域、目标区域、卡牌类型限制、选择方式、触发时机、作用范围、支付条件、使用限制。
```

## 十、节点与扩展对象边界

节点是场上规则实体，不属于标准版13种手牌基本动作对象。

节点技能可以拥有独立系统能力，但必须在卡表中明确登记，并符合城市基础设施定位。

当前节点技能可以包括：

```text
查看节点信息。
保护节点。
移动游侠或装备。
临时调整游侠或装备。
从公共弃牌区回收卡牌。
系统级实体调度。
明确登记的控制权例外。
```

控制权不是标准版普通手牌技能的通用独立对象。改变控制权的卡牌必须明确映射为“获得场上游侠”或“获得场上装备”，或作为节点技能例外登记。

权限电荷属于资源对象。支付电荷是总规则资源支出，不属于消除。

---

## 十一、装备与位置调整边界

```text
节点装备可以作为独立部署实体重新部署。
游侠装备不能脱离携带者单独普通移动。
通用装备独立部署时可以重新部署；被游侠携带时随游侠移动。
同节点内改变通用装备的独立部署或携带状态，属于调整装备。
```

普通移动与重新部署属于行动阶段通用行动，不是卡牌技能。

当前标准版旧映射换算：

```text
原10. 调整游侠 → 现9. 调整游侠。
原11. 调整装备 → 现10. 调整装备。
原13. 消除手牌 → 现11. 消除手牌。
原14. 消除手牌牌库 → 扩展动作：消除手牌牌库。
原16. 消除游侠 → 现12. 消除游侠。
原17. 消除装备 → 现13. 消除装备。
```

---

## 十二、返回与携带装备

```text
返回手牌属于调整，不属于消除。
若效果选择游侠，游侠携带的装备默认随游侠返回手牌，除非效果明确只处理游侠。
若效果选择被携带装备，可以只处理该装备。
```
