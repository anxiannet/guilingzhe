# 归零者｜标准版基础动作池

> 文件定位：本文件记录《归零者 / ZERO ACCESS》标准版中游侠与物品卡技能可使用的基础动作池，并区分总规则行动、手牌技能与节点技能的动作边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、当前结论

```text
普通移动己方实体是03-标准版规则.md中的通用行动，不是卡牌技能。
玩家可以支付与派出1个实体攻击节点相同的电荷，在自己的节点之间移动1个己方游侠或装备。
普通移动的存在不会自动把“移动实体”加入游侠或物品卡技能基础动作池。
标准版游侠与物品卡技能只能使用本文件中的基础动作或明确例外。
```

---

## 二、三类移动的边界

### 1. 普通移动

来源：

```text
03-标准版规则.md。
```

口径：

```text
属于行动阶段的通用行动。
只能移动己方实体。
只能移动到自己控制的另一座节点。
每移动1个实体，支付与派出1个实体攻击节点相同的电荷；当前为1电荷。
不属于技能，不触发入场技能。
```

### 2. 技能驱离

当前明确例外：

```text
武器装备可以在上场时强制驱离1个对手游侠或装备。
```

限制：

```text
只能移动对手实体。
目标节点必须由该实体控制者控制。
不能改变控制权。
不能移动己方实体。
```

### 3. 节点技能移动

```text
节点技能可以使用更宽的移动动作。
节点技能可以按卡面明确允许移动己方、对手或任意实体。
节点技能也可以按技能文本重映射控制权。
具体目标、费用和限制以节点卡表为准。
```

## 三、标准版手牌技能基础动作池

当前主池：

```text
1. 抽牌。
2. 查看牌库顶。
3. 调整牌库顶。
4. 返回牌库。
5. 查看手牌。
6. 返回手牌。
7. 消灭。
```

备选动作：

```text
重洗牌库。
```

说明：

```text
普通移动不是手牌技能基础动作。
游侠和普通装备不能仅因为总规则允许普通移动，就获得移动其他实体的入场技能。
```

---

## 四、基础动作定义

### 抽牌

```text
从手牌牌库抽牌，加入指定玩家手牌。
```

### 查看牌库顶

```text
查看指定手牌牌库顶部的牌，不改变牌库顺序。
```

### 调整牌库顶

```text
改变指定手牌牌库顶部牌的顺序或位置。
```

### 返回牌库

```text
将目标牌放回其控制者或拥有者的手牌牌库顶、牌库底，或洗入牌库。
```

限制：

```text
不保留己方场上游侠返回己方牌库。
不保留己方场上装备返回己方牌库。
```

### 查看手牌

```text
查看指定玩家的手牌。
```

### 返回手牌

```text
将1个场上的游侠或装备移回其控制者手牌。
```

限制：

```text
可部署实体的入场技能不得将己方场上实体返回手牌。
己方场上实体回手只允许出现在一次性物品上。
返回手牌不是移动、伤害、消灭或销毁。
```

### 消灭

```text
将1个场上的对手游侠或装备移入公共弃牌区。
```

## 五、武器驱离例外

动作名称：

```text
强制驱离。
```

动作定义：

```text
将1个对手场上的游侠或装备，从当前节点移动到其控制者控制的另一座合法节点。
```

允许载体：

```text
武器类装备的入场技能。
节点技能。
```

标准限制：

```text
只能选择对手控制的游侠或装备。
不能选择己方实体。
目标节点必须由该实体的控制者控制。
目标节点可以在线或离线。
目标节点不得超过3个实体。
强制驱离不改变目标控制者。
强制驱离不视为部署，不触发入场技能。
若来源节点清空，立即结算节点回流。
没有合法目标节点时，装备仍可正常上场，但不执行驱离。
```

当前标准版例子：

```text
驱离枪：此装备上场时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

## 六、当前明确移除的手牌技能动作

```text
弃牌。
公开手牌。
隔离。
禁用。
费用变化。
攻击变化。
生命变化。
伤害。
治疗。
获得电荷。
己方返回牌库。
己方消灭。
可部署实体回手己方场上实体。
任意单体空间转运。
多目标空间移动。
手牌技能控制权重映射。
```

说明：

```text
原一次性物品“相位转运卡”及其任意单体空间移动例外废止。
普通移动属于总规则行动，不恢复该手牌技能例外。
```

## 七、规则维护口径

```text
普通移动规则以03-标准版规则.md为准。
具体卡牌技能以04-测试版卡表.md或05-标准版卡表.md为准。
节点技能的具体目标、费用与限制以当前节点卡表为准。
新增移动类手牌技能前必须先修改并通过本文件审核。
节点技能使用更宽动作集合，不自动扩展手牌技能池。
```
