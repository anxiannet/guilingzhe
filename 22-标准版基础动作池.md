# 归零者｜标准版基础动作池

> 文件定位：本文件记录《归零者 / ZERO ACCESS》标准版中游侠与物品卡技能可使用的基础动作池，并说明节点技能与手牌技能的动作边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、当前结论

标准版游侠与物品卡技能必须遵守：

```text
标准版游侠技能，只能由本文件中的基础动作进行变体或组合形成。
标准版物品卡技能，只能由本文件中的基础动作进行变体或组合形成。
不在基础动作池或明确例外中的效果，不进入标准版游侠与物品卡技能。
可部署实体的入场技能不得将己方场上实体返回手牌。
己方场上实体回手效果只允许出现在一次性物品上。
场上实体的空间移动只允许出现在一次性物品与节点技能上。
```

节点技能动作边界：

```text
节点技能可以使用手牌基础动作池中的动作。
节点技能也可以使用节点专属动作。
手牌技能不能因为节点技能使用了某个动作，就自动获得该动作使用权。
```

---

## 二、基础动作池与节点动作池边界

```text
基础动作池主要限制游侠与物品卡技能。
节点技能可以使用抽牌、查看牌库顶、调整牌库顶、返回牌库、查看手牌、返回手牌、消灭与重洗牌库。
节点技能也可以使用查看节点、上线、离线、保护、移动、临时强化、从弃牌区回收与控制权重映射等节点专属动作。
游侠和装备不能通过入场技能移动场上实体。
一次性物品可以使用“单体空间移动”例外，但必须明确目标、目标节点与合法容纳条件。
```

当前节点专属动作：

```text
查看节点。
使节点在线。
使节点离线。
保护节点。
移动场上实体。
临时强化场上实体。
从弃牌区回收游侠或装备。
重映射实体控制权。
```

## 三、标准版基础动作筛选原则

```text
1. 规则直观。
2. 易于自动测试。
3. 不依赖真人反应。
4. 不制造长期状态记录。
5. 不直接围绕费用、ATK、HP持续修改。
6. 不与节点技能创建平行体系。
7. 后续技能可通过目标、数量、区域、次数和时机限制扩展。
8. 不允许可部署实体通过入场技能回手己方场上实体。
9. 空间移动作为窄例外，只允许一次性物品进行单体移动。
```

以下动作不进入常规游侠与物品卡基础池：

```text
获得电荷。
费用减少。
攻击力增加或减少。
不能攻击。
生命值增加或恢复。
造成伤害。
上线或离线节点。
保护节点。
临时强化。
从弃牌区直接部署实体。
```

说明：

```text
上述限制不限制节点技能。
一次性物品的单体空间移动属于明确例外，不代表“移动”成为所有手牌卡的通用动作。
```

## 四、当前基础动作主池

当前标准版基础动作主池为：

```text
1. 抽牌。
2. 查看牌库顶。
3. 调整牌库顶。
4. 返回牌库。
5. 查看手牌。
6. 返回手牌。
7. 消灭。
```

备选动作：

```text
重洗牌库。
```

一次性物品专属例外：

```text
8. 单体空间移动。
```

## 五、基础动作定义

### 1. 抽牌

```text
从手牌牌库抽牌，加入指定玩家手牌。
```

常见变体：

```text
抽1张牌。
抽2张牌。
选择1名玩家，其抽1张牌。
```

### 2. 查看牌库顶

```text
查看指定手牌牌库顶部的牌，不改变牌库顺序。
```

### 3. 调整牌库顶

```text
改变指定手牌牌库顶部牌的顺序或位置。
```

### 4. 返回牌库

```text
将目标牌放回其控制者或拥有者的手牌牌库顶、牌库底，或洗入牌库。
```

限制：

```text
标准版手牌基础池不保留己方场上游侠返回己方牌库。
标准版手牌基础池不保留己方场上装备返回己方牌库。
```

### 5. 查看手牌

```text
查看指定玩家的手牌。
```

限制：

```text
公开手牌不进入基础动作池。
弃牌不进入基础动作池。
```

### 6. 返回手牌

```text
将1个场上的游侠或装备从当前部署状态移回其控制者手牌。
```

常见变体：

```text
将1个对手场上游侠返回其控制者手牌。
将1个对手场上装备返回其控制者手牌。
将1个己方场上游侠返回你的手牌。仅限一次性物品。
将1个己方场上装备返回你的手牌。仅限一次性物品。
```

限制：

```text
可部署实体的入场技能不得将己方场上实体返回手牌。
己方场上实体回手效果只允许出现在一次性物品上。
返回手牌不是移动、伤害或消灭。
```

### 7. 消灭

```text
将1个场上的对手游侠或装备移入公共弃牌区。
```

常见变体：

```text
消灭1个对手场上游侠。
消灭1个对手场上装备。
消灭1个对手场上游侠或装备。
```

当前不保留：

```text
消灭1个己方场上游侠。
消灭1个己方场上装备。
```

### 8. 单体空间移动（一次性物品专属例外）

```text
选择场上1个游侠或装备，将其从当前节点移动到另1座合法节点。
```

适用载体：

```text
一次性物品。
节点技能。
```

禁止载体：

```text
游侠入场技能。
装备入场技能。
装备持续技能。
```

标准限制：

```text
目标必须是场上的游侠或装备。
目标节点必须能够合法容纳该实体。
目标节点不得超过3个实体。
移动后不能使同一节点出现多个玩家控制的实体。
移动不视为部署，不触发入场技能。
移动不改变目标实体控制者，除非节点技能明确写出控制权变化。
若原节点因目标离开而没有任何实体，立即结算节点回流。
若目标移动到无主节点，该实体控制者获得该节点控制权。
技能可以明确允许选择离线节点。
```

当前标准版例子：

```text
相位转运卡：选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点，包括离线节点。
```

## 六、当前明确移除的手牌技能动作

```text
弃牌。
公开手牌。
隔离。
禁用。
费用变化。
攻击变化。
生命变化。
伤害。
治疗。
获得电荷。
己方返回牌库。
己方消灭。
可部署实体回手己方场上实体。
多目标空间移动。
控制权重映射。
```

说明：

```text
多目标空间移动与控制权重映射只允许作为节点技能。
一次性物品只获得单体空间移动例外。
```

## 七、规则维护口径

```text
具体卡牌技能以 04-测试版卡表.md 或 05-标准版卡表.md 为准。
节点技能的具体目标、费用与限制以当前节点卡表为准。
新增手牌技能前必须先检查本文件是否存在对应基础动作或明确例外。
节点技能使用更宽动作集合，不自动扩展手牌技能池。
```
