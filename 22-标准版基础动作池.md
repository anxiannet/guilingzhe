# 归零者｜标准版基础动作池

> 文件定位：本文件记录《归零者 / ZERO ACCESS》标准版中游侠与物品卡技能使用的基础动作、作用对象及合法组合，并区分总规则行动、标准版手牌技能与扩展包内容的边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、当前结论

```text
底层基础动作统一为4类：
1. 查看。
2. 获得。
3. 调整。
4. 消除。
```

对应“增、删、改、查”：

```text
增：获得。
删：消除。
改：调整。
查：查看。
```

查看主要用于读取信息，不进入标准版手牌技能的基本动作组合统计。

标准版手牌技能使用3类有效动作：

```text
获得。
调整。
消除。
```

标准版手牌技能使用6个作用对象：

```text
2. 手牌。
3. 手牌牌库。
4. 公共弃牌区。
5. 游侠。
6. 装备。
7. 一次性物品。
```

因此标准版手牌技能共有：

```text
3类有效动作 × 6个作用对象 = 18种基本动作。
```

标准版中所有游侠与物品卡的手牌技能，都必须是这18种基本动作的变体，或由其中两种及以上基本动作组合形成。

不属于这18种基本动作、也未登记为明确例外的效果，不得进入标准版手牌技能。

抽牌、抽取、给予、移动、返回、置顶、置底、排序、重洗、消灭、销毁和弃置，不再作为独立的底层基础动作，而作为基本动作的具体执行方式或卡面表达。

## 二、作用对象判定标准

```text
作用对象必须能够被“查看、获得、调整、消除”中的至少一个动作直接处理。
```

不能被四个基础动作直接处理的内容，不算作用对象，只能作为：

```text
执行者。
接收者。
控制者。
条件。
归属标签。
区域说明。
状态限制。
选择方式。
```

因此，下列内容不属于作用对象：

```text
玩家。
你。
对手。
任意玩家。
控制者。
所有者。
发动者。
接收者。
```

下列内容也不属于独立作用对象：

```text
己方。
对手。
任意。
随机。
最多。
全部。
同一节点。
同一系统域。
在线。
离线。
独立部署。
被携带。
```

这些分别用于描述归属、数量、位置、状态或选择方式。

## 三、技能结构

每个技能由以下要素组成：

```text
基本动作
+ 作用对象
+ 对象归属
+ 数量
+ 来源区域
+ 目标区域或结果
+ 选择方式
+ 时机与限制
```

示例：

```text
从1名对手的手牌中随机获得1张牌。
```

拆解：

```text
基础动作：获得。
作用对象：手牌。
对象归属：对手。
数量：1张。
来源区域：对手手牌。
目标区域：你的手牌。
选择方式：随机。
接收者：你。
```

玩家不是作用对象，只用于限定谁的手牌、谁获得卡牌、谁控制实体或谁执行技能。

## 四、四类底层动作定义

### 查看

```text
获取作用对象的信息，但不改变其位置、区域、顺序、状态、归属或控制关系。
```

常见卡面表达：

```text
查看手牌。
查看手牌牌库顶。
查看公共弃牌区。
查看节点。
```

查看仍是合法底层动作，但因其规则影响较弱，不计入标准版18种手牌基本动作。

### 获得

```text
使作用对象中的卡牌、实体或资源进入指定接收者的持有范围、控制范围或资源范围。
```

常见卡面表达：

```text
抽牌。
从手牌中随机抽取卡牌。
从公共弃牌区取回卡牌。
获得指定对象中的卡牌。
```

规则口径：

```text
“抽牌”属于从手牌牌库获得卡牌。
“抽取”属于按随机方式获得目标。
“给予”不作为独立动作；从接收者角度统一视为获得。
```

### 调整

```text
改变作用对象的位置、所在区域、顺序、状态、部署方式或内部结构，但不使其进入公共弃牌区。
```

常见卡面表达：

```text
移动。
重新部署。
返回手牌。
返回牌库。
置顶。
置底。
排序。
重洗。
转移。
改变装备部署方式。
```

规则口径：

```text
移动属于位置调整。
返回属于区域调整。
置顶、置底和排序属于顺序调整。
重洗属于随机顺序调整。
```

### 消除

```text
使作用对象中的卡牌或实体离开当前有效区域并进入公共弃牌区。
```

按对象使用不同卡面词：

```text
游侠：消灭。
装备：销毁。
手牌中的卡牌：弃置。
一次性物品：弃置或销毁。
底层基础动作统一为“消除”。
```

游侠被消除时，其携带装备同时进入公共弃牌区，除非技能另有说明。

## 五、固定作用对象序号

### 1. 卡牌

```text
定位：上位对象。
```

用于描述：

```text
手牌中的卡牌。
手牌牌库中的卡牌。
公共弃牌区中的卡牌。
手牌牌库顶的卡牌。
```

卡牌可被查看、获得、调整或消除。

但“卡牌”本身过于宽泛，不直接作为标准版技能对象使用。实际技能必须附带区域、类型或状态限定。

### 2. 手牌

```text
定位：区域或卡牌集合对象。
版本：标准版手牌技能对象。
```

手牌整体可以被查看或调整；其中的卡牌可以被获得、调整或消除。

### 3. 手牌牌库

```text
定位：区域或卡牌集合对象。
版本：标准版手牌技能对象。
```

手牌牌库可以被查看或调整，例如排序、置顶、置底与重洗；其中的卡牌可以被获得或消除。

### 4. 公共弃牌区

```text
定位：区域或卡牌集合对象。
版本：标准版手牌技能对象。
```

公共弃牌区可以被查看或调整；其中的卡牌可以被获得、调整或消除。

### 5. 游侠

```text
定位：场上实体对象。
版本：标准版手牌技能对象。
```

游侠可以被查看、获得、调整或消除。

获得游侠不等同于获得控制权；具体结果必须写明游侠进入手牌、进入场上或其他合法区域。

### 6. 装备

```text
定位：场上实体对象。
版本：标准版手牌技能对象。
```

装备可以被查看、获得、调整或消除。

获得装备不等同于获得控制权；具体结果必须写明装备进入手牌、进入场上或其他合法区域。

以下仅是装备类型或状态限定，不另列一级作用对象：

```text
节点装备。
游侠装备。
通用装备。
独立部署。
被游侠携带。
```

### 7. 一次性物品

```text
定位：卡牌子类型对象。
版本：标准版手牌技能对象。
```

一次性物品可作为手牌或公共弃牌区中的具体卡牌类型，被查看、获得、调整或消除。

### 8. 节点

```text
定位：场上规则实体对象。
版本：扩展包对象。
```

节点可以被查看或调整，包括在线、离线、保护、回流等状态或区域变化。

节点通常不被消除。

### 9. 控制权

```text
定位：关系对象。
版本：扩展包对象。
当前限制：只能由总规则改变。
```

控制权可以被获得或调整，具体包括：

```text
游侠控制权。
装备控制权。
节点控制权。
```

当前不开放给标准版手牌技能，也不作为常规卡牌技能对象使用。现阶段控制权只依据部署、攻击占领、节点清空及其他总规则发生变化。

已有卡牌或节点技能若明确改变控制权，必须作为单独例外登记，不能反向开放控制权基础组合。

### 10. 权限电荷

```text
定位：资源对象。
版本：扩展包对象。
```

权限电荷可以被获得或调整数量。

支付电荷属于总规则中的资源支出，不属于消除。权限电荷不会进入公共弃牌区，因此不能使用“消除权限电荷”的口径。

## 六、标准版18种手牌基本动作

标准版去除意义较弱的“查看”组合，只保留“获得、调整、消除”与2至7号对象形成的18种基本动作。

### 获得类

```text
1. 获得手牌。
2. 获得手牌牌库。
3. 获得公共弃牌区。
4. 获得游侠。
5. 获得装备。
6. 获得一次性物品。
```

解释：

```text
获得手牌：从指定手牌中获得卡牌。
获得手牌牌库：从手牌牌库中获得卡牌，常见表达为抽牌。
获得公共弃牌区：从公共弃牌区中获得卡牌。
获得游侠：使游侠进入指定接收者的合法区域。
获得装备：使装备进入指定接收者的合法区域。
获得一次性物品：使一次性物品进入指定接收者手牌或其他合法区域。
```

### 调整类

```text
7. 调整手牌。
8. 调整手牌牌库。
9. 调整公共弃牌区。
10. 调整游侠。
11. 调整装备。
12. 调整一次性物品。
```

解释：

```text
调整手牌：改变手牌中卡牌的位置、归属、排列或结构。
调整手牌牌库：改变牌库顺序、顶底位置或重洗牌库。
调整公共弃牌区：改变弃牌区中卡牌的位置、顺序或去向。
调整游侠：改变游侠的位置、所在区域或状态。
调整装备：改变装备的位置、所在区域、携带关系或部署方式。
调整一次性物品：改变一次性物品所在区域或使用状态。
```

### 消除类

```text
13. 消除手牌。
14. 消除手牌牌库。
15. 消除公共弃牌区。
16. 消除游侠。
17. 消除装备。
18. 消除一次性物品。
```

解释：

```text
消除手牌：将手牌中的卡牌送入公共弃牌区。
消除手牌牌库：将手牌牌库中的卡牌送入公共弃牌区。
消除公共弃牌区：将公共弃牌区中的卡牌移出该区域；具体去向必须由技能明确。
消除游侠：使游侠离场并进入公共弃牌区，卡面通常写作“消灭”。
消除装备：使装备离场并进入公共弃牌区，卡面通常写作“销毁”。
消除一次性物品：使一次性物品进入公共弃牌区或按技能指定方式离开当前区域。
```

组合矩阵：

| 作用对象 | 获得 | 调整 | 消除 |
|---|---|---|---|
| 手牌 | 获得手牌 | 调整手牌 | 消除手牌 |
| 手牌牌库 | 获得手牌牌库 | 调整手牌牌库 | 消除手牌牌库 |
| 公共弃牌区 | 获得公共弃牌区 | 调整公共弃牌区 | 消除公共弃牌区 |
| 游侠 | 获得游侠 | 调整游侠 | 消除游侠 |
| 装备 | 获得装备 | 调整装备 | 消除装备 |
| 一次性物品 | 获得一次性物品 | 调整一次性物品 | 消除一次性物品 |

## 七、标准版手牌技能生成规则

```text
标准版所有游侠技能，只能由18种基本动作中的一种进行变体，或由其中两种及以上基本动作组合形成。
标准版所有物品卡技能，只能由18种基本动作中的一种进行变体，或由其中两种及以上基本动作组合形成。
标准版手牌技能不得直接使用8至10号扩展包对象。
标准版手牌技能不得新增第19种平行基本动作。
卡面自然语言可以不同，但必须能够完整映射回18种基本动作。
```

允许的变体维度：

```text
对象归属。
数量。
来源区域。
目标区域。
选择方式。
触发时机。
作用范围。
支付条件。
使用限制。
```

允许的组合方式：

```text
同一基本动作连续执行。
同一动作作用于不同对象。
不同动作作用于同一对象。
不同动作作用于不同对象。
条件满足后执行另一基本动作。
```

不允许：

```text
通过变体名称实际创造新的底层动作。
使用扩展包对象绕过标准版对象边界。
将执行者、接收者、控制者或条件误写为作用对象。
仅因卡面措辞不同就视为新的基本动作。
```

## 八、版本分层

```text
1号：上位对象，不直接作为标准版技能对象。
2至7号：标准版手牌技能对象。
8至10号：扩展包对象。
9号：目前只能由总规则改变。
```

对象进入标准版或扩展包，只表示可以进入对应版本的“动作 × 对象”组合审核，不代表所有动作在实际卡表中都必须出现，也不代表每种组合强度相同。

## 九、装备与位置调整边界

```text
节点装备可以作为独立部署实体重新部署。
游侠装备不能脱离携带者单独普通移动。
通用装备独立部署时可以重新部署；被游侠携带时随游侠移动。
同节点内改变通用装备的独立部署或携带状态，属于部署方式调整，不支付普通移动费用。
```

### 普通移动与重新部署

```text
属于行动阶段通用行动。
只能处理己方部署实体。
目标只能是自己控制的另一座合法节点。
普通移动和重新部署不是卡牌技能。
```

### 定向驱离

```text
属于“调整游侠”或“调整装备”的明确变体。
只能选择对手控制的游侠或装备。
目标节点必须由目标控制者控制。
不能改变目标控制权。
不能处理己方实体。
```

驱离枪当前文本：

```text
此装备给游侠装备时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

## 十、返回与携带装备

```text
返回手牌属于区域调整，不属于消除。
将1个场上的游侠或装备返回其控制者手牌时，目标不会进入公共弃牌区。
```

涉及携带装备时：

```text
若效果选择游侠，游侠携带的装备默认随游侠返回手牌，除非效果明确只处理游侠。
若效果选择被携带装备，可以只将该装备返回手牌。
```

## 十一、当前明确禁止或未开放的普通手牌技能组合

```text
直接作用于节点。
获得或调整控制权。
直接获得或调整权限电荷。
公开对手手牌。
隔离卡牌或实体。
禁用卡牌、技能或节点。
直接改变卡牌费用。
直接改变生命值。
直接造成伤害。
直接治疗。
任意单体空间转运。
多目标空间移动。
```

上述效果若无法映射为18种基本动作的合法变体或组合，不进入标准版手牌技能。

## 十二、卡面与基本动作的对应关系

| 卡面表达 | 标准版基本动作 |
|---|---|
| 抽牌 | 获得手牌牌库 |
| 随机抽取对手手牌 | 获得手牌 |
| 给予另一玩家卡牌 | 获得手牌 |
| 从弃牌区取回 | 获得公共弃牌区 |
| 移动游侠 | 调整游侠 |
| 移动或重新部署装备 | 调整装备 |
| 返回游侠手牌 | 调整游侠 |
| 返回装备手牌 | 调整装备 |
| 返回牌库 | 调整对应对象 |
| 置顶、置底、排序或重洗 | 调整手牌牌库 |
| 消灭游侠 | 消除游侠 |
| 销毁装备 | 消除装备 |
| 弃置手牌中的牌 | 消除手牌 |

## 十三、当前状态

```text
底层基础动作仍为4类：查看、获得、调整、消除。
标准版去除查看组合，只使用获得、调整、消除3类有效动作。
标准版手牌技能对象为2至7号，共6个对象。
标准版手牌技能空间固定为18种基本动作。
标准版所有手牌技能必须是18种基本动作的变体或组合。
扩展包对象为8至10号。
9号控制权目前只能由总规则改变。
卡面仍可使用抽牌、返回、移动、重洗、消灭、销毁等自然语言。
后续卡表审核应将旧技能统一映射到18种基本动作。
```
