# 归零者｜标准版基础动作池

> 文件定位：本文件记录《归零者 / ZERO ACCESS》标准版中游侠与物品卡技能可使用的基础动作池，并区分总规则行动、手牌技能与节点技能的动作边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、当前结论

```text
普通移动己方部署实体是03-标准版规则.md中的通用行动，不是卡牌技能。
玩家可以支付与派出1个实体攻击节点相同的电荷，在自己的节点之间移动1个己方部署实体。
普通移动的存在不会自动把“移动实体”加入游侠或物品卡技能基础动作池。
标准版游侠与物品卡技能只能使用本文件中的基础动作或明确例外。
```

---

## 二、装备与移动边界

```text
节点装备可以作为独立部署实体普通移动。
游侠装备不能脱离携带者单独移动。
通用装备独立部署时可普通移动；被游侠携带时随游侠移动。
同节点内改变通用装备的独立部署或携带状态，不属于普通移动，不支付电荷。
```

## 三、三类移动

### 普通移动

```text
属于行动阶段通用行动。
只能移动己方部署实体。
只能移动到自己控制的另一座节点。
当前每移动1个部署实体支付1电荷。
不属于技能，不触发入场或装备触发技能。
```

### 强制驱离

当前明确例外：

```text
游侠装备“驱离枪”在装备给游侠时，可以强制驱离1个对手游侠或装备。
```

限制：

```text
只能移动对手实体。
目标节点必须由目标控制者控制。
不能改变控制权。
不能移动己方实体。
同节点内转交或重新调整驱离枪，不重复触发强制驱离。
```

### 节点技能移动

```text
节点技能可以使用更宽的移动动作。
节点技能可以按卡面明确允许移动己方、对手或任意实体。
节点技能也可以按技能文本重映射控制权。
```

## 四、标准版手牌技能基础动作池

```text
1. 抽牌。
2. 查看牌库顶。
3. 调整牌库顶。
4. 返回牌库。
5. 查看手牌。
6. 返回手牌。
7. 消灭。
```

备选：

```text
重洗牌库。
```

## 五、基础动作定义

### 返回手牌

```text
将1个场上的游侠或装备移回其控制者手牌。
返回手牌不是移动、伤害、消灭或销毁。
```

涉及携带装备时：

```text
若效果选择游侠，游侠携带装备默认随游侠返回手牌，除非效果明确只处理游侠。
若效果选择被携带装备，可以只将该装备返回手牌。
```

### 消灭或销毁

```text
游侠离场破坏使用“被消灭”。
装备离场破坏使用“被销毁”。
游侠被消灭时，其携带装备同时进入公共弃牌区，除非装备技能另有说明。
```

## 六、强制驱离例外

动作名称：

```text
强制驱离。
```

动作定义：

```text
将1个对手场上的游侠或装备，从当前节点移动到其控制者控制的另一座合法节点。
```

允许载体：

```text
游侠装备在首次装备给游侠时触发。
节点技能。
```

标准限制：

```text
只能选择对手控制的游侠或装备。
目标节点可以在线或离线。
目标节点必须合法容纳移动后的部署实体。
强制驱离不改变目标控制者。
强制驱离不视为部署，不触发入场技能。
若目标是携带装备的游侠，其携带装备随游侠移动。
若目标是被携带装备，默认只移动该装备需要技能明确给出新的合法携带者，否则不能选择。
若来源节点清空，立即结算节点回流。
没有合法目标时，驱离枪仍可装备，但不执行驱离。
```

当前例子：

```text
驱离枪：此装备给游侠装备时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

## 七、当前明确移除的手牌技能动作

```text
弃牌。
公开手牌。
隔离。
禁用。
费用变化。
生命变化。
伤害。
治疗。
获得电荷。
己方返回牌库。
己方消灭。
任意单体空间转运。
多目标空间移动。
手牌技能控制权重映射。
```

说明：

```text
普通移动属于总规则行动，不恢复任意移动类手牌技能。
同节点内通用装备转换属于装备位置调整，不属于技能动作。
```
