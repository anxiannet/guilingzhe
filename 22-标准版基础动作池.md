# 归零者｜标准版基础动作池

> 文件定位：本文件记录《归零者 / ZERO ACCESS》标准版中游侠与物品卡技能使用的基础动作、作用对象及合法组合，并区分双人标准版动作、扩展包动作与节点技能边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、当前结论

底层基础动作统一为4类：

```text
1. 查看。
2. 获得。
3. 调整。
4. 消除。
```

双人标准版当前登记10种有效手牌基本动作。

查看是合法底层动作，但不计入10种标准版有效动作。

动作池只保留具有独立规则结果、适合标准版牌库规模，并且不会在双人对战中造成玩家卡牌归属混合的动作。

---

## 二、双人对战混牌判定

以下任一情况视为会造成混牌：

```text
其他玩家控制或持有的卡牌进入你的手牌。
其他玩家控制或持有的卡牌进入你的牌库。
你取得其他玩家场上卡牌的控制权。
技能结算后需要持续记录卡牌原控制者、原持有者或归还对象。
```

双人标准版不使用会造成混牌的技能。

以下不视为混牌：

```text
将对方场上卡牌返回其控制者手牌。
将对方场上卡牌返回公共手牌牌库。
将卡牌置入公共弃牌区。
只改变对方卡牌的位置、数值或状态，但不改变控制权和持有者。
```

---

## 三、技能结构

每个技能由以下要素组成：

```text
基本动作
+ 来源区域或场上对象
+ 对象归属
+ 卡牌类型限制
+ 数量
+ 目标区域或结果
+ 选择方式
+ 时机与限制
```

卡牌类型限制不单独建立基础动作。

“消耗品”是卡牌类型，不是独立基础动作对象。

---

## 四、四类底层动作定义

### 查看

获取信息，但不改变卡牌或实体的位置、区域、顺序、状态、归属或控制关系。

### 获得

使卡牌或场上实体进入指定玩家的手牌或控制范围。

双人标准版只允许：

```text
从公共手牌牌库获得卡牌。
从公共弃牌区获得合法目标。
```

从其他玩家手牌取得卡牌，或取得其他玩家场上实体的控制权，属于扩展包动作。

### 调整

改变卡牌或实体的位置、区域、顺序、状态、部署方式或内部结构，但不使其进入公共弃牌区，也不改变其控制权。

常见表达：

```text
移动。
重新部署。
返回手牌。
返回牌库。
置顶。
置底。
排序。
重洗。
交换自己手牌与公共牌库中的牌。
改变装备部署方式。
```

### 消除

使手牌中的卡牌或场上实体进入公共弃牌区。

按对象使用不同卡面词：

```text
游侠：消灭。
装备：销毁。
手牌中的卡牌：弃置。
```

将手牌牌库中的卡牌直接置入公共弃牌区属于扩展包动作。

## 六、双人标准版动作定义

### 1. 获得手牌牌库

从公共手牌牌库获得卡牌。

```text
卡面示例：从手牌牌库顶抽1张牌。
```

### 2. 获得公共弃牌区

将公共弃牌区中的合法目标加入你的手牌。

```text
卡牌必须明确目标类型。
取回游侠必须具有复苏、备份或重构等合法解释。
取得装备可解释为回收或修复。
取得消耗品仍映射为本动作，消耗品只是类型限制。
```

### 3. 调整手牌

改变自己手牌中卡牌的位置，或将其移至非弃牌区域。

```text
可将自己手牌放回公共手牌牌库。
可与公共牌库进行合法交换。
不得通过本动作取得其他玩家手牌。
```

### 4. 调整手牌牌库

改变公共手牌牌库中的顺序或结构，但卡牌仍留在牌库中。

```text
置顶、置底、排序、重洗均属于本动作。
```

### 5. 调整公共弃牌区

将公共弃牌区中的卡牌移至非手牌区域，但不加入任何玩家手牌。

```text
典型结果：置于手牌牌库顶、置于牌库底、洗回牌库。
单纯调整弃牌区顺序没有规则意义。
```

### 6. 调整游侠

改变场上游侠的位置、状态、数值或所在区域，但不将其置入公共弃牌区，也不改变其控制权。

```text
包括移动、返回其控制者手牌、返回公共牌库、临时强化或削弱。
```

### 7. 调整装备

改变场上装备的位置、状态、部署方式、数值或所在区域，但不将其置入公共弃牌区，也不改变其控制权。

```text
包括移动、重新部署、返回其控制者手牌、返回公共牌库和改变携带关系。
```

### 8. 消除手牌

将手牌中的卡牌置入公共弃牌区。

```text
卡面优先使用“弃置”。
消耗品可作为目标类型限制。
```

### 9. 消除游侠

将场上游侠置入公共弃牌区。

```text
卡面使用“消灭”。
```

### 10. 消除装备

将场上装备置入公共弃牌区。

```text
卡面使用“销毁”。
```

## 七、扩展包动作

### 获得其他玩家手牌

将其他玩家手牌中的卡牌加入你的手牌。

```text
会造成玩家持有卡牌混合。
不进入双人标准版动作池。
```

### 获得场上游侠

取得其他玩家场上游侠的控制权，或将其加入你的手牌。

```text
会改变卡牌控制关系，并可能造成玩家卡牌混合。
不进入双人标准版动作池。
```

### 获得场上装备

取得其他玩家场上装备的控制权，或将其加入你的手牌。

```text
会改变卡牌控制关系，并可能造成玩家卡牌混合。
不进入双人标准版动作池。
```

### 消除手牌牌库

将公共手牌牌库中的卡牌直接置入公共弃牌区。

```text
会直接减少公共手牌牌库牌数。
标准版牌库规模有限，因此不进入双人标准版动作池。
使用前必须补齐牌库耗尽、空牌库抽牌和弃牌区是否重洗等规则。
```

扩展包技能映射统一写为：

```text
扩展动作：获得其他玩家手牌。
扩展动作：获得场上游侠。
扩展动作：获得场上装备。
扩展动作：消除手牌牌库。
```

扩展包动作不占用双人标准版10种动作编号，也不计入标准版动作覆盖率。

## 八、已删除动作

以下旧动作已删除：

```text
获得消耗品。
调整消耗品。
消除公共弃牌区。
消除消耗品。
```

删除原因：

```text
消耗品只是卡牌类型限制，应映射到对应来源区域动作。
公共弃牌区已经是消除结果区域，没有合法的再次消除结果。
```

## 九、标准版技能生成规则

```text
双人标准版所有游侠与物品卡技能，只能由10种标准版基本动作中的一种进行变体，或由多种标准版基本动作组合形成。
不得在双人标准版中使用扩展包动作。
不得通过技能改变其他玩家卡牌的控制权或将其加入自己的手牌、牌库。
卡面自然语言可以不同，但必须完整映射回基本动作。
卡牌类型限制不单独建立基础动作。
```

---

## 十、节点技能边界

双人标准版节点技能同样不得造成混牌。

```text
节点技能可以移动、保护、强化、削弱或回收卡牌。
节点技能不得在双人标准版中取得其他玩家游侠或装备的控制权。
节点技能不得将其他玩家持有或控制的卡牌加入发动者手牌或牌库。
```

具有夺取控制权效果的节点技能属于扩展包或多人混战内容。

当前涉及该限制的节点：

```text
维相枢纽：全域调度中的“获得该实体控制权”属于扩展内容，不用于双人标准版。
指挥中心：紧急征用属于扩展内容或多人混战内容，不用于双人标准版。
```

| 卡面表达 | 当前动作 |
|---|---|
| 抽牌 | 1. 获得手牌牌库 |
