# 归零者｜标准版基础动作池

> 文件定位：本文件记录《归零者 / ZERO ACCESS》标准版中游侠与物品卡技能使用的基础动作、作用对象及合法组合，并区分总规则行动、手牌技能与节点技能的动作边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、当前结论

```text
标准版底层基础动作统一为4类：
1. 查看。
2. 获得。
3. 调整。
4. 消除。
```

对应“增、删、改、查”：

```text
增：获得。
删：消除。
改：调整。
查：查看。
```

抽牌、抽取、给予、移动、返回、置顶、置底、排序、重洗、消灭、销毁和弃置，不再作为独立的底层基础动作。

它们作为4类基础动作的具体执行方式或卡面表达使用。

标准版游侠与物品卡技能，只能由本文件规定的基础动作、作用对象和合法组合形成，或使用本文件登记的明确例外。

---

## 二、技能结构

每个技能由以下要素组成：

```text
基础动作
+ 作用对象
+ 对象归属
+ 数量
+ 来源区域
+ 目标区域或结果
+ 选择方式
+ 时机与限制
```

示例：

```text
从1名对手的手牌中随机获得1张牌。
```

拆解：

```text
基础动作：获得。
作用对象：手牌中的卡牌。
对象归属：对手。
数量：1张。
来源区域：对手手牌。
目标区域：你的手牌。
选择方式：随机。
```

## 三、四类基础动作定义

### 查看

```text
获取目标的隐藏信息，但不改变目标的位置、区域、顺序、状态、归属或控制权。
```

常见卡面表达：

```text
查看手牌。
查看牌库顶。
查看节点牌库顶。
查看公共弃牌区。
```

### 获得

```text
使目标卡牌、实体、控制权或资源进入指定玩家的持有范围、控制范围或资源范围。
```

常见卡面表达：

```text
抽牌。
从牌库获得卡牌。
从对手手牌中随机抽取卡牌。
从弃牌区取回卡牌。
获得场上实体控制权。
获得节点控制权。
```

规则口径：

```text
“抽牌”属于从手牌牌库顶获得卡牌。
“抽取”属于按随机方式获得目标。
“给予”不作为独立动作；从接收者角度统一视为获得。
```

### 调整

```text
改变目标的位置、所在区域、顺序、状态、部署方式或控制关系，但不使目标进入公共弃牌区。
```

常见卡面表达：

```text
移动。
重新部署。
返回手牌。
返回牌库。
置顶。
置底。
排序。
重洗。
转移。
上线。
离线。
改变装备部署方式。
```

规则口径：

```text
移动属于位置调整。
返回属于区域调整。
置顶、置底和排序属于顺序调整。
重洗属于随机顺序调整。
控制权转移属于控制关系调整。
```

### 消除

```text
使目标卡牌或场上实体离开当前有效区域并进入公共弃牌区。
```

按对象使用不同卡面词：

```text
游侠：消灭。
装备：销毁。
手牌中的卡牌：弃置。
底层基础动作统一为“消除”。
```

游侠被消除时，其携带装备同时进入公共弃牌区，除非技能另有说明。

## 五、标准手牌技能的当前合法范围

### 查看

当前允许：

```text
查看1名对手的手牌。
查看手牌牌库顶若干张。
```

### 获得

当前允许：

```text
指定玩家从手牌牌库顶获得卡牌。
从1名对手的手牌中随机获得卡牌。
```

以下仍需单独审核后才能进入普通手牌技能：

```text
从公共弃牌区获得卡牌。
获得场上实体控制权。
获得节点控制权。
获得权限电荷。
```

### 调整

当前允许：

```text
调整手牌牌库顶的顺序。
将手牌牌库顶的牌置顶或置底。
重洗手牌牌库。
将场上游侠或装备返回手牌。
将场上游侠或装备返回手牌牌库。
```

当前明确例外：

```text
驱离枪可以定向调整1个对手游侠或装备所在的节点。
```

普通移动己方部署实体属于总规则行动，不因“调整”成为基础动作而自动开放为所有手牌技能。

### 消除

当前允许：

```text
消除1个对手场上游侠。
消除1个对手场上装备。
```

卡面分别写作：

```text
消灭1个对手场上游侠。
销毁1个对手场上装备。
```

## 六、装备与移动边界

```text
节点装备可以作为独立部署实体重新部署。
游侠装备不能脱离携带者单独普通移动。
通用装备独立部署时可以重新部署；被游侠携带时随游侠移动。
同节点内改变通用装备的独立部署或携带状态，属于部署方式调整，不支付普通移动费用。
```

## 七、三类位置调整

### 普通移动与重新部署

```text
属于行动阶段通用行动。
只能处理己方部署实体。
目标只能是自己控制的另一座合法节点。
普通移动和重新部署不是卡牌技能。
```

### 定向驱离

```text
属于“调整 × 对手场上实体”的明确例外。
只能选择对手控制的游侠或装备。
目标节点必须由目标控制者控制。
不能改变目标控制权。
不能处理己方实体。
```

驱离枪当前文本：

```text
此装备给游侠装备时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

### 节点技能调整

```text
节点技能可以使用比普通手牌技能更宽的调整范围。
节点技能可以按卡面明确允许调整己方、对手或任意实体的位置。
节点技能也可以按卡面调整节点状态或实体控制权。
节点技能的合法范围不反向扩展普通手牌技能。
```

## 八、返回与携带装备

```text
返回手牌属于区域调整，不属于消除。
将1个场上的游侠或装备返回其控制者手牌时，目标不会进入公共弃牌区。
```

涉及携带装备时：

```text
若效果选择游侠，游侠携带的装备默认随游侠返回手牌，除非效果明确只处理游侠。
若效果选择被携带装备，可以只将该装备返回手牌。
```

## 九、当前明确禁止或未开放的普通手牌技能组合

```text
公开对手手牌。

禁用卡牌、技能或节点。
直接改变卡牌费用。
直接改变生命值。
直接造成伤害。
直接治疗。

消除己方场上实体。

多目标空间移动。
普通手牌技能重映射节点控制权。
普通手牌技能任意重映射实体控制权。
```

## 十一、当前状态

```text
标准版底层基础动作已统一为4类：查看、获得、调整、消除。
动作统计必须同时标明作用对象，不能只统计卡面动词。
卡面仍可使用抽牌、返回、移动、重洗、消灭、销毁等自然语言。
后续卡表审核应将旧“基础动作拆解”统一映射到这4类动作。
```
