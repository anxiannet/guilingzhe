# 归零者｜标准版基础动作池

> 文件定位：本文件记录《归零者 / ZERO ACCESS》标准版中游侠与物品卡技能可使用的基础动作池，并说明节点技能与手牌技能的动作边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、当前结论

```text
标准版游侠与物品卡技能只能使用本文件中的基础动作或明确例外。
可部署实体的入场技能不得将己方场上实体返回手牌。
己方场上实体回手只允许出现在一次性物品上。
场上实体移动通常属于节点动作。
标准版当前保留一个装备例外：武器装备可以通过入场技能强制驱离1个对手实体。
```

---

## 二、基础动作与节点动作边界

基础动作主池：

```text
1. 抽牌。
2. 查看牌库顶。
3. 调整牌库顶。
4. 返回牌库。
5. 查看手牌。
6. 返回手牌。
7. 消灭。
```

备选动作：

```text
重洗牌库。
```

节点专属动作：

```text
查看节点。
使节点在线或离线。
保护节点。
移动场上实体。
临时强化场上实体。
从弃牌区回收游侠或装备。
重映射实体控制权。
```

明确例外：

```text
武器装备可以在上场时强制驱离1个对手游侠或装备。
该例外不使“移动”成为所有游侠或装备的通用动作。
```

---

## 三、基础动作定义

### 1. 抽牌

```text
从手牌牌库抽牌，加入指定玩家手牌。
```

### 2. 查看牌库顶

```text
查看指定手牌牌库顶部的牌，不改变牌库顺序。
```

### 3. 调整牌库顶

```text
改变指定手牌牌库顶部牌的顺序或位置。
```

### 4. 返回牌库

```text
将目标牌放回其控制者或拥有者的手牌牌库顶、牌库底，或洗入牌库。
```

限制：

```text
不保留己方场上游侠返回己方牌库。
不保留己方场上装备返回己方牌库。
```

### 5. 查看手牌

```text
查看指定玩家的手牌。
```

### 6. 返回手牌

```text
将1个场上的游侠或装备移回其控制者手牌。
```

限制：

```text
可部署实体的入场技能不得将己方场上实体返回手牌。
己方场上实体回手只允许出现在一次性物品上。
返回手牌不是移动、伤害、消灭或销毁。
```

### 7. 消灭

```text
将1个场上的对手游侠或装备移入公共弃牌区。
```

## 四、武器驱离例外

动作名称：

```text
强制驱离。
```

动作定义：

```text
将1个对手场上的游侠或装备，从当前节点移动到其控制者控制的另一座合法节点。
```

允许载体：

```text
武器类装备的入场技能。
节点技能。
```

禁止载体：

```text
游侠入场技能。
非武器装备。
装备持续技能。
一次性物品。
```

标准限制：

```text
只能选择对手控制的游侠或装备。
不能选择己方实体。
目标节点必须由该实体的控制者控制。
目标节点必须能够合法容纳该实体。
目标节点可以是在线节点或离线节点。
目标节点不得超过3个实体。
强制驱离不改变目标控制者。
强制驱离不视为部署，不触发入场技能。
若原节点因目标离开而没有任何实体，立即结算节点回流。
没有合法目标节点时，不能选择目标；装备仍可正常上场。
```

当前标准版例子：

```text
驱离枪：此装备上场时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

---

## 五、节点移动与控制权重映射

```text
节点技能可以移动任意合法目标，并可按技能文本改变控制权。
维相枢纽“全域重映射”属于节点专属的移动与控制权重映射。
武器驱离例外不能改变控制权，也不能把目标移动到发动者控制的节点。
```

## 六、当前明确移除的手牌技能动作

```text
弃牌。
公开手牌。
隔离。
禁用。
费用变化。
攻击变化。
生命变化。
伤害。
治疗。
获得电荷。
己方返回牌库。
己方消灭。
可部署实体回手己方场上实体。
任意单体空间转运。
多目标空间移动。
手牌技能控制权重映射。
```

说明：

```text
原一次性物品“相位转运卡”及其任意单体空间移动例外废止。
当前手牌中的空间位置干预只保留武器装备“驱离枪”的受限敌方驱离。
```

## 七、规则维护口径

```text
具体卡牌技能以 04-测试版卡表.md 或 05-标准版卡表.md 为准。
节点技能的具体目标、费用与限制以当前节点卡表为准。
新增移动类手牌技能前必须先修改并通过本文件审核。
节点技能使用更宽动作集合，不自动扩展手牌技能池。
```
