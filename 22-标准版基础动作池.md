# 归零者｜标准版基础动作池

> 文件定位：本文件记录《归零者 / ZERO ACCESS》标准版中游侠与物品卡技能使用的基础动作、作用对象及合法组合，并区分标准版手牌动作、节点技能与扩展包动作边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、标准版支持模式

标准版支持：

```text
2至4人公共牌库混战。
玩家各自构筑独立牌组进行对战。
```

标准版手牌技能与节点技能使用不同边界：

```text
标准版手牌技能不得造成混牌。
标准版节点技能可以造成跨玩家卡牌转移或控制权变化。
不使用节点的对战模式不受节点混牌技能影响。
```

---

## 二、底层动作

底层动作统一为：

```text
查看。
获得。
调整。
消除。
```

“查看”是信息步骤，不计入标准版手牌动作覆盖率。

消耗品是卡牌类型限制，不是独立基础动作对象。

## 三、标准版手牌9种有效动作

### 获得类

```text
1. 获得手牌牌库。
2. 获得公共弃牌区。
```

### 调整类

```text
3. 调整手牌。
4. 调整手牌牌库。
5. 调整公共弃牌区。
6. 调整游侠。
7. 调整装备。
```

### 消除类

```text
8. 消除游侠。
9. 消除装备。
```

## 四、标准版手牌动作定义

### 1. 获得手牌牌库

从当前模式的手牌牌库获得卡牌。

```text
公共牌库模式：从公共手牌牌库获得。
独立牌组模式：从自己的牌库获得，除非卡面另有明确说明。
```

### 2. 获得公共弃牌区

将公共弃牌区中的合法目标加入手牌。

```text
卡牌必须明确目标类型。
独立牌组模式下，默认只能获得自己所有的卡牌。
标准版手牌技能不得通过本动作取得其他玩家所有的卡牌。
```

### 3. 调整手牌

改变自己手牌中卡牌的位置或所在非弃牌区域。

```text
包括放回牌库、与自己牌库中的卡牌交换。
不得取得其他玩家手牌。
不得将手牌直接置入公共弃牌区。
```

### 4. 调整手牌牌库

改变手牌牌库中的顺序或结构，但卡牌仍留在牌库中。

```text
置顶、置底、排序、重洗均属于本动作。
```

### 5. 调整公共弃牌区

将公共弃牌区中的卡牌移至非手牌区域，但不加入任何玩家手牌。

```text
典型结果：置于牌库顶、置于牌库底、洗回牌库。
独立牌组模式下，返回牌库时应返回卡牌所有者的牌库。
```

### 6. 调整游侠

改变场上游侠的位置、状态、数值或所在区域，但不将其置入公共弃牌区，也不改变其控制权。

### 7. 调整装备

改变场上装备的位置、状态、部署方式、数值或所在区域，但不将其置入公共弃牌区，也不改变其控制权。

### 8. 消除游侠

将场上游侠置入公共弃牌区。卡面使用“消灭”。

### 9. 消除装备

将场上装备置入公共弃牌区。卡面使用“销毁”。

## 五、扩展包手牌动作

### 获得其他玩家手牌

将其他玩家手牌中的卡牌加入你的手牌。

### 获得场上游侠

取得其他玩家场上游侠的控制权，或将其加入你的手牌。

### 获得场上装备

取得其他玩家场上装备的控制权，或将其加入你的手牌。

### 消除手牌

将玩家手牌中的卡牌直接置入公共弃牌区。

### 消除手牌牌库

将手牌牌库中的卡牌直接置入公共弃牌区。

扩展包手牌技能映射统一写为：

```text
扩展动作：获得其他玩家手牌。
扩展动作：获得场上游侠。
扩展动作：获得场上装备。
扩展动作：消除手牌。
扩展动作：消除手牌牌库。
```

以上动作不占用标准版手牌9种动作编号，也不计入标准版手牌动作覆盖率。

## 六、标准版节点技能边界

节点属于标准版，但节点技能不受标准版手牌技能“禁止混牌”限制。

标准版节点技能可以：

```text
取得其他玩家游侠或装备的控制权。
将其他玩家控制的卡牌移动到技能发动者控制的节点。
在卡面明确时，将其他玩家卡牌加入指定玩家手牌。
```

节点技能使用独立节点动作池，不计入标准版手牌9种动作覆盖率。

独立牌组模式下：

```text
节点技能改变控制权时，不改变卡牌所有权。
被取得控制权的卡牌必须保持所有权可识别。
卡牌返回牌库时，默认返回其所有者牌库。
游戏结束后归还原所有者。
```

不使用节点的对战模式不会触发任何节点混牌技能，因此不需要额外限制标准版节点卡。

节点技能仍不得绕过扩展包牌库限制，直接弃置玩家手牌或牌库卡牌，除非该节点被明确标记为扩展内容。

---

## 七、独立牌组模式所有权规则

```text
卡牌所有权由组牌时的牌组归属决定。
标准版手牌技能不得使其他玩家所有的卡牌进入你的手牌、牌库或控制区。
标准版节点技能可以改变控制权，但不改变所有权。
卡牌返回牌库时，默认返回其所有者的牌库。
卡牌进入公共弃牌区时，仍保留所有权信息。
游戏结束后，所有卡牌归还其原所有者。
```

## 八、已删除动作

```text
获得消耗品。
调整消耗品。
消除公共弃牌区。
消除消耗品。
```

原因：消耗品只是卡牌类型限制；公共弃牌区已经是消除结果区域。

---

## 九、查看的定位

```text
查看只获取隐藏信息，不改变游戏状态。
查看不是标准版手牌基础动作，也不参与动作覆盖率。
查看后排序、置顶或置底，映射为4. 调整手牌牌库。
纯查看其他玩家手牌或牌库顶，可作为信息效果单独审核。
查看公共弃牌区、场上卡牌与公开节点信息属于玩家基础信息权限，不是技能。
```
