## 2026-07-09｜4人测试发现可部署实体自回手循环

### 一、事件背景

在 v0.9 平衡模拟器同步为当前标准流程后，进行了 4 人共用牌库模式测试。

当前测试流程为：

```text
2–4 人共用手牌牌库。
公共弃牌区。
12 张标准节点牌组成节点牌库。
中央节点区保持 3 座无主节点。
玩家部署到中央节点区无主节点后获得节点控制权。
节点失去全部己方实体后回流节点牌库并重洗。
```

测试中发现 `G-007 回收地堡` 使用次数异常高，并进一步暴露“可部署实体入场技能回手己方场上实体”会制造循环。

---

### 二、测试观察

4 人局测试参数：

```text
玩家数：4
模式：共用牌库
模拟器：scripts/balance_simulator.py v0.9
测试局数：5000
最大回合：30
```

主要观察结果：

```text
Root Access 正常胜利约 60%。
超时测试结算约 39%。
平局低于 1%。
平均结束回合约 18。
Root Access 中位达成回合约 9。
```

异常观察：

```text
G-007 回收地堡使用次数异常高。
R-007 烟幕手使用次数也偏高。
节点控制权变化次数明显偏高。
```

初步判断：

```text
异常并非来自 Root Access 胜利条件，而是来自可部署实体入场技能的回手循环。
```

---

### 三、问题链条

#### 1. 自己回自己

当前通用流程为：

```text
先部署实体。
后结算该牌卡面技能。
```

若某张可部署实体的技能为：

```text
将1个己方场上游侠返回你的手牌。
将1个己方场上装备返回你的手牌。
```

则该实体部署后已经成为“己方场上实体”，可以选择自己返回手牌。

示例：

```text
打出 G-007 回收地堡。
G-007 进入节点。
结算 G-007 技能。
G-007 选择自己返回手牌。
节点可能变空并回流。
后续仍可再次打出 G-007。
```

这会造成重复部署、重复触发、重复占领 / 回流。

---

#### 2. “另1个”不能完全解决问题

曾讨论将文本改为：

```text
将另1个己方场上游侠返回你的手牌。
将另1个己方场上装备返回你的手牌。
```

该写法可以阻止“自己回自己”，但不能阻止同名互回。

示例：

```text
烟幕手 A 已在场。
打出烟幕手 B。
B 不能选择自己，但可以选择 A 返回手牌。
之后再打出 A。
A 又可以选择 B 返回手牌。
```

因此，“另1个”不是根本解法。

---

#### 3. “非同名”仍不能解决跨类型循环

曾讨论将文本改为：

```text
将1个非同名的己方场上游侠返回你的手牌。
将1个非同名的己方场上装备返回你的手牌。
```

该写法可以阻止自己回自己和同名互回，但仍无法阻止跨类型互回。

示例：

```text
某游侠 A：将1个己方装备返回手牌。
某装备 B：将1个己方游侠返回手牌。

打出 A，回收 B。
再打出 B，回收 A。
再打出 A，回收 B。
```

因此，“非同名”仍不是根本解法。

---

### 四、曾讨论但不采用的方案

#### 方案：先结算技能，再入场

该方案可以避免可部署实体选择刚上场的自己。

但该方案会引入新的流程问题：

```text
实体尚未入场时，如何判断技能来源？
尚未选择或确认部署节点时，如何处理目标？
技能结算改变节点状态后，是否还能部署到原目标节点？
Root Access 应在技能后检查，还是入场后检查，还是两次检查？
```

设计判断：

```text
不采用“所有游侠 / 装备先结算技能再入场”。
保留“先部署，后结算卡面技能”的主流程。
```

理由：

```text
更符合实体进入节点后释放战术效果的直觉。
不重写打出手牌流程。
不增加 Root Access 检查时点复杂度。
```

---

### 五、最终规则判断

最终采用规则设计限制：

```text
标准版中，可部署实体的入场技能不得将己方场上实体返回手牌。
己方场上实体回手效果只允许出现在一次性物品上。
```

该限制用于一次性堵住：

```text
自己回自己。
同名互回。
人回收人。
装备回收装备。
人回收装备。
装备回收人。
```

同时保留当前主流程：

```text
先部署游侠 / 装备。
后结算该牌卡面技能。
结算后检查 Root Access。
```

### 六、对卡表的影响

需要重审所有可部署实体中包含己方回手的技能。

当前重点对象：

```text
R-007 烟幕手：原方向涉及己方游侠回手。
R-015 扳手：原方向涉及己方装备回手。
G-007 回收地堡：原方向涉及己方装备回手。
```

后续处理方向：

```text
1. 这些卡不应继续作为可部署实体入场回手己方场上实体。
2. 若保留己方回手效果，应迁移为一次性物品。
3. 若保留为可部署实体，应改为查看、抽牌、对手回手、返回牌库、消灭等不形成己方回手循环的效果。
```
