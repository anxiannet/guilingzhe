# 归零者｜部署实体与攻防生命系统

> 文件定位：本文件定义《归零者 / ZERO ACCESS》中可部署实体、攻击力、生命值、部署到权限节点与战斗结算的基础口径。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、当前结论

《归零者》中的手牌牌库不再只包含抽象技能卡。

```text
游侠默认是可部署实体。
越权模块 / Override Gear 是所有非游侠卡的统称。
越权模块可以是装备、设备、工具、武器、权限卡、纳米机器人、电磁手雷、程序芯片或一次性越权物品。
越权模块若可以留在场上，就可以作为可部署实体部署在权限节点上。
每个可部署实体都有攻击力和生命值。
在线节点和离线节点都可以部署、争夺和控制。
```

说明：

```text
越权模块不再只理解为电子模块。
它是所有非游侠物品的一级卡牌类型。
部分越权模块是部署实体，部分越权模块是一次性物品。
离线节点不是不可用地点，只是节点技能关闭，并且正面信息不公开。
```

## 二、可部署实体

正式术语：

```text
可部署实体
```

英文建议：

```text
Deployable Entity
```

包括：

```text
1. 游侠。
2. 可部署越权模块。
```

可部署越权模块可以是：

```text
装备
设备
工具
武器
权限卡
纳米机器人
电磁手雷
程序芯片
数据钥匙
临时权限载体
防御工事
侦察无人机
其他可留场越权物品
```

暂不作为可部署实体：

```text
只结算一次效果、结算后进入手牌弃牌区的越权模块。
```

说明：

```text
后续需要为每张越权模块标注“实体模块”或“一次性模块”。
实体模块拥有攻击力和生命值，可以部署在权限节点。
一次性模块只结算效果，不留在节点上，可以没有攻击力和生命值。
```

## 三、越权模块与实体模块

越权模块是一级卡牌类型：

```text
越权模块 / Override Gear = 所有非游侠卡。
```

实体模块是越权模块中的一个子类：

```text
实体模块 = 可以部署在权限节点上的越权模块。
```

一次性模块是越权模块中的另一个子类：

```text
一次性模块 = 只结算效果，不留在权限节点上的越权模块。
```

示例：

```text
蜂鸟无人机：越权模块 / 设备 / 锚点 / 实体模块。
防御地堡：越权模块 / 设备 / 防御工事 / 实体模块。
零日芯片：越权模块 / 程序芯片 / 一次性反制。
强制回滚扇区：越权模块 / 一次性回滚。
电磁手雷：越权模块 / 武器 / 一次性干扰。
权限卡：越权模块 / 权限载体。
纳米机器人：越权模块 / 纳米设备。
```

## 四、攻击力与生命值

每个可部署实体都有两个基础数值：

```text
攻击力 / ATK
生命值 / HP
```

攻击力代表：

```text
该实体在接管权限节点、压制敌方实体或参与节点争夺时提供的进攻强度。
```

生命值代表：

```text
该实体在权限节点上承受攻击、清理、回滚或系统压制时的生存能力。
```

卡面建议显示：

```text
ATK / HP
```

## 五、部署到权限节点

部署定义：

```text
打出一个可部署实体时，选择1座权限节点，将该实体部署到该权限节点上。
```

部署限制：

```text
在线权限节点可以部署新实体。
离线权限节点也可以部署新实体。
离线状态不限制部署、争夺或控制。
```

控制关系：

```text
部署在权限节点上的实体仍由其控制者控制。
```

节点归属：

```text
权限节点的控制者由当前规则的接管结果决定。
部署实体不必然等于控制该节点。
```

说明：

```text
同一座权限节点上可以存在多个玩家控制的实体。
节点争夺通过接管行动和战斗结算处理。
```

## 六、在线 / 离线对部署和争夺的影响

在线节点：

```text
可以部署实体。
可以发起争夺。
可以被接管。
可以在已控制时发动节点技能。
正面信息公开。
可以计入 Root Access 胜利条件。
```

离线节点：

```text
可以部署实体。
可以发起争夺。
可以被接管。
可以保留控制归属。
不能发动节点技能。
正面信息不公开。
不计入 Root Access 胜利条件。
```

核心区别：

```text
离线只关闭节点技能与正面公开信息，不关闭部署和争夺。
```

## 七、游侠部署

游侠是最基础的可部署实体。

```text
打出游侠时，支付其电荷费用，然后将其部署到1座权限节点。
```

游侠费用：

```text
游侠费用默认等于当前星级。
```

游侠数值：

```text
游侠拥有攻击力和生命值。
星级影响数值上限和装备层级，但不直接替代攻击力和生命值。
```

---

## 八、越权模块部署

可部署越权模块打出时：

```text
支付模块费用，然后部署到1座权限节点。
```

可部署越权模块类型倾向：

```text
装备：偏向保护、增强、生存。
设备：偏向侦察、干扰、节点控制。
工具：偏向移动、修复、临时处理。
武器：偏向攻击、清理、压制。
权限卡：偏向临时权限、上线、解锁或接入。
纳米机器人：偏向治疗、破坏、修复或持续影响。
电磁手雷：偏向干扰、离线、瘫痪或范围压制。
防御工事：偏向守点和承伤。
锚点：偏向占位、侦察、连接节点或作为战术落点。
```

实体模块数值：

```text
实体模块同样拥有攻击力和生命值。
```

## 九、一次性模块

部分越权模块保持一次性效果。

```text
一次性模块没有持续部署状态。
一次性模块可以没有攻击力和生命值。
一次性模块结算后进入手牌弃牌区。
```

当前判断：

```text
G-002 零日芯片：更适合作为一次性反制模块。
G-005 强制回滚扇区：更适合作为一次性大范围模块。
```

## 十、节点战斗基础口径

接管权限节点时，比较参与接管的实体战斗能力。

基础方向：

```text
进攻方选择参与接管的实体。
防守方选择该节点上的防守实体。
双方根据攻击力、生命值、技能和节点效果结算。
```

当前暂定：

```text
攻击力用于造成伤害。
生命值降到0的实体被处理，置入来源手牌弃牌区。
```

说明：

```text
具体战斗流程需要后续单独测试，不在本文件一次性定死。
```

## 十一、与旧星级系统的关系

星级不废除。

```text
星级继续代表游侠当前装备状态、打出费用和大致强度层级。
```

但星级不再替代战斗数值。

```text
具体战斗由攻击力、生命值、技能、节点状态共同决定。
```
