## 二、权限节点在线 / 离线 / 保护状态

```text
正面朝上：在线状态。
背面朝上：离线状态。
竖置：保护状态。
```

在线权限节点：

```text
正面信息公开。
当前控制者公开。
该节点上的部署实体公开。
可以部署实体。
可以争夺和被接管。
可以使用节点技能。
可以计入 Root Access 胜利条件。
```

离线权限节点：

```text
正面信息不公开。
当前控制者公开。
该节点上的部署实体公开。
可以部署实体。
可以争夺和被接管。
不能使用节点技能。
不计入 Root Access 胜利条件。
```

保护状态：

```text
被保护节点竖置。
保护持续到保护者下回合开始。
被保护节点不能被接管。
被保护节点不能被技能变为离线。
被保护节点仍可以部署实体。
竖置只表示保护状态，不改变该节点在线 / 离线状态本身。
```

离线节点隐藏信息只包括：

```text
节点正面信息。
```

离线节点公开信息包括：

```text
当前控制者。
该节点上的部署实体。
```

胜利句：

```text
控制同一系统域的3座在线权限节点，立即获得 Root Access 并获胜。
```

## 三、可部署实体与 ATK / HP

```text
游侠、装备、设备、武器都视为可部署实体。
每个可部署实体都有攻击力和生命值。
每个可部署实体都可以部署在权限节点上。
在线节点和离线节点都可以部署可部署实体。
```

当前卡表处理：

```text
23张游侠默认都是可部署实体，ATK / HP 待审核分配。
G-006 蜂鸟无人机为设备实体，ATK 0，HP 1。
G-007 防御地堡为防御设备实体，ATK 4，HP 6，不能主动进攻。
其他越权模块继续逐张判断：实体模块或一次性模块。
实体模块需要 ATK / HP。
一次性模块可以没有 ATK / HP。
```

锚点：

```text
锚点是低成本部署实体，用于占位、侦察、连接节点或作为战术落点。
蜂鸟无人机是当前第一张锚点设备。
```

## 四、权限电荷与费用口径

```text
权限电荷 / Access Charge
卡面短称：电荷
```

当前测试资源规则：

```text
每回合开始时，当前玩家获得等同于当前回合数的权限电荷。
本回合电荷 = 当前回合数，最高为10。
第10回合起，每回合获得10电荷。
回合结束时，未使用的权限电荷全部清空。
```

当前费用口径：

```text
打出游侠：费用等于当前星级。
打出越权模块：费用按模块强度单独标注。
发动权限节点技能：默认花费1电荷，且节点必须在线。
```

## 五、权限节点技能原则

```text
每个节点技能都不一样。
每个节点技能只做一件事。
节点技能要简单、易懂。
节点技能必须符合节点世界观。
节点技能不写连锁效果。
节点技能不写条件限制。
节点技能卡面不使用“若 / 当 / 如果 / 然后再”结构。
```

保护关键词：

```text
保护1座权限节点。
```

说明：

```text
保护的持续时间和效果写在规则书中，不塞进卡面技能文案。
```
