## 二、权限电荷与费用口径

正式资源名：

```text
权限电荷 / Access Charge
```

卡面短称：

```text
电荷
```

世界观定义：

```text
权限电荷，是归零者小队通过劫持漏洞、占用废弃权限、回收系统残留认证、短时接入权限节点后积累的临时越权资源。
```

当前费用口径：

```text
打出游侠：费用等于当前星级。
打出越权模块：费用按模块强度单独标注。
发动权限节点技能：默认花费1电荷。
```

当前测试资源规则：

```text
每名玩家回合开始时获得3电荷。
每名玩家最多保留6电荷。
```

详细来源：

```text
16-权限电荷与费用系统.md
```

## 三、标准手牌牌库与中性卡面文案

```text
标准手牌牌库 = 游侠牌 + 越权模块牌组成的 60 张抽牌牌库。
```

使用方式：

```text
2 到 4 人共用游玩：所有玩家共用同一副标准手牌牌库与同一个手牌弃牌区。
对战卡组游玩：每名玩家可以各自使用一副标准手牌牌库与对应手牌弃牌区。
```

卡面区位统一使用：

```text
手牌牌库
手牌弃牌区
控制者手牌
来源手牌弃牌区
来源手牌牌库
己方权限节点
```

卡面避免使用：

```text
你的牌组
你的牌库
你的手牌牌库
```

## 五、权限节点技能结构

每个系统域固定 3 座权限节点。

每个系统域内，3 座权限节点分别承担：

```text
侦察类
进攻类
防守类
```

分类说明：

```text
侦察类：查看、筛选、预判手牌或信息。
进攻类：强化接管、推进争夺、提升进攻游侠星级。
防守类：保护己方节点、恢复游侠、阻止或削弱敌方接管。
```
