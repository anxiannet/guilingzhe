## 二、手牌牌库与不混牌口径

```text
手牌牌库 = 游侠牌 + 越权模块牌组成的抽牌牌库。
```

双人对战时：

```text
每名玩家各自使用一副手牌牌库。
每名玩家各自拥有手牌、手牌牌库、手牌弃牌区。
每张游侠牌与越权模块牌始终归属于其拥有者。
任何技能都不能改变卡牌拥有者。
```

统一处理：

```text
抽牌：从你的手牌牌库抽。
弃牌：放入你的手牌弃牌区。
处理 / 清理：置入其拥有者手牌弃牌区。
吹回：回到其拥有者手牌。
放回牌库：放回其拥有者手牌牌库。
隔离：放入其拥有者隔离区，不改变拥有者。
```

禁用旧词：

```text
公共牌库
弃牌堆
```

## 三、封存卡号与节点位址口径

权限节点卡号采用：

```text
全局存储序列 / 系统域节点位址
```

示例：

```text
N-007 / DN-D1
```

说明：

```text
N-007 = 首版节点封存卡体系中的第 7 号存储卡。
DN-D1 = 无人机域第 1 号节点位址。
```

卡号不是后台管理编号，而是世界观内真实存在的封存卡存储序列。

系统域信息也不是辅助说明，而是该封存卡的正式域标签。

版本号不属于世界观内信息，只进入资产文件名和样卡审核记录。
