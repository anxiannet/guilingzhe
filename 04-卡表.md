## 二、手牌牌库与不混牌口径

```text
手牌牌库 = 游侠牌 + 越权模块牌组成的抽牌牌库。
```

双人对战时：

```text
每名玩家各自使用一副手牌牌库。
每名玩家各自拥有手牌、手牌牌库、手牌弃牌区。
每张游侠牌与越权模块牌始终归属于其拥有者。
任何技能都不能改变卡牌拥有者。
```

统一处理：

```text
抽牌：从你的手牌牌库抽。
弃牌：放入你的手牌弃牌区。
处理 / 清理：置入其拥有者手牌弃牌区。
吹回：回到其拥有者手牌。
放回牌库：放回其拥有者手牌牌库。
隔离：放入其拥有者隔离区，不改变拥有者。
支援游侠：不是实体牌，不进入任何玩家手牌、手牌牌库或手牌弃牌区。
```

禁用旧词：

```text
公共牌库
弃牌堆
节点牌库
```

## 三、星级与装备层级映射

游侠星级体现装备和资源层级，不是军衔，也不是命名依据。

| 星级 / 类型 | 装备层级 | 说明 |
|---:|---|---|
| 1星 | 匕首类冷兵器、简易工具，几乎无护甲 | 当前装备状态最低 |
| 2星 | 手枪、自制枪械或土改武器，弹药和稳定性不足 | 有基础火力 |
| 3星 | 残缺或低配制式装备，品质不高，弹药不足 | 接近专业行动，但资源紧张 |
| 4星 | 全套制式装备，头盔、防弹衣齐全，弹药充足 | 稳定武装能力 |
