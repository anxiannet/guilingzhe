## 二、权限节点在线 / 离线状态

```text
正面朝上：在线状态。
背面朝上：离线状态。
```

在线权限节点：

```text
正面信息公开。
当前控制者公开。
该节点上的部署实体公开。
可以部署实体。
可以争夺和被接管。
可以使用节点技能。
可以计入 Root Access 胜利条件。
```

离线权限节点：

```text
正面信息不公开。
当前控制者公开。
该节点上的部署实体公开。
可以部署实体。
可以争夺和被接管。
不能使用节点技能。
不计入 Root Access 胜利条件。
```

离线节点隐藏信息只包括：

```text
节点正面信息。
```

离线节点公开信息包括：

```text
当前控制者。
该节点上的部署实体。
```

胜利句：

```text
控制同一系统域的3座在线权限节点，立即获得 Root Access 并获胜。
```

## 四、权限电荷与费用口径

```text
权限电荷 / Access Charge
卡面短称：电荷
```

当前测试资源规则：

```text
每回合开始时，当前玩家获得等同于当前回合数的权限电荷。
本回合电荷 = 当前回合数，最高为10。
第10回合起，每回合获得10电荷。
回合结束时，未使用的权限电荷全部清空。
```

当前费用口径：

```text
打出游侠：费用等于当前星级。
打出越权模块：费用按模块强度单独标注。
发动权限节点技能：默认花费1电荷，且节点必须在线。
```
