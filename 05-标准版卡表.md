## 二、标准版数值口径

```text
游侠没有星级概念。
星级不作为规则字段。
如果卡面出现星级，只作为 ATK 的视觉锚点或美术标识。
卡牌价值主要通过费用体现，也就是打出或发动时消耗的权限电荷数。
游侠与装备的规则字段为：费用、ATK、HP、技能。
一次性物品的规则字段为：费用、技能。
节点的规则字段为：系统域、节点技能费用、节点技能。

装备离场破坏用“被销毁”。
一次性物品结算后视为被使用，进入弃牌区。
```

说明：

```text
标准版卡表不得新增独立“星级”数值列。
如需做视觉分级，应由 ATK 或费用映射到卡面视觉，而不是新增规则资源。

HP 是防御与驻守能力。
费用是卡牌总价值。
```

## 三、ATK 数值锚点

| ATK | 规则含义 | 视觉锚点 | 设计定位 |
|---:|---|---|---|
| 0 | 不能主动攻击节点，防守时造成0点伤害。 | 侦察、工具、后勤、无人设备、信标、数据设备。 | 有功能，但不负责打。 |
| 1 | 造成1点伤害，可以处理1HP目标。 | 匕首、撬棍、短刀、旧工具、临时护具。 | 轻量攻击，低费或辅助单位。 |
| 2 | 造成2点伤害，可稳定清理1HP / 2HP目标。 | 手枪、短枪、自制枪械、土改武器、低配制式装备。 | 标准攻击，普通战斗主力。 |
| 3 | 造成3点伤害，可突破多数普通守点实体。 | 制式步枪、爆破装备、完整战术装备、重火力、外骨骼部件。 | 强攻击，节点攻坚核心。 |

说明：

```text
旧星级体系只作为 ATK 的视觉锚点。
旧1星约等于 ATK 1 的低攻击视觉。
旧2星至3星约等于 ATK 2 的标准攻击视觉。
旧4星至5星约等于 ATK 3 的强攻击视觉。
ATK 0 不进入旧1星至5星战斗层级，偏工具和侦察。
```

## 四、HP / 防御数值锚点

| HP | 规则含义 | 视觉锚点 | 设计定位 |
|---:|---|---|---|
| 1 | 受到1点伤害就离场。 | 无护甲、轻便衣物、临时护具、工具型设备。 | 脆弱，但通常费用低、攻击高或技能直接。 |
| 2 | 能承受1点伤害，受到2点伤害离场。 | 厚外套、拼接护具、局部护甲、简易战术背心。 | 基础防御，普通耐久。 |
| 3 | 能承受1到2点伤害，需要3点伤害才离场。 | 低配制式护甲、战术背心、护肩、头盔、强化护具。 | 稳定驻守，有留场能力。 |
| 4 | 普通1到2 ATK单位很难单独清掉。 | 完整防弹衣、重型护甲、盾牌、强化外套。 | 强防御，主要价值是占住节点。 |
| 5 | 极难被单个普通单位清除。 | 重型装甲、固定工事、防御地堡、外骨骼防护结构。 | 重型防御 / 工事，核心价值是占住节点。 |

说明：

```text
生命值可以在玩家说明中理解为防御力。
正式规则字段仍写 HP / 生命值。
HP 表示驻守能力、承伤能力和留场能力。
游侠因伤害离场称为被消灭。
装备因伤害离场称为被销毁。
```

## 五、ATK / HP 组合锚点

| 组合 | 设计定位 | 适合方向 |
|---|---|---|
| 1 / 1 | 低费基础单位。能打，但很脆。 | 侦察员、信使、扒手。 |
| 1 / 2 | 辅助驻守单位。攻击弱，稍能站。 | 烟幕手、预读者、广播员、扳手、中间人。 |
| 2 / 1 | 快攻单位。攻击高于生命，容易死。 | 甩棍、快手、突击型游侠。 |
| 1 / 3 | 防守 / 续航单位。攻击弱，留场强。 | 不倒翁、防守型游侠。 |
| 3 / 2 | 高压攻击单位。能突破多数普通守点实体。 | 清除者、爆破兵、强攻型游侠。 |
| 3 / 3 | 核心战斗单位。攻击强，生命也稳定。 | 推土机、清道夫、高费主战单位。 |
| 1 / 4 | 重防守单位。攻击弱，但极难清掉。 | 门神、防御型游侠。 |
| 1 / 5 | 重型装备 / 工事。核心价值是节点驻守和拖延。 | 回收地堡、重型防守装备。 |

## 六、费用与总价值锚点

```text
1费：轻量单位，低 ATK / 低 HP，技能简单。
2费：标准低费牌，可以有基础技能，或中等身材。
3费：中费牌，可以有组合技能、较好留场，或特殊被动。
4费：强牌，要么攻击强，要么防御强，要么技能强。
5费：高价值核心牌，应具备强清除、万能解、强攻坚或关键节奏能力。
10费：终局锚点费用，用于加快收束、集中战力或帮助玩家完成胜利条件。
```

说明：

```text
高 ATK 不一定代表卡牌总价值高。
高 HP 不一定代表卡牌总价值高。
卡牌价值主要通过费用，也就是权限电荷消耗体现。
10费效果应明显影响终局节奏，但不应自动跳过胜利条件检查。
```

## 十一、迁入条件

测试版卡表迁入标准版卡表前，必须完成：

```text
1. 2人共用牌库测试通过。
2. 3人共用牌库测试通过。
3. 4人共用牌库测试通过。
4. 最高权限平均达成回合处于可接受范围。
5. 单卡使用率无异常膨胀。
6. 无自回手、同名互回、跨类型互回等循环风险。
7. 所有游侠、物品卡、节点技能通过 19、20、21、22 文档审核。
8. 标准版复制数、费用、ATK、HP 完成最终复核。
9. 标准版卡表确认未将星级作为独立规则字段。
10. 新增装备候选完成费用、复制数和技能强度复核。
11. 无规则技能卡确认其技能描述位置文案不会被误读为规则效果。
12. 纳米科技域是否进入正式标准版完成多域结构复核。
13. 蜂鸟巢“紧急召回”完成10费锚点、节点上限、超限回手与胜利速度复核。
14. 蜂鸟“被销毁后放到手牌牌库底”完成持续供应强度复核。
15. 空白储存卡完成目标范围、控制权转移、节点回流、再部署与5费强度复核。
16. 空白储存卡确认不会与回手、复制或弃牌回收效果形成无限循环。
17. 空白储存卡的维相科技制造来源、DPT-P0 型号与维相转换视觉通过样卡审核。
18. 维度空间域补足3座节点，并完成节点技能、节点牌库密度和最高权限胜利速度复核。
19. 维度空间域确认不引入时间旅行、平行宇宙或实体无限复制规则。
20. 维相枢纽完成任意实体目标范围、无主节点控制权、离线节点、保护状态与节点回流测试。
21. 空间传送确认不触发部署或入场技能，并且不会造成混合控制者节点或突破3实体上限。
```
