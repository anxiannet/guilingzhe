## 二、卡牌复制数规则

```text
通用装备：每种4张。
其他非唯一卡牌：每种2张。
具有“唯一”属性的卡牌：每种1张。
```

“唯一”优先于卡牌类型的基础复制数。节点卡默认具有“唯一”属性，因此每座节点各1张。

## 三、装备分类口径

```text
节点装备：只能独立部署，占用节点部署位。
游侠装备：只能由游侠携带，不占节点部署位。
通用装备：可以独立部署，也可以由游侠携带。
每个游侠最多携带1件游侠装备或通用装备。
携带装备的ATK与HP通常分别增加游侠ATK与HP。
HP为0或卡面明确不提供HP加成的装备，不增加游侠HP。
驱离枪与基因备份卡均为HP 0，不提供HP加成。
```

装备文本区分：

```text
技能：具有独立名称、触发条件或结算动作的特殊能力。
功能：装备持续提供的基础作用，不设置技能名。
基础功能不得为了卡面包装而强行命名为技能。
```

```text
制造商：负责产品结构、生产、装配和销售的企业。
研发部门：制造商内部负责具体技术研发的部门。
联合研发：共同参与产品核心技术研发的外部企业。
技术支持：提供核心授权技术、接口标准或关键模块，但不作为制造商。
```

### 基因备份卡

卡面建议文案：

```text
基因备份卡

基因备份
此游侠将被消灭时，改为将该游侠返回其控制者手牌，并将此装备置入公共弃牌区。
```

规则口径：

```text
基因备份卡为游侠装备，ATK 0 / HP 0，不提供属性加成。
“基因备份”是替代消灭的触发技能。
触发后，游侠返回其控制者手牌，基因备份卡进入公共弃牌区。
该游侠没有被消灭，因此不触发要求“游侠被消灭”的效果。
返回手牌后，清除该游侠受到的伤害、临时属性和回合状态。
若基因备份卡先被单独销毁、返回或夺取，则不再保护原游侠。
```

企业分工：

```text
制造商：纳米科技公司。
研发部门：生物基因部，负责游侠基因提取、编码和备份协议。
联合研发：维相科技 / DPT，负责二维封存、数据回传和实体重构接口。
维相科技不是基因备份卡制造商。
```
