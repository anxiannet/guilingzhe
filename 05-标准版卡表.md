
## 三、ATK 数值锚点

| ATK | 规则含义 | 视觉锚点 | 设计定位 |
|---:|---|---|---|
| 0 | 不能主动攻击节点，防守时造成0点伤害。 | 侦察、工具、后勤、无人设备、信标、数据设备。 | 有功能，但不负责打。 |
| 1 | 造成1点伤害，可以处理1HP目标。 | 匕首、撬棍、短刀、旧工具、临时护具。 | 轻量攻击，低费或辅助单位。 |
| 2 | 造成2点伤害，可稳定清理1HP / 2HP目标。 | 手枪、短枪、自制枪械、土改武器、低配制式装备。 | 标准攻击，普通战斗主力。 |
| 3 | 造成3点伤害，可突破多数普通守点实体。 | 制式步枪、爆破装备、完整战术装备、重火力、外骨骼部件。 | 强攻击，节点攻坚核心。 |

## 四、HP / 防御数值锚点

| HP | 规则含义 | 视觉锚点 | 设计定位 |
|---:|---|---|---|
| 1 | 受到1点伤害就离场。 | 无护甲、轻便衣物、临时护具、工具型设备。 | 脆弱，但通常费用低、攻击高或技能直接。 |
| 2 | 能承受1点伤害，受到2点伤害离场。 | 厚外套、拼接护具、局部护甲、简易战术背心。 | 基础防御，普通耐久。 |
| 3 | 能承受1到2点伤害，需要3点伤害才离场。 | 低配制式护甲、战术背心、护肩、头盔、强化护具。 | 稳定驻守，有留场能力。 |
| 4 | 普通1到2 ATK单位很难单独清掉。 | 完整防弹衣、重型护甲、盾牌、强化外套。 | 强防御，主要价值是占住节点。 |
| 5 | 极难被单个普通单位清除。 | 重型装甲、固定工事、防御地堡、外骨骼防护结构。 | 重型防御 / 工事，核心价值是占住节点。 |

| 组合 | 设计定位 | 适合方向 |
|---|---|---|
| 1 / 1 | 低费基础单位。能打，但很脆。 | 侦察员、信使、扒手。 |
| 1 / 2 | 辅助驻守单位。攻击弱，稍能站。 | 烟幕手、预读者、广播员、扳手、中间人。 |

| 1 / 3 | 防守 / 续航单位。攻击弱，留场强。 | 不倒翁、防守型游侠。 |
| 3 / 2 | 高压攻击单位。能突破多数普通守点实体。 | 清除者、爆破兵、强攻型游侠。 |
| 3 / 3 | 核心战斗单位。攻击强，生命也稳定。 | 推土机、清道夫、高费主战单位。 |
| 1 / 4 | 重防守单位。攻击弱，但极难清掉。 | 门神、防御型游侠。 |
| 1 / 5 | 重型装备 / 工事。核心价值是节点驻守和拖延。 | 回收地堡、重型防守装备。 |

## 六、费用与总价值锚点

```text
1费：轻量单位，低 ATK / 低 HP，技能简单。
2费：标准低费牌，可以有基础技能，或中等身材。
3费：中费牌，可以有组合技能、较好留场，或特殊被动。
4费：强牌，要么攻击强，要么防御强，要么技能强。
5费：高价值核心牌，应具备强清除、万能解、强攻坚或关键节奏能力。
10费：终局锚点费用，用于加快收束、集中战力或帮助玩家完成胜利条件。
```

维相枢纽规则说明：

```text
全域重映射是“空间传送”的强化版本。
相位转运卡只能移动实体，不能改变控制权。
维相枢纽支付10电荷后，可以把对手实体传送到己方节点并夺取其控制权。
目标节点在结算后必须合法：不得超过3个实体，也不能存在多个玩家控制的实体。
若移动到无主节点，目标实体原控制者获得该节点控制权。
若移动到第三名玩家控制的节点，目标实体必须与该节点现有实体属于同一控制者，否则该目标节点不合法。
维相枢纽自身必须在线并由发动者控制。
全域重映射不直接宣告胜利，结算后仍按最高权限条件检查。
```

## 十一、迁入条件

```text
1. 2、3、4人共用牌库测试通过。
2. 最高权限平均达成回合处于可接受范围。
3. 所有卡牌与节点技能通过19、20、21、22文档审核。
4. 空白储存卡完成控制权转移、节点回流与5费强度测试。
5. 相位转运卡完成单体移动、离线节点、无主节点控制权与4费强度测试。
6. 相位转运卡确认不触发部署或入场技能，也不改变控制权。
7. 维相枢纽完成敌方实体夺取、己方节点容量、原节点回流与10费强度测试。
8. 全域重映射确认不会制造混合控制者节点，也不会直接跳过最高权限胜利条件。
9. 维度空间域补足3座节点并完成节点牌库密度测试。
```
