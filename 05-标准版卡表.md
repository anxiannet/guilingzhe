## 二、卡牌复制数规则

```text
通用装备：每种4张。
其他非唯一卡牌：每种2张。
具有“唯一”属性的卡牌：每种1张。
```

“唯一”优先于卡牌类型的基础复制数。节点卡默认具有“唯一”属性，因此每座节点各1张。

## 三、装备分类口径

```text
节点装备：只能独立部署，占用节点部署位。
游侠装备：只能由游侠携带，不占节点部署位。
通用装备：可以独立部署，也可以由游侠携带。
每个游侠最多携带1件游侠装备或通用装备。
携带装备的ATK与HP通常分别增加游侠ATK与HP。
HP为0或卡面明确不提供HP加成的装备，不增加游侠HP。
驱离枪为HP 0，只提供ATK加成。
```

装备文本区分：

```text
技能：具有独立名称、触发条件或结算动作的特殊能力。
功能：装备持续提供的基础作用，不设置技能名。
基础功能不得为了卡面包装而强行命名为技能。
```

```text
制造商：负责产品结构、武器化设计、生产、装配和销售的企业。
技术支持：向产品提供核心授权技术、接口标准或关键模块的企业。
技术支持方不等同于制造商。
```

轻动力甲卡面建议文案：

```text
轻动力甲

此游侠进行普通移动时，不支付电荷。
```

轻动力甲规则口径：

```text
轻动力甲没有技能名。
“此游侠进行普通移动时，不支付电荷”是持续装备功能。
该功能不增加游侠每回合可普通移动的次数。
刚上场状态仍然禁止普通移动。
技能移动不属于普通移动，不适用此功能。
```

驱离枪卡面建议文案：

```text
驱离枪

定向驱离
此装备给游侠装备时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

驱离枪企业口径：

```text
制造商：阈锋军械 / THRESHOLD ARMAMENTS。
简称：TA。
技术支持：维相科技 / DPT。
维相科技不是驱离枪制造商。
```

费用核定：

```text
驱离枪费用为3。
它需要场上已有未携带装备的己方游侠，不能独立部署，也不能单独维持节点控制。
它占用游侠唯一装备位，HP为0，只提供+2 ATK。
定向驱离是一次条件性效果；若对手没有另一座合法节点，则不能执行驱离。

轻动力甲费用为3。
它占用游侠唯一装备位，不提供ATK，只提供+2 HP。
免费普通移动是稳定机动功能，但不突破每回合普通移动1次与刚上场限制。
```

蜂鸟两种状态：

```text
独立部署：作为通用装备实体，占1个节点部署位，ATK 3 / HP 1。
游侠携带：不占节点部署位，携带者获得+3 ATK与+1 HP。
```
