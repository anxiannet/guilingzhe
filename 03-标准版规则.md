# 归零者｜标准版规则

> 文件定位：本文件记录 `归零者 / ZERO ACCESS` 独立项目当前标准版总规则、游戏模式、通用规则、回合流程与结算规则。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`
>
> 说明：具体卡牌费用与技能以当前卡表为准；本文件维护总规则。

---

## 一、游戏目标

```text
控制同一系统域的3座节点，立即获得 Root Access 并获胜。
```

---

## 二、标准版支持模式

标准版支持以下两种组局方式：

```text
1. 2至4人公共牌库混战。
2. 玩家各自构筑独立牌组进行对战。
```

### 2至4人公共牌库混战

```text
所有玩家共用同一手牌牌库。
所有玩家共用同一公共弃牌区。
所有玩家共用节点牌库与中央节点区。
跨玩家获得卡牌或改变控制权时，不需要记录实体卡所有权。
```

### 独立牌组对战

```text
每名玩家使用自己构筑的独立牌组。
每名玩家从自己的牌库抽牌。
节点牌库与中央节点区仍为公共区域。
公共弃牌区是否共用，以组牌规则或比赛规则为准；未另行规定时使用公共弃牌区。
```

独立牌组所有权规则：

```text
卡牌所有权由游戏开始时所属牌组决定。
卡牌控制权可以因技能改变，但所有权不改变。
其他玩家的卡牌进入你的手牌或控制区时，必须保持所有权可识别。
卡牌返回牌库时，默认返回其所有者的牌库，除非卡面明确规定其他结果。
卡牌进入公共弃牌区时仍保留所有权信息。
游戏结束后，所有卡牌归还其原所有者。
```

独立牌组的牌组数量、同名卡上限与牌库耗尽规则另行登记；在这些规则确定前，不得自行建立平行组牌规则。

## 三、卡牌与装备类型

一级卡牌类型：

```text
游侠。
物品卡。
节点。
```

物品卡分为：

```text
装备。
消耗品。
```

### 消耗品

```text
支付费用后立即结算效果。
结算后进入公共弃牌区，除非卡面另有说明。
不作为场上实体存在。
只显示费用，不显示ATK与HP。
```

装备按部署方式分为：

```text
节点装备。
游侠装备。
通用装备。
```

### 节点装备

```text
只能作为独立实体部署到节点。
占用节点的1个部署位。
不能由游侠携带。
拥有自己的ATK与HP。
不能通过“移动”行动改变节点，只能执行“重新部署”。
```

### 游侠装备

```text
不能直接部署到节点。
打出时必须装备给同一玩家控制的1个游侠。
不占用节点部署位。
每个游侠最多携带1件装备。
不是独立实体，不能单独攻击、被攻击或控制节点。
ATK作为携带者的ATK加成。
HP作为携带者的HP加成；HP为0或卡面明确不提供HP加成的装备除外。
```

### 通用装备

```text
既可以作为节点装备独立部署，也可以作为游侠装备由游侠携带。
独立部署时占用节点1个部署位，并使用自己的ATK与HP。
由游侠携带时不占用节点部署位，其ATK与HP分别加入游侠的ATK与HP。
独立部署时按照节点装备规则执行重新部署。
```

## 四、节点部署位与实体

```text
每座节点最多容纳3个部署实体。
计入部署位的只有游侠与独立部署的节点装备或通用装备。
由游侠携带的装备不占部署位。
同一节点上的部署实体必须全部属于该节点控制者。
无主节点不能存在任何部署实体或被携带装备。
控制节点的必要条件是节点上至少有1个己方部署实体。
```

示例：

```text
3个游侠，每个各携带1件装备，仍只占3个部署位。
2个游侠加1个独立部署装备，共占3个部署位。
```

## 五、游戏准备

### 公共牌库混战

```text
1. 将标准手牌洗混，作为公共手牌牌库。
2. 设置公共弃牌区。
3. 将标准节点牌洗混，作为节点牌库。
4. 翻开3张节点放入中央节点区。
5. 中央节点初始为无主、无实体。
6. 每名玩家从公共手牌牌库抽5张起始手牌。
7. 随机决定起始玩家。
```

### 独立牌组对战

```text
1. 每名玩家展示符合当前组牌规则的独立牌组并分别洗混。
2. 设置公共弃牌区；所有进入其中的卡牌必须保持所有权可识别。
3. 将标准节点牌洗混，作为公共节点牌库。
4. 翻开3张节点放入中央节点区。
5. 中央节点初始为无主、无实体。
6. 每名玩家从自己的牌库抽5张起始手牌。
7. 随机决定起始玩家。
```

## 六、权限电荷

```text
每回合开始时，当前玩家获得等同于当前回合数的权限电荷，最高为10。
第10回合起，每回合获得10电荷。
回合结束时，未使用电荷全部清空。
```

费用口径：

```text
打出卡牌支付卡面费用。
发动节点技能支付节点卡面费用。
攻击节点时，每派出1个攻击实体支付1电荷。
游侠普通移动时，每移动1个游侠支付1电荷。
节点装备重新部署时支付1电荷。
同一节点内改变通用装备部署方式不支付电荷。
```

---

## 七、回合流程

```text
1. 回合开始。
2. 获得权限电荷。
3. 抽1张牌。
4. 行动阶段。
5. 结束阶段。
```

抽牌来源：

```text
公共牌库混战：从公共手牌牌库抽牌。
独立牌组对战：从自己的牌库抽牌。
```

行动阶段可以按任意顺序执行：

```text
打出游侠。
打出节点装备。
打出游侠装备。
打出通用装备。
打出消耗品。
移动己方游侠。
重新部署己方节点装备或独立部署的通用装备。
改变同节点内通用装备的部署方式。
攻击节点。
使用节点技能。
主动结束行动阶段。
```

```text
每名玩家每回合最多使用1次节点技能。
节点技能可在行动阶段任意合法时点发动。
```

## 八、打出游侠与装备

### 打出游侠

```text
支付费用，将游侠部署到无主节点，或自己控制且未满3个部署实体的节点。
部署到无主节点后，获得该节点控制权。
```

### 打出节点装备

```text
支付费用，将节点装备独立部署到无主节点，或自己控制且未满的节点。
节点装备占用1个部署位。
```

### 打出游侠装备

```text
支付费用，选择自己控制的1个未携带装备的游侠。
将游侠装备置于该游侠下方或旁边。
不能直接部署到节点。
```

### 打出通用装备

支付费用后选择：

```text
1. 作为节点装备独立部署，占用1个部署位。
2. 装备给自己控制的1个未携带装备的游侠，不占部署位。
```

刚打出的游侠本回合不能普通移动或参与攻击。刚独立部署的装备本回合不能重新部署或参与攻击。

---

## 九、携带装备规则

```text
每个游侠最多携带1件装备。
携带装备与游侠视为位于同一节点。
携带装备随游侠一起移动、攻击、进入目标节点或返回来源节点。
携带装备不单独支付移动或攻击费用，也不单独占用攻击实体数量。
```

属性结算：

```text
最终ATK = 游侠ATK + 携带装备ATK。
最终HP = 游侠HP + 携带装备HP；HP为0或明确不增加HP的装备除外。
```

战斗结算：

```text
游侠与其携带装备视为1个战斗单位。
携带装备不单独造成伤害，也不单独承受普通战斗伤害。
若战斗单位受到的伤害大于等于最终HP，游侠被消灭，其携带装备同时进入公共弃牌区。
```

装备可以被技能单独返回、销毁、夺取或处理，只要技能文本允许。

离场结算：

```text
游侠被消灭时，其携带装备同时进入公共弃牌区，除非装备技能另有说明。
游侠返回手牌时，其携带装备也返回其控制者手牌。
游侠进入牌库时，其携带装备默认跟随；独立牌组模式下分别返回各自所有者的牌库。
若携带装备被单独销毁，游侠保留在原位置并重新计算ATK与HP。
```

## 十、通用装备的同节点转换

```text
独立部署的通用装备，可以改由同节点内1个未携带装备的己方游侠携带。
由游侠携带的通用装备，可以卸下并独立留在同一节点，但节点必须有空余部署位。
由1个游侠携带的通用装备，可以转交给同节点内另1个未携带装备的己方游侠。
```

转换规则：

```text
不支付电荷。
不是打出、部署、移动、重新部署或重新上场。
不触发上场、部署或入场技能。
转换后立即重新计算相关游侠的ATK与HP。
必须继续满足每个游侠最多携带1件装备和节点最多3个部署实体的限制。
```

---

## 十一、游侠普通移动

```text
只能移动自己控制的游侠。
目标必须是自己控制的另一座未满节点。
不能移动到无主节点或其他玩家节点。
每个游侠每回合最多普通移动1次。
刚上场的游侠本回合不能普通移动。
```

同一游侠每回合可以普通移动1次并参与攻击1次，两项分别支付费用。

若来源节点因移动而没有任何己方部署实体，立即结算节点回流。

---

## 十二、节点装备重新部署

```text
只能选择自己控制的节点装备或独立部署的通用装备。
支付1电荷，将其放到自己控制的另一座未满节点。
不能重新部署到无主节点或其他玩家节点。
每个装备实体每回合最多重新部署1次。
刚部署或刚转为独立部署状态的装备，本回合不能重新部署。
```

重新部署不是移动、打出、部署或重新上场，不触发相关技能。

---

## 十三、攻击节点

攻方可以选择：

```text
游侠。
独立部署且ATK大于0、卡面未禁止攻击的装备。
```

```text
携带装备的游侠使用合并后的最终ATK与最终HP。
每个部署实体每回合最多参与攻击节点1次。
每派出1个攻击实体支付1电荷。
```

占领成功后，存活攻方进入目标节点。

---

## 十四、节点回流

```text
若已控制节点上没有任何己方部署实体，该节点立即变为无主。
携带装备不能单独维持节点控制权。
节点回流时，节点放回节点牌库并重洗，随后补充中央节点区。
```

---

## 十五、节点技能与技能移动

```text
发动节点技能时，玩家必须控制该节点。
每名玩家每回合最多使用1次节点技能。
节点技能可以在行动阶段任意合法时点发动。
目标、费用和限制以节点卡面为准。
```

涉及控制权与所有权：

```text
获得控制权不改变卡牌所有权。
公共牌库模式无需记录实体所有者。
独立牌组模式必须保持卡牌所有权可识别。
技能使卡牌返回牌库时，默认返回其所有者牌库。
```

涉及装备的技能：

```text
“游侠或装备”包括独立部署装备和被游侠携带的装备。
若技能移动被携带装备，默认连同携带者一起移动，除非明确写“卸下”或“转移装备”。
若效果返回、夺取或销毁被携带装备，可以只处理该装备。
技能造成的位置变化不是节点装备的重新部署。
```

## 十六、Root Access检查

以下时点立即检查：

```text
获得节点控制权后。
攻击并占领节点后。
游侠普通移动、装备重新部署或技能移动后。
装备转换导致节点部署实体数量变化后。
节点技能结算后。
节点变为无主或回流后。
```

## 十八、扩展包动作边界

以下动作不进入当前标准版：

```text
消除手牌。
消除手牌牌库。
```

原因：它们会使有限牌库中的卡牌更快进入弃牌区。具体扩展规则以 `22-标准版基础动作池.md` 为准。
