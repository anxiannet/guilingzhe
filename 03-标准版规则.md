# 归零者｜标准版规则

> 文件定位：本文件记录 `归零者 / ZERO ACCESS` 独立项目当前标准版总规则、游戏模式、通用规则、回合流程与结算规则。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`
>
> 说明：具体卡牌费用与技能以当前卡表为准；本文件维护总规则。

---

## 一、游戏目标

```text
控制同一系统域的3座节点，立即获得 Root Access 并获胜。
```

---

## 二、标准版支持模式

标准版支持：

```text
1. 2至4人公共牌库混战。
2. 玩家各自构筑独立牌组进行对战。
```

节点可以作为对战模块使用，也可以在特定对战中不使用。

### 公共牌库混战

```text
所有玩家共用同一手牌牌库。
所有玩家共用同一公共弃牌区。
所有玩家共用节点牌库与中央节点区。
```

### 独立牌组对战

```text
每名玩家使用自己构筑的独立牌组。
每名玩家从自己的牌库抽牌。
未另行规定时使用公共弃牌区。
使用节点时，节点牌库与中央节点区仍为公共区域。
```

独立牌组所有权规则：

```text
卡牌所有权由游戏开始时所属牌组决定。
获得控制权不改变卡牌所有权。
卡牌返回牌库时，默认返回其所有者的牌库。
卡牌进入公共弃牌区时仍保留所有权信息。
游戏结束后，所有卡牌归还其原所有者。
```

## 三、手牌技能与节点技能边界

### 标准版手牌技能

标准版游侠、装备和消耗品的技能不得造成混牌：

```text
不得取得其他玩家手牌。
不得取得其他玩家场上游侠或装备的控制权。
不得将其他玩家卡牌加入自己的手牌、牌库或控制区。
不得使用需要持续记录原控制者、原持有者或归还对象的手牌技能。
```

涉及这些效果的手牌卡属于扩展包。

### 标准版节点技能

节点属于标准版，节点技能可以造成混牌或控制权变化：

```text
可以取得其他玩家游侠或装备的控制权。
可以将其他玩家控制的实体移动到发动者控制的节点。
卡面明确时，可以将其他玩家卡牌加入指定玩家手牌。
```

节点技能使用独立节点动作池，不计入标准版手牌动作覆盖率。

不使用节点的对战模式不会触发节点混牌技能，因此不受影响。

独立牌组模式中，节点技能改变控制权时必须保持卡牌所有权可识别，游戏结束后归还原所有者。

## 四、卡牌与装备类型

一级卡牌类型：

```text
游侠。
物品卡。
节点。
```

物品卡分为：

```text
装备。
消耗品。
```

### 消耗品

```text
支付费用后立即结算效果。
结算后进入公共弃牌区，除非卡面另有说明。
不作为场上实体存在。
只显示费用，不显示ATK与HP。
```

装备按部署方式分为：

```text
节点装备。
游侠装备。
通用装备。
```

### 节点装备

```text
只能作为独立实体部署到节点。
占用节点的1个部署位。
不能由游侠携带。
拥有自己的ATK与HP。
不能通过“移动”行动改变节点，只能执行“重新部署”。
```

### 游侠装备

```text
不能直接部署到节点。
打出时必须装备给同一玩家控制的1个游侠。
不占用节点部署位。
每个游侠最多携带1件装备。
不是独立实体，不能单独攻击、被攻击或控制节点。
ATK作为携带者的ATK加成。
HP作为携带者的HP加成；HP为0或卡面明确不提供HP加成的装备除外。
```

### 通用装备

```text
既可以作为节点装备独立部署，也可以作为游侠装备由游侠携带。
独立部署时占用节点1个部署位，并使用自己的ATK与HP。
由游侠携带时不占用节点部署位，其ATK与HP分别加入游侠的ATK与HP。
独立部署时按照节点装备规则执行重新部署。
```

---

## 五、节点部署位与实体

```text
每座节点最多容纳3个部署实体。
计入部署位的只有游侠与独立部署的节点装备或通用装备。
由游侠携带的装备不占部署位。
同一节点上的部署实体必须全部属于该节点控制者。
无主节点不能存在任何部署实体或被携带装备。
控制节点的必要条件是节点上至少有1个己方部署实体。
```

## 六、游戏准备

### 使用公共牌库与节点

```text
1. 将标准手牌洗混，作为公共手牌牌库。
2. 设置公共弃牌区。
3. 将标准节点牌洗混，作为节点牌库。
4. 翻开3张节点放入中央节点区。
5. 每名玩家从公共手牌牌库抽5张起始手牌。
6. 随机决定起始玩家。
```

### 使用独立牌组与节点

```text
1. 每名玩家分别洗混自己的牌组。
2. 设置公共弃牌区。
3. 将标准节点牌洗混，作为公共节点牌库。
4. 翻开3张节点放入中央节点区。
5. 每名玩家从自己的牌库抽5张起始手牌。
6. 随机决定起始玩家。
```

### 不使用节点

不设置节点牌库与中央节点区。具体胜利条件由对应无节点对战规则另行规定；不得直接套用 Root Access 节点胜利条件。

---

## 七、权限电荷

```text
每回合开始时，当前玩家获得等同于当前回合数的权限电荷，最高为10。
第10回合起，每回合获得10电荷。
回合结束时，未使用电荷全部清空。
```

---

## 八、回合流程

```text
1. 回合开始。
2. 获得权限电荷。
3. 抽1张牌。
4. 行动阶段。
5. 结束阶段。
```

抽牌来源：

```text
公共牌库混战：从公共手牌牌库抽牌。
独立牌组对战：从自己的牌库抽牌。
```

行动阶段可以执行：

```text
打出游侠。
打出节点装备。
打出游侠装备。
打出通用装备。
打出消耗品。
移动己方游侠。
重新部署己方节点装备或独立部署的通用装备。
改变同节点内通用装备的部署方式。
攻击节点。
使用节点技能。
主动结束行动阶段。
```

不使用节点的模式跳过所有节点相关行动。

---

## 九、打出游侠与装备

### 游侠

```text
支付费用，将游侠部署到无主节点，或自己控制且未满3个部署实体的节点。
部署到无主节点后，获得该节点控制权。
```

### 节点装备

```text
支付费用，将节点装备独立部署到无主节点，或自己控制且未满的节点。
占用1个部署位。
```

### 游侠装备

```text
支付费用，选择自己控制的1个未携带装备的游侠。
将装备置于该游侠下方或旁边。
```

### 通用装备

支付费用后选择：

```text
1. 独立部署，占用1个部署位。
2. 装备给自己控制的1个未携带装备的游侠。
```

刚打出的游侠本回合不能普通移动或参与攻击。刚独立部署的装备本回合不能重新部署或参与攻击。

---

## 十、携带装备规则

```text
每个游侠最多携带1件装备。
携带装备与游侠视为位于同一节点。
携带装备随游侠一起移动、攻击、进入目标节点或返回来源节点。
携带装备不单独支付移动或攻击费用，也不单独占用攻击实体数量。
```

最终属性：

```text
最终ATK = 游侠ATK + 携带装备ATK。
最终HP = 游侠HP + 携带装备HP。
```

游侠被消灭时，其携带装备同时进入公共弃牌区，除非装备技能另有说明。

标准版手牌技能可以单独返回或销毁被携带装备，但不能夺取其控制权。标准版节点技能可以按卡面夺取控制权。

---

## 十一、通用装备转换

```text
独立部署的通用装备可以改由同节点内己方游侠携带。
被游侠携带的通用装备可以卸下并独立留在同一节点。
通用装备可以在同节点己方游侠之间转交。
```

转换不支付电荷，不视为打出、部署、移动或重新部署。

---

## 十二、游侠普通移动

```text
只能移动自己控制的游侠。
目标必须是自己控制的另一座未满节点。
不能移动到无主节点或其他玩家节点。
每个游侠每回合最多普通移动1次。
刚上场的游侠本回合不能普通移动。
```

---

## 十三、节点装备重新部署

```text
只能选择自己控制的节点装备或独立部署的通用装备。
支付1电荷，将其放到自己控制的另一座未满节点。
不能重新部署到无主节点或其他玩家节点。
每个装备实体每回合最多重新部署1次。
```

---

## 十四、攻击节点

```text
游侠和符合条件的独立部署装备可以攻击节点。
每个部署实体每回合最多参与攻击节点1次。
每派出1个攻击实体支付1电荷。
```

占领成功后，存活攻方进入目标节点。

---

## 十五、节点回流

```text
若已控制节点上没有任何己方部署实体，该节点立即变为无主。
携带装备不能单独维持节点控制权。
节点回流时，节点放回节点牌库并重洗，随后补充中央节点区。
```

---

## 十六、节点技能

```text
发动节点技能时，玩家必须控制该节点。
每名玩家每回合最多使用1次节点技能。
节点技能可以在行动阶段任意合法时点发动。
目标、费用和限制以节点卡面为准。
```

节点技能可以改变其他玩家游侠或装备的控制权。

独立牌组模式下：

```text
控制权变化不改变所有权。
卡牌必须保持所有权可识别。
卡牌返回牌库时，默认返回其所有者牌库。
游戏结束后归还原所有者。
```

---

## 十七、Root Access检查

以下时点立即检查：

```text
获得节点控制权后。
攻击并占领节点后。
游侠移动、装备重新部署或技能移动后。
节点技能结算后。
节点变为无主或回流后。
```

---

## 十八、扩展包手牌动作

以下手牌动作不进入当前标准版手牌卡表：

```text
获得其他玩家手牌。
获得场上游侠。
获得场上装备。
消除手牌。
消除手牌牌库。
```

其中前三项会造成手牌卡技能混牌；后两项会加速有限牌库进入弃牌区。

节点技能不受前三项限制。
