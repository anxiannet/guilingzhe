# 归零者｜标准版规则

> 文件定位：本文件记录 `归零者 / ZERO ACCESS` 独立项目当前标准版总规则、游戏模式、通用规则、回合流程与结算规则。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`
>
> 说明：具体卡牌费用与技能以当前卡表为准；本文件维护总规则。

---

## 一、标准版模式

标准版分为两种明确独立的模式：

```text
1. 2至4人公共牌库节点混战。
2. 玩家各自构筑独立牌组的无节点对战。
```

两种模式不得混用准备流程和胜利条件。

### 公共牌库节点混战

```text
所有玩家共用同一手牌牌库。
所有玩家共用同一公共弃牌区。
使用节点牌库、中央节点区和节点技能。
胜利目标是获得 Root Access。
```

胜利条件：

```text
控制同一系统域的3座节点，立即获得 Root Access 并获胜。
```

### 独立牌组无节点对战

```text
每名玩家使用自己构筑的独立牌组。
每名玩家从自己的牌库抽牌。
不使用节点牌、节点牌库、中央节点区或节点技能。
不使用 Root Access 节点胜利条件。
独立牌组模式的组牌规则和胜利条件另行登记。
```

独立牌组所有权规则：

```text
卡牌所有权由游戏开始时所属牌组决定。
卡牌返回牌库时，默认返回其所有者的牌库。
卡牌进入公共弃牌区时仍保留所有权信息。
游戏结束后，所有卡牌归还其原所有者。
```

## 二、手牌技能与节点技能边界

### 标准版手牌技能

标准版游侠、装备和消耗品的技能不得造成混牌：

```text
不得取得其他玩家手牌。
不得取得其他玩家场上游侠或装备的控制权。
不得将其他玩家卡牌加入自己的手牌、牌库或控制区。
不得使用需要持续记录原控制者、原持有者或归还对象的手牌技能。
```

涉及这些效果的手牌卡属于扩展包。

### 标准版节点技能

节点属于公共牌库节点混战模块，节点技能可以造成混牌或控制权变化：

```text
可以取得其他玩家游侠或装备的控制权。
可以将其他玩家控制的实体移动到发动者控制的节点。
卡面明确时，可以将其他玩家卡牌加入指定玩家手牌。
```

节点技能使用独立节点动作池，不计入标准版手牌动作覆盖率。

独立牌组模式固定不使用节点，因此不会触发节点混牌技能。

## 三、卡牌与装备类型

一级卡牌类型：

```text
游侠。
物品卡。
节点。
```

物品卡分为：

```text
装备。
消耗品。
```

### 消耗品

```text
支付费用后立即结算效果。
结算后进入公共弃牌区，除非卡面另有说明。
不作为场上实体存在。
只显示费用，不显示ATK与HP。
```

装备按部署方式分为：

```text
节点装备。
游侠装备。
通用装备。
```

### 节点装备

```text
只能作为独立实体部署到节点。
占用节点的1个部署位。
不能由游侠携带。
拥有自己的ATK与HP。
不能通过“移动”行动改变节点，只能执行“重新部署”。
```

### 游侠装备

```text
不能直接部署到节点。
打出时必须装备给同一玩家控制的1个游侠。
不占用节点部署位。
每个游侠最多携带1件装备。
不是独立实体，不能单独攻击、被攻击或控制节点。
ATK作为携带者的ATK加成。
HP作为携带者的HP加成；HP为0或卡面明确不提供HP加成的装备除外。
```

### 通用装备

```text
既可以作为节点装备独立部署，也可以作为游侠装备由游侠携带。
独立部署时占用节点1个部署位，并使用自己的ATK与HP。
由游侠携带时不占用节点部署位，其ATK与HP分别加入游侠的ATK与HP。
独立部署时按照节点装备规则执行重新部署。
```

独立牌组无节点对战中，节点装备不能使用；通用装备只能以游侠装备方式使用，除非无节点规则另有规定。

## 四、公共牌库节点混战准备

```text
1. 将标准手牌洗混，作为公共手牌牌库。
2. 设置公共弃牌区。
3. 将标准节点牌洗混，作为节点牌库。
4. 翻开3张节点放入中央节点区。
5. 中央节点初始为无主、无实体。
6. 每名玩家从公共手牌牌库抽5张起始手牌。
7. 随机决定起始玩家。
```

---

## 五、独立牌组无节点对战准备

```text
1. 每名玩家分别洗混自己的独立牌组。
2. 设置公共弃牌区，或按独立牌组规则设置各自弃牌区。
3. 不设置节点牌库。
4. 不设置中央节点区。
5. 每名玩家从自己的牌库抽5张起始手牌。
6. 随机决定起始玩家。
```

独立牌组的牌组数量、同名卡上限、胜利条件与牌库耗尽规则另行登记；在这些规则确定前，不得自行建立平行规则。

---

## 六、权限电荷

```text
每回合开始时，当前玩家获得等同于当前回合数的权限电荷，最高为10。
第10回合起，每回合获得10电荷。
回合结束时，未使用电荷全部清空。
```

公共牌库节点混战费用：

```text
打出卡牌支付卡面费用。
发动节点技能支付节点卡面费用。
攻击节点时，每派出1个攻击实体支付1电荷。
游侠普通移动时，每移动1个游侠支付1电荷。
节点装备重新部署时支付1电荷。
同一节点内改变通用装备部署方式不支付电荷。
```

独立牌组无节点对战中的电荷使用方式，沿用手牌费用规则；与节点、节点移动或攻击节点有关的费用不适用。

## 七、回合流程

```text
1. 回合开始。
2. 获得权限电荷。
3. 抽1张牌。
4. 行动阶段。
5. 结束阶段。
```

抽牌来源：

```text
公共牌库节点混战：从公共手牌牌库抽牌。
独立牌组无节点对战：从自己的牌库抽牌。
```

公共牌库节点混战行动：

```text
打出游侠。
打出节点装备。
打出游侠装备。
打出通用装备。
打出消耗品。
移动己方游侠。
重新部署己方节点装备或独立部署的通用装备。
改变同节点内通用装备的部署方式。
攻击节点。
使用节点技能。
主动结束行动阶段。
```

独立牌组无节点对战不执行任何节点相关行动。其战斗与胜利流程另行登记。

---

## 八、节点部署位与实体

本节只适用于公共牌库节点混战。

```text
每座节点最多容纳3个部署实体。
计入部署位的只有游侠与独立部署的节点装备或通用装备。
由游侠携带的装备不占部署位。
同一节点上的部署实体必须全部属于该节点控制者。
无主节点不能存在任何部署实体或被携带装备。
控制节点的必要条件是节点上至少有1个己方部署实体。
```

---

## 九、打出游侠与装备

### 公共牌库节点混战

游侠：

```text
支付费用，将游侠部署到无主节点，或自己控制且未满3个部署实体的节点。
部署到无主节点后，获得该节点控制权。
```

节点装备：

```text
支付费用，将节点装备独立部署到无主节点，或自己控制且未满的节点。
占用1个部署位。
```

游侠装备：

```text
支付费用，选择自己控制的1个未携带装备的游侠。
将装备置于该游侠下方或旁边。
```

通用装备：

```text
支付费用后，可以独立部署，或装备给自己控制的1个未携带装备的游侠。
```

### 独立牌组无节点对战

```text
游侠、游侠装备、通用装备和消耗品按照无节点对战规则使用。
节点装备不能进入独立牌组或在该模式中打出。
通用装备默认只能由游侠携带，除非无节点规则另有规定。
```

## 十、携带装备规则

```text
每个游侠最多携带1件装备。
携带装备与游侠视为1个战斗单位。
携带装备不单独造成伤害，也不单独承受普通战斗伤害。
```

最终属性：

```text
最终ATK = 游侠ATK + 携带装备ATK。
最终HP = 游侠HP + 携带装备HP。
```

游侠被消灭时，其携带装备同时进入公共弃牌区，除非装备技能另有说明。

标准版手牌技能可以单独返回或销毁被携带装备，但不能夺取其控制权。标准版节点技能可以按卡面夺取控制权。

---

## 十一、公共牌库节点混战移动与攻击

本节只适用于公共牌库节点混战。

### 游侠普通移动

```text
只能移动自己控制的游侠。
目标必须是自己控制的另一座未满节点。
不能移动到无主节点或其他玩家节点。
每个游侠每回合最多普通移动1次。
刚上场的游侠本回合不能普通移动。
```

### 节点装备重新部署

```text
只能选择自己控制的节点装备或独立部署的通用装备。
支付1电荷，将其放到自己控制的另一座未满节点。
不能重新部署到无主节点或其他玩家节点。
每个装备实体每回合最多重新部署1次。
```

### 攻击节点

```text
游侠和符合条件的独立部署装备可以攻击节点。
每个部署实体每回合最多参与攻击节点1次。
每派出1个攻击实体支付1电荷。
```

占领成功后，存活攻方进入目标节点。

---

## 十二、节点回流

本节只适用于公共牌库节点混战。

```text
若已控制节点上没有任何己方部署实体，该节点立即变为无主。
携带装备不能单独维持节点控制权。
节点回流时，节点放回节点牌库并重洗，随后补充中央节点区。
```

---

## 十三、节点技能

本节只适用于公共牌库节点混战。

```text
发动节点技能时，玩家必须控制该节点。
每名玩家每回合最多使用1次节点技能。
节点技能可以在行动阶段任意合法时点发动。
目标、费用和限制以节点卡面为准。
节点技能可以改变其他玩家游侠或装备的控制权。
```

---

## 十四、Root Access检查

本节只适用于公共牌库节点混战。

以下时点立即检查：

```text
获得节点控制权后。
攻击并占领节点后。
游侠移动、装备重新部署或技能移动后。
节点技能结算后。
节点变为无主或回流后。
```

---

## 十五、扩展包手牌动作

以下手牌动作不进入当前标准版手牌卡表：

```text
获得其他玩家手牌。
获得场上游侠。
获得场上装备。
消除手牌。
消除手牌牌库。
```

其中前三项会造成手牌卡技能混牌；后两项会加速有限牌库中的卡牌进入弃牌区。

节点技能不受前三项限制，但节点只用于公共牌库节点混战。
