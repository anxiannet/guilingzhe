# 归零者｜标准版规则

> 文件定位：本文件记录 `归零者 / ZERO ACCESS` 标准版总规则、模式边界、通用流程与结算原则。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、标准版战争模式

标准版包含两种不同规模的战争：

```text
混战：全局战争。
对战：单个节点的局部战争。
```

### 混战｜全局战争

```text
人数：2至4人。
牌库：所有玩家共用公共手牌牌库。
战场：由多座节点组成的城市系统战场。
节点：使用节点卡、节点牌库、中央节点区与节点技能。
目标：跨节点争夺系统域，取得 Root Access。
```

胜利条件：

```text
控制同一系统域的3座节点，立即获得 Root Access 并获胜。
```

### 对战｜局部战争

```text
每名玩家使用自己构筑的独立牌组。
每名玩家从自己的牌库抽牌。
战场固定为1座唯一节点。
该节点只是局部战争的战场，不是节点卡。
不使用节点牌库、中央节点区或节点技能。
不使用 Root Access 的3节点胜利条件。
```

对战模式可以使用：

```text
游侠。
游侠装备。
节点装备。
通用装备。
消耗品。
```

节点装备可以加入独立牌组，并部署到唯一节点战场。

对战模式的牌组数量、同名卡上限、唯一节点容量、战斗流程与胜利条件另行登记；在正式确定前不得自行建立平行规则。

## 二、手牌技能与节点技能边界

### 标准版手牌技能

标准版游侠、装备和消耗品的技能不得造成混牌：

```text
不得取得其他玩家手牌。
不得取得其他玩家场上游侠或装备的控制权。
不得将其他玩家卡牌加入自己的手牌、牌库或控制区。
不得使用需要持续记录原控制者、原持有者或归还对象的手牌技能。
```

涉及这些效果的手牌卡属于扩展包。

### 标准版节点技能

节点卡只用于混战的全局战争。

节点技能可以造成跨玩家卡牌转移或控制权变化：

```text
可以取得其他玩家游侠或装备的控制权。
可以将其他玩家控制的实体移动到发动者控制的节点。
卡面明确时，可以将其他玩家卡牌加入指定玩家手牌。
```

节点技能使用独立节点动作池，不计入标准版手牌动作覆盖率。

对战是单个节点的局部战争，不使用节点卡或节点技能，因此不会受到节点混牌效果影响。

## 三、卡牌类型

一级卡牌类型：

```text
游侠。
物品卡。
节点。
```

物品卡分为：

```text
装备。
消耗品。
```

### 消耗品

```text
支付费用后立即结算效果。
结算后进入弃牌区，除非卡面另有说明。
不作为场上实体存在。
只显示费用，不显示ATK与HP。
```

装备按部署方式分为：

```text
节点装备。
游侠装备。
通用装备。
```

### 节点装备

```text
只能作为独立实体部署到节点。
不能由游侠携带。
拥有自己的ATK与HP。
```

混战中，节点装备部署到玩家控制的节点。

对战中，节点装备部署到唯一节点战场。

### 游侠装备

```text
不能独立部署。
打出时必须装备给自己控制的1个未携带装备的游侠。
每个游侠最多携带1件装备。
ATK和HP按卡面加入携带者属性。
```

### 通用装备

```text
可以作为节点装备独立部署。
也可以作为游侠装备由游侠携带。
```

两种模式中均可使用两种部署方式。

## 四、混战准备｜全局战争

```text
1. 将标准手牌洗混，作为公共手牌牌库。
2. 设置公共弃牌区。
3. 将标准节点卡洗混，作为节点牌库。
4. 翻开3张节点卡放入中央节点区。
5. 中央节点初始为无主、无实体。
6. 每名玩家从公共手牌牌库抽5张起始手牌。
7. 随机决定起始玩家。
```

---

## 五、对战准备｜局部战争

```text
1. 每名玩家分别洗混自己的独立牌组。
2. 设置弃牌区；具体使用公共弃牌区还是各自弃牌区，由对战规则确定。
3. 在战场中央设置1座唯一节点。
4. 不设置节点牌库。
5. 不设置中央节点区。
6. 唯一节点没有节点卡名称、系统域、节点技能或技能费用。
7. 每名玩家从自己的牌库抽5张起始手牌。
8. 随机决定起始玩家。
```

唯一节点是对战空间，不属于任何玩家，也不作为可获得、回流或进入牌库的卡牌。

---

## 六、卡牌所有权

独立牌组对战中：

```text
卡牌所有权由游戏开始时所属牌组决定。
标准版手牌技能不得改变其他玩家卡牌的控制权或持有者。
卡牌返回牌库时，默认返回其所有者的牌库。
游戏结束后，所有卡牌归还原所有者。
```

---

## 七、权限电荷

```text
每回合开始时，当前玩家获得等同于当前回合数的权限电荷，最高为10。
第10回合起，每回合获得10电荷。
回合结束时，未使用电荷全部清空。
```

两种模式中，打出卡牌均支付卡面费用。

混战额外使用：

```text
节点技能费用。
跨节点移动费用。
节点装备重新部署费用。
攻击节点费用。
```

对战只有唯一节点，不存在跨节点移动或重新部署到另一节点。

---

## 八、回合流程

```text
1. 回合开始。
2. 获得权限电荷。
3. 抽1张牌。
4. 行动阶段。
5. 结束阶段。
```

抽牌来源：

```text
混战：从公共手牌牌库抽牌。
对战：从自己的牌库抽牌。
```

---

## 九、混战行动｜全局战争

行动阶段可以执行：

```text
打出游侠。
打出节点装备。
打出游侠装备。
打出通用装备。
打出消耗品。
移动己方游侠。
重新部署己方节点装备或独立部署的通用装备。
改变同节点内通用装备的部署方式。
攻击节点。
使用节点技能。
主动结束行动阶段。
```

混战中的部署位、节点控制、节点回流、跨节点移动和 Root Access 检查按节点规则结算。

---

## 十、对战行动｜局部战争

行动阶段可以执行：

```text
打出游侠到唯一节点。
打出节点装备到唯一节点。
打出游侠装备。
将通用装备独立部署到唯一节点，或装备给游侠。
打出消耗品。
执行卡牌技能。
进行局部战斗规则允许的攻击或防守行动。
主动结束行动阶段。
```

对战中不存在：

```text
跨节点移动。
节点装备重新部署到另一节点。
节点控制权争夺。
节点回流。
节点技能。
系统域检查。
Root Access检查。
```

---

## 十一、携带装备

```text
每个游侠最多携带1件装备。
携带装备与游侠视为1个战斗单位。
最终ATK = 游侠ATK + 携带装备ATK。
最终HP = 游侠HP + 携带装备HP。
```

游侠被消灭时，其携带装备同时进入弃牌区，除非装备技能另有说明。

标准版手牌技能可以返回或销毁其他玩家的实体，但不能取得其控制权或将其加入自己区域。

---

## 十二、模式兼容审核

标准版手牌卡原则上必须兼容全局战争与局部战争。

依赖以下内容、且无法在局部战争中产生有效结果的手牌卡，默认归入扩展包：

```text
另一座节点。
跨节点移动。
节点控制权。
节点回流。
系统域。
节点技能。
```

当前处理结果：

```text
驱离枪：依赖“另一座节点”，已移入扩展包。
轻动力甲：依赖跨节点普通移动，已移入扩展包。
```

两张卡均不计入标准版卡表、标准版牌数或标准版动作覆盖率。

---

## 十三、扩展包手牌动作

以下动作不进入当前标准版手牌卡表：

```text
获得其他玩家手牌。
获得场上游侠。
获得场上装备。
消除手牌。
消除手牌牌库。
```

其中前三项会造成手牌技能混牌；后两项会加速有限牌库中的卡牌进入弃牌区。

节点技能不受前三项限制，但节点卡只用于混战的全局战争。
