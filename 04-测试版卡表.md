## 三、标准版技能生成规则

标准版游侠与物品卡技能必须遵守：

```text
标准版游侠技能，只能由基础动作池中的动作进行变体或组合形成。
标准版物品卡技能，只能由基础动作池中的动作进行变体或组合形成。
不在基础动作池中的效果，不进入标准版游侠与物品卡技能。
可部署实体的入场技能不得将己方场上实体返回手牌。
己方场上实体回手效果只允许出现在一次性物品上。
```

当前基础动作主池：

```text
抽牌。
查看牌库顶。
调整牌库顶。
返回牌库。
查看手牌。
返回手牌。
消灭。
```

备选动作：

```text
重洗牌库。
```

## 四、节点技能动作边界

```text
基础动作池只限制游侠与物品卡技能。
节点技能使用独立节点动作池，包括查看节点、保护、移动、临时强化、从弃牌区回收。
节点不再具有在线、离线状态。
节点技能不得反向扩展为标准版手牌技能。
```

移动类节点技能口径：

```text
移动类节点技能只能移动己方场上实体。
移动目标必须是你已控制且未满3个实体的节点。
不能移动到无主节点。
不能移动到其他玩家控制的节点。
移动不是上场；被移动的旧实体若本回合尚未攻击，仍可参与攻击节点。
```

## 五、三维复核原则

```text
1. 信息类技能费用最低。
2. 抽牌类技能略高于单纯查看。
3. 返回手牌类技能为中等强度。
4. 返回牌库强于返回手牌，费用更高。
5. 消灭类技能为高强度。
6. 多目标技能高于单目标技能。
7. 组合技能高于单动作技能。
8. 一次性物品无ATK / HP。
9. 装备拥有ATK / HP，并可部署在节点上。
```

## 六、卡名与技能直觉原则

```text
卡名必须让玩家能直觉联想到技能。
侦察类名字对应查看。
信使、抽取、广播类名字对应抽牌。
筛选、预读、算盘、洗牌类名字对应牌库调整。
鹰眼、爆破兵、清道夫类名字对应消灭。
回滚、回收、沙盒类名字对应返回牌库或返回手牌。
信标类名字对应撤离或返回手牌。
```

## 九、节点卡费用标注规则

```text
节点卡必须在卡面标明节点技能费用。
节点技能费用使用权限电荷图标 + 数字表示。
技能句不重复写“电荷”或“控制此节点”。
节点技能只要求玩家控制该节点，不再检查在线状态。
```

## 十、节点技能费用复核原则

```text
1电荷：查看、轻量移动、单体临时强化。
2电荷：保护节点、从弃牌区回收。
3电荷：强控制或直接影响胜利条件的节点技能。
```
