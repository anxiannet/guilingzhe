## 二、权限节点状态与胜利条件

```text
正面朝上：在线状态。
背面朝上：离线状态。
```

在线权限节点：

```text
节点信息公开。
可以部署实体。
可以争夺和被接管。
可以在已控制时发动节点技能。
可以计入 Root Access 胜利条件。
```

离线权限节点：

```text
节点信息不公开。
可以部署实体。
可以争夺和被接管。
可以保留控制归属。
不能发动节点技能。
不计入 Root Access 胜利条件。
```

离线节点隐藏信息包括：

```text
节点正面卡名。
系统域。
节点技能。
当前控制者。
该节点上的部署实体。
```

胜利句：

```text
控制同一系统域的3座在线权限节点，立即获得 Root Access 并获胜。
```

英文：

```text
Control all 3 online Access Nodes in one System Domain to gain Root Access and win.
```
