## 一、当前结论

```text
所有卡都要记录。
每张唯一卡只有一个卡号。
重复数量不拆新卡号。
同名复制牌共享同一个唯一卡号。
版本记录归属到唯一卡号下面。
版本号不拼进卡号。
游侠 R-001 至 R-023 的版本记录已全部建立。
越权模块 G-001 至 G-005 的版本记录已全部建立。
游侠中文名称按技能命名，不按英文名、星级或装备等级命名。
```

编号前缀：

```text
R = Operative / 游侠
G = Override Gear / 越权模块
N = Access Node / 权限节点
```

当前已确定：

```text
N-001 至 N-012：12 张权限节点。
R-001 至 R-023：23 张唯一游侠。
G-001 至 G-005：5 张唯一越权模块。
```

说明：

```text
标准版公共牌库总数仍为 42 张。
扩展包公共牌总数仍为 18 张。
编号记录的是“唯一卡”，不是实体复制张数。
星级代表当前装备状态，不作为游侠名称依据。
```

## 六、游侠等级与装备来源

```text
游侠星级体现装备和资源层级，不是军衔。
1星游侠：匕首类冷兵器和简易工具。
2星游侠：手枪、自制武器、低质弹药。
3星游侠：低配制式装备，弹药不足，品质不高。
4星游侠：全套制式装备，头盔、防弹衣齐全，弹药充足。
5星游侠：轻型动力机甲或高度集成外骨骼。
```

说明：

```text
以上装备层级只影响视觉表现，不作为游侠中文命名依据。
```
