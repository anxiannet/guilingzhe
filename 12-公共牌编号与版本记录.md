# 归零者｜公共牌编号与版本记录

> 文件定位：本文件记录 `归零者 / ZERO ACCESS` 公共牌库中执行者与越权模块的唯一卡号、复制数和版本记录入口。
>
> 本文件补充 `04-卡表.md`，不替代正式卡表。

---

## 一、当前结论

```text
所有卡都要记录。
每张唯一卡只有一个卡号。
重复数量不拆新卡号。
同名复制牌共享同一个唯一卡号。
版本记录归属到唯一卡号下面。
版本号不拼进卡号。
```

编号前缀：

```text
R = Runner / Operator / 执行者
G = Override Gear / 越权模块
N = Access Node / 权限节点
```

当前已确定：

```text
N-001 至 N-012：12 张权限节点。
R-001 至 R-023：23 张唯一执行者。
G-001 至 G-005：5 张唯一越权模块。
```

说明：

```text
标准版公共牌库总数仍为 42 张。
扩展包公共牌总数仍为 18 张。
编号记录的是“唯一卡”，不是实体复制张数。
```
