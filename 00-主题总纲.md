```text
归零者：产品名 / 项目名 / 阵营名。

零权限：世界观状态 / 起点状态。

```

使用原则：

```text
项目标题统一写“归零者”。

中文规则和中文世界观优先写“最高权限”。

```

## 三、玩家身份

玩家扮演一支“归零者小队 / 零权限小队”的首领。

零权限不是强者身份，而是起点。

在这个未来城市中，失去系统权限的人无法开门、无法买药、无法乘车、无法证明身份，也无法获得任何公共系统响应。系统不会逮捕他们，因为系统已经从档案里删除了他们。

他们被称为：

```text
零权限者
Zero Access
```

玩家的目标不是接受任务、拿报酬或帮别人回收物品，而是带领被系统删除的人夺取城市系统的最高权限。

```text
起点：Access Level 0
目标：最高权限 / Root Access
幻想：从系统弃民变成系统主人
```

---

## 四、核心争夺理由

未来城市由“权限系统”统治。

档案、中继、屏蔽、防火墙、传送、货运、生化改造、纳米医疗、复苏舱、武器研究、装甲研究与回收中心都接入同一套城市权限网络。拥有权限的人可以读取节点、上线节点、屏蔽节点、保护节点、调动游侠、调动装备，并把弃置资源重新纳入小队权限。

每个系统域同一时间只能存在一个最高权限。

一支小队夺取最高权限后，其他小队会被降权、封锁、踢出节点，甚至再次归零。

因此玩家对抗的理由不是单纯打架，而是：

```text
同一个系统域只承认一个最高权限。
你不夺权，别人就会用权限定义你的生存边界。
```

## 五、首版系统域

首版固定 4 个系统域，每个系统域 3 个节点。

系统域名称要像城市权限系统中的区域名。节点名要像机构、建筑或地标，不写成纯技能按钮。

```text
1. 行政域 Administrative Domain
2. 安防域 Security Domain
3. 生化域 Biotech Domain
4. 科研域 Research Domain
```

## 七、胜利幻想

控制同一系统域的三座节点，即获得该系统的最高权限。

这不是占领一块地，也不是完成外包任务，而是直接接管城市现实的一部分。

示例：

```text
控制行政域：你决定哪些节点被记录、上线或屏蔽。
控制安防域：你保护关键节点，并掌握游侠和装备的安全流转。
控制生化域：你的游侠获得改造、修复与复苏机会。
控制科研域：你的装备获得武装强化、装甲加固与回收再利用。
```

胜利句：

```text
控制同一系统域的3座在线节点，立即获得最高权限并获胜。
```

英文：

```text
Control all 3 online Nodes in one System Domain to gain Root Access and win.
```

## 九、主题边界

本主题不应变成普通赛博朋克、普通黑客游戏或普通雇佣兵撤离游戏。

必须保留的核心差异：

```text
玩家争夺的不是钱、货物、黑箱、地堡或悬赏。
玩家争夺的是城市系统的最高权限。
```

核心判断句：

```text
系统不给你权限，你就夺取最高权限。
```
