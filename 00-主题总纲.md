## 三、玩家身份

玩家扮演一支“归零者小队 / 零权限小队”的首领。

零权限不是强者身份，而是起点。

在这个未来城市中，失去系统权限的人无法开门、无法买药、无法乘车、无法证明身份，也无法获得任何公共系统响应。系统不会逮捕他们，因为系统已经从档案里删除了他们。

他们被称为：

```text
零权限者
Zero Access
```

玩家的目标不是接受任务、拿报酬或帮别人回收物品，而是带领被系统删除的人夺取城市系统的最高权限。

```text
起点：Access Level 0
目标：Root Access
幻想：从系统弃民变成系统主人
```

---

## 四、核心争夺理由

未来城市由“权限系统”统治。

门禁、医疗、无人机、信用、身份、交通、住房、武器与基础服务都接入同一套权限网络。拥有权限的人可以调用城市服务；失去权限的人连存在本身都无法证明。

每个系统域同一时间只能存在一个 Root Access。

一支小队夺取 Root Access 后，其他小队会被降权、封锁、踢出节点，甚至再次归零。

因此玩家对抗的理由不是单纯打架，而是：

```text
同一个系统域只承认一个最高权限。
你不夺权，别人就会用权限定义你的生存边界。
```

## 五、首版系统域

首版固定 4 个系统域，每个系统域 3 个权限节点。

```text
1. 门禁域 Access Domain
2. 医疗域 Medical Domain
3. 无人机域 Drone Domain
4. 信用域 Credit Domain
```

`身份域 Identity Domain` 暂不进入首版，作为扩展候选。

## 七、胜利幻想

控制同一系统域的三座权限节点，即获得该系统的 Root Access。

这不是占领一块地，也不是完成外包任务，而是直接接管城市现实的一部分。

示例：

```text
控制门禁域：所有门为你打开，敌人被锁在门外。
控制医疗域：你的同伴被优先救治，敌人的急救请求被拒绝。
控制无人机域：天空中的无人机群成为你的眼睛和手。
控制信用域：你的债务被清零，敌人的账户被冻结。
```

胜利句：

```text
控制同一系统域的3座权限节点，立即获得 Root Access 并获胜。
```

英文：

```text
Control all 3 Access Nodes in one System Domain to gain Root Access and win.
```

## 九、主题边界

本主题不应变成普通赛博朋克、普通黑客游戏或普通雇佣兵撤离游戏。

必须保留的核心差异：

```text
玩家争夺的不是钱、货物、黑箱、地堡或悬赏。
玩家争夺的是城市系统的最高权限。
```

核心判断句：

```text
系统不给你权限，你就夺取 Root Access。
```
