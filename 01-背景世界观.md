# 归零者｜背景世界观

> 主题定位：系统弃民、权限节点、Root Access 夺权。
>
> 本文件记录《归零者 / ZERO ACCESS》的世界观背景。
>
> “零权限”在本项目中是世界观起点状态，不再作为项目名使用。

---

## 一、核心背景

未来城市不再由法律、现金或武力直接统治，而是由“权限系统”统治。

档案、中继、屏蔽、防火墙、传送、货运、生化改造、纳米医疗、复苏舱、武器研究、装甲研究与回收中心，全部接入同一套城市权限网络。拥有权限的人可以读取节点、保护节点、调动游侠、调动装备，并把弃置资源重新纳入小队权限。

而失去权限的人，连自己的名字都无法证明。

他们无法开门，无法买药，无法搭乘列车，无法进入住所，也无法通过任何身份验证。系统不会逮捕他们，因为系统已经从档案里删除了他们。

他们被称为：

```text
零权限者
Zero Access
```

“零权限”不是强者称号，而是系统给他们的最低判定。

```text
Access Level: 0
```

但在城市底层，零权限者并没有消失。他们组建小队，改装设备，偷接废弃线路，驯服蜂鸟无人机，穿上机修装甲，在高塔阴影下争夺一个个节点。

他们的目标不是重新申请权限。

他们要直接夺取 Root Access。

---

## 二、玩家身份

玩家扮演一支归零者小队 / 零权限小队的首领。

你带领的是一群被系统删除、被高塔拒绝、被城市服务排除在外的人。

他们不再等待系统承认自己。

他们要把系统改写成承认自己。

---

## 三、起点与目标

```text
起点：零权限 / Access Level 0
目标：根权限 / Root Access
爽点：从系统弃民，变成系统主人
```

核心反差：

```text
你从没有任何权限开始。
你的目标是夺取最高权限。
```

---

## 四、玩家为什么对抗

每个系统域同一时间只能存在一个 Root Access。

一支小队夺取 Root Access 后，其他小队会被降权、封锁、踢出节点，甚至重新归零。

因此玩家之间不是为了普通资源互相争斗，而是在争夺“谁能定义这套系统”。

```text
同一个系统域只承认一个最高权限。
你不夺权，别人就会用权限决定你的生存边界。
```

---

## 五、控制后的幻想

每个系统域都由三座关键节点构成。控制同一系统域的三座节点，即获得该系统的 Root Access。

### 行政域 Administrative Domain

控制后，你决定哪些节点信息被记录、检索或屏蔽。系统档案、中继站与屏蔽塔都会按你的权限重排城市信息。

### 安防域 Security Domain

控制后，防火墙会保护你的关键节点，传送站与货运站会优先调动你的游侠和装备。

### 生化域 Biotech Domain

控制后，蜂鸟巢、纳米诊所与复苏舱会优先强化、修复和召回你的游侠。

### 科研域 Research Domain

控制后，武研所、装研所与回收中心会优先强化、加固和回收你的装备。

控制同一系统域的三座节点，即可获得 Root Access。

纳米诊所会修复你的游侠。
