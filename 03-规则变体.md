## 当前建议

《归零者 / ZERO ACCESS》当前已经作为独立项目维护。

当前规则处理原则：

```text
不主动读取或修改旧的 qiangdipan 仓库。
不把历史来源当成当前规则依赖。
扩展包并入标准版，形成一套 60 张标准手牌牌库。
一副标准牌既可以支持 2 到 4 人共用手牌牌库游玩，也可以作为单人对战卡组使用。
卡面文案避免使用“你的牌组 / 你的牌库 / 你的手牌牌库”。
双人 / 多人游玩时不使用节点牌库，也不从节点卡中抽取随机节点。
权限节点作为场上固定争夺对象存在。
权限节点正面朝上为在线状态，背面朝上为离线状态。
离线权限节点不能发动节点技能，也不计入 Root Access 胜利条件。
游侠、装备、设备、武器都可以作为可部署实体部署在权限节点上。
每个可部署实体都有攻击力和生命值。
每个系统域的 3 座权限节点，分别承担侦察类、进攻类、防守类技能。
打出游侠、打出越权模块、发动权限节点技能都需要消耗权限电荷。
权限电荷不能保留，按回合递增，最高10电荷。
```

## 二、可部署实体

正式术语：

```text
可部署实体
```

可部署实体包括：

```text
1. 游侠。
2. 装备。
3. 设备。
4. 武器。
```

基础规则：

```text
每个可部署实体都有攻击力和生命值。
每个可部署实体都可以部署在在线权限节点上。
```

当前卡牌类型关系：

```text
游侠牌：默认都是可部署实体。
越权模块牌：若表现为装备、设备或武器，也视为可部署实体。
一次性反制 / 指令 / 回滚类越权模块，可以不是可部署实体。
```

详细规则来源：

```text
17-部署实体与攻防生命系统.md
```

## 三、部署规则

部署定义：

```text
打出一个可部署实体时，选择1座在线权限节点，将该实体部署到该权限节点上。
```

部署限制：

```text
可部署实体只能部署到在线权限节点。
离线权限节点不能部署新实体。
```

控制关系：

```text
部署在权限节点上的实体仍由其控制者控制。
部署实体不必然等于控制该节点。
权限节点的控制者由接管结果决定。
```

节点共存：

```text
同一座权限节点上可以存在多个玩家控制的实体。
```

## 四、攻击力与生命值

每个可部署实体都有两个基础数值：

```text
攻击力 / ATK
生命值 / HP
```

攻击力用于：

```text
接管权限节点、压制敌方实体或参与节点争夺。
```

生命值用于：

```text
承受攻击、清理、回滚或系统压制。
```

基础结算口径：

```text
攻击力用于造成伤害。
生命值降到0的实体被处理，置入来源手牌弃牌区。
```

说明：

```text
具体战斗流程后续单独测试，不在当前文件一次性定死。
```

## 五、权限电荷

正式资源名：

```text
权限电荷 / Access Charge
```

卡面短称：

```text
电荷
```

世界观定义：

```text
权限电荷，是归零者小队在当前回合临时劫持到的短时权限窗口。
```

当前测试资源规则：

```text
每回合开始时，当前玩家获得等同于当前回合数的权限电荷。
本回合电荷 = 当前回合数，最高为10。
第10回合起，每回合获得10电荷。
回合结束时，未使用的权限电荷全部清空。
没有足够电荷时，不能打出对应卡牌，也不能发动对应技能。
```

权限电荷用于支付：

```text
1. 打出游侠。
2. 打出越权模块。
3. 发动权限节点技能。
```

## 八、权限节点在线 / 离线状态

权限节点有两种状态：

```text
正面朝上：在线状态。
背面朝上：离线状态。
```

在线权限节点：

```text
可以被接管。
可以部署可部署实体。
可以在已控制时发动节点技能。
可以计入 Root Access 胜利条件。
```

离线权限节点：

```text
可以保留控制归属，但不能部署新实体。
不能发动节点技能。
不计入 Root Access 胜利条件。
不能用于满足“控制同一系统域3座在线权限节点”。
```

## 九、胜利条件

中文胜利句：

```text
控制同一系统域的3座在线权限节点，立即获得 Root Access 并获胜。
```

英文胜利句：

```text
Control all 3 online Access Nodes in one System Domain to gain Root Access and win.
```

## 十、权限节点技能测试规则

### 使用时机

```text
在你的回合中，你可以花费1电荷，使用1次你已控制且在线权限节点的节点技能。
```

### 使用限制

```text
每名玩家每回合最多使用1次节点技能。
即使你控制多个在线权限节点，每回合也只能选择其中1座权限节点发动技能。
你只能发动自己已控制且在线权限节点的技能。
离线权限节点不能发动节点技能。
中央区无主权限节点不能发动节点技能。
没有合法目标时，不能发动对应节点技能。
没有足够权限电荷时，不能发动节点技能。
```

## 十一、权限节点技能分类规则

每个系统域固定包含 3 座权限节点。

每个系统域内，3 座权限节点分别承担：

```text
1. 侦察类：查看、筛选、预判手牌或信息。
2. 进攻类：强化接管、推进争夺、提升进攻游侠星级。
3. 防守类：保护己方节点、恢复游侠、阻止或削弱敌方接管。
```
