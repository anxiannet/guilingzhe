>

## 当前建议

《归零者 / ZERO ACCESS》当前已经作为独立项目维护。

当前规则处理原则：

```text
不主动读取或修改旧的 qiangdipan 仓库。
不把历史来源当成当前规则依赖。
保留当前已迁入的新仓库规则文件作为测试基础。
扩展包并入标准版，形成一套 60 张标准手牌牌库。
一副标准牌既可以支持 2 到 4 人共用手牌牌库游玩，也可以作为单人对战卡组使用。
卡面文案避免使用“你的牌组 / 你的牌库 / 你的手牌牌库”。
双人 / 多人游玩时不使用节点牌库，也不从节点卡中抽取随机节点。
权限节点作为场上固定争夺对象存在。
权限节点技能作为归零者当前重点测试规则。
每个系统域的 3 座权限节点，分别承担侦察类、进攻类、防守类技能。
打出游侠、打出越权模块、发动权限节点技能都需要消耗权限电荷。
权限电荷不能保留，按回合递增，最高10电荷。
```

## 二、权限电荷

正式资源名：

```text
权限电荷
```

英文名：

```text
Access Charge
```

卡面短称：

```text
电荷
```

世界观定义：

```text
权限电荷，是归零者小队在当前回合临时劫持到的短时权限窗口。
```

权限电荷用于支付：

```text
1. 打出游侠。
2. 打出越权模块。
3. 发动权限节点技能。
```

当前测试资源规则：

```text
每回合开始时，当前玩家获得等同于当前回合数的权限电荷。
本回合电荷 = 当前回合数，最高为10。
第10回合起，每回合获得10电荷。
回合结束时，未使用的权限电荷全部清空。
没有足够电荷时，不能打出对应卡牌，也不能发动对应技能。
```

电荷增长表：

| 回合 | 本回合可用电荷 |
|---:|---:|
| 第1回合 | 1电荷 |
| 第2回合 | 2电荷 |
| 第3回合 | 3电荷 |
| 第4回合 | 4电荷 |
| 第5回合 | 5电荷 |
| 第6回合 | 6电荷 |
| 第7回合 | 7电荷 |
| 第8回合 | 8电荷 |
| 第9回合 | 9电荷 |
| 第10回合及以后 | 10电荷 |

详细规则来源：


## 三、费用口径

### 游侠费用

游侠打出费用默认等于当前星级。

```text
1星游侠：花费1电荷。
2星游侠：花费2电荷。
3星游侠：花费3电荷。
4星游侠：花费4电荷。
5星游侠：花费5电荷。
```

游侠入场效果：

```text
游侠的入场效果包含在该游侠的打出费用中，不额外支付技能费用。
```

### 越权模块费用

当前越权模块费用草案：

```text
G-001 隔离沙盒：2电荷。
G-002 零日芯片：1电荷。
G-003 权限束带：2电荷。
G-004 离线地堡：2电荷。
G-005 强制回滚扇区：3电荷。
```

### 权限节点技能费用

权限节点技能默认费用：

```text
花费1电荷。
```

说明：

```text
若卡框已有费用槽，可不在规则文字中重复写“花费X电荷”。
若没有费用槽，必须在技能文字中写明费用。
```

## 四、标准手牌牌库

扩展包并入标准版后，当前标准手牌牌库为 60 张：

```text
游侠：55 张
越权模块：5 张
合计：60 张
```

使用方式：

```text
2 到 4 人共用游玩：所有玩家共用同一副标准手牌牌库与同一个手牌弃牌区。
对战卡组游玩：每名玩家可以各自使用一副标准手牌牌库与对应手牌弃牌区。
```

因此卡面描述采用模式中性写法：

```text
手牌牌库
手牌弃牌区
控制者手牌
来源手牌弃牌区
来源手牌牌库
己方权限节点
```

卡面避免写法：

```text
你的牌组
你的牌库
你的手牌牌库
```

## 五、资源节奏目的

```text
第1回合只能做低成本动作。
第3回合开始可以稳定打出3星游侠。
第5回合开始可以直接打出5星游侠。
第10回合后进入满权限对抗阶段。
```

核心体验：

```text
你不是积攒资源，而是在每个回合燃烧当场偷来的权限窗口。
```
