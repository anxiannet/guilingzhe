# 归零者｜规则变体

> 文件定位：本文件记录 `归零者 / ZERO ACCESS` 独立项目当前规则映射、测试规则与未来变体候选。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 当前建议

《归零者 / ZERO ACCESS》当前已经作为独立项目维护。

当前规则处理原则：

```text
不主动读取或修改旧的 qiangdipan 仓库。
不把历史来源当成当前规则依赖。
扩展包并入标准版，形成一套 60 张标准手牌牌库。
一副标准牌既可以支持 2 到 4 人共用手牌牌库游玩，也可以作为单人对战卡组使用。
卡面文案避免使用“你的牌组 / 你的牌库 / 你的手牌牌库”。
双人 / 多人游玩时不使用节点牌库，也不从节点卡中抽取随机节点。
权限节点作为场上固定争夺对象存在。
权限节点正面朝上为在线状态，背面朝上为离线状态。
离线权限节点不能发动节点技能，也不计入 Root Access 胜利条件。
每个系统域的 3 座权限节点，分别承担侦察类、进攻类、防守类技能。
打出游侠、打出越权模块、发动权限节点技能都需要消耗权限电荷。
权限电荷不能保留，按回合递增，最高10电荷。
```

## 二、权限电荷

正式资源名：

```text
权限电荷 / Access Charge
```

卡面短称：

```text
电荷
```

世界观定义：

```text
权限电荷，是归零者小队在当前回合临时劫持到的短时权限窗口。
```

当前测试资源规则：

```text
每回合开始时，当前玩家获得等同于当前回合数的权限电荷。
本回合电荷 = 当前回合数，最高为10。
第10回合起，每回合获得10电荷。
回合结束时，未使用的权限电荷全部清空。
没有足够电荷时，不能打出对应卡牌，也不能发动对应技能。
```

权限电荷用于支付：

```text
1. 打出游侠。
2. 打出越权模块。
3. 发动权限节点技能。
```

## 三、费用口径

### 游侠费用

游侠打出费用默认等于当前星级。

```text
1星游侠：花费1电荷。
2星游侠：花费2电荷。
3星游侠：花费3电荷。
4星游侠：花费4电荷。
5星游侠：花费5电荷。
```

游侠入场效果：

```text
游侠的入场效果包含在该游侠的打出费用中，不额外支付技能费用。
```

### 越权模块费用

```text
G-001 隔离沙盒：2电荷。
G-002 零日芯片：1电荷。
G-003 权限束带：2电荷。
G-004 离线地堡：2电荷。
G-005 强制回滚扇区：3电荷。
```

### 权限节点技能费用

```text
权限节点技能默认花费1电荷。
只有已控制且在线的权限节点可以发动节点技能。
离线权限节点不能发动节点技能。
```

## 四、标准手牌牌库

扩展包并入标准版后，当前标准手牌牌库为 60 张：

```text
游侠：55 张
越权模块：5 张
合计：60 张
```

使用方式：

```text
2 到 4 人共用游玩：所有玩家共用同一副标准手牌牌库与同一个手牌弃牌区。
对战卡组游玩：每名玩家可以各自使用一副标准手牌牌库与对应手牌弃牌区。
```

卡面描述采用模式中性写法：

```text
手牌牌库

来源手牌弃牌区
来源手牌牌库
己方权限节点
```

卡面避免写法：

```text
你的牌组
你的牌库
你的手牌牌库
```

## 五、权限节点在线 / 离线状态

权限节点有两种状态：

```text
正面朝上：在线状态。
背面朝上：离线状态。
```

在线权限节点：

```text
可以被接管。
可以在已控制时发动节点技能。
可以计入 Root Access 胜利条件。
```

离线权限节点：

```text
可以保留控制归属，但不能发动节点技能。
不计入 Root Access 胜利条件。
不能用于满足“控制同一系统域3座在线权限节点”。
```

状态处理口径：

```text
权限节点翻到正面，表示该节点上线。
权限节点翻到背面，表示该节点离线。
若节点离线，其卡面技能视为关闭。
```

## 六、胜利条件

中文胜利句：

```text
控制同一系统域的3座在线权限节点，立即获得 Root Access 并获胜。
```

英文胜利句：

```text
Control all 3 online Access Nodes in one System Domain to gain Root Access and win.
```

说明：

```text
只控制3座节点但其中有离线节点时，不能获得 Root Access。
必须同时控制同一系统域内3座在线权限节点。
```

## 七、权限节点技能测试规则

### 使用时机

```text
在你的回合中，你可以花费1电荷，使用1次你已控制且在线权限节点的节点技能。
```

### 使用限制

```text
每名玩家每回合最多使用1次节点技能。
即使你控制多个在线权限节点，每回合也只能选择其中1座权限节点发动技能。
你只能发动自己已控制且在线权限节点的技能。
离线权限节点不能发动节点技能。
中央区无主权限节点不能发动节点技能。
没有合法目标时，不能发动对应节点技能。
没有足够权限电荷时，不能发动节点技能。
```

### 与主要行动的关系

```text
节点技能不占用主要行动。
```

卡面写法：

```text
若卡框已有费用槽，卡面只写具体效果。
若没有费用槽，卡面写作“花费1电荷：……” 。
```

## 八、权限节点技能分类规则

每个系统域固定包含 3 座权限节点。

每个系统域内，3 座权限节点分别承担：

```text
1. 侦察类：查看、筛选、预判手牌或信息。
2. 进攻类：强化接管、推进争夺、提升进攻游侠星级。
3. 防守类：保护己方节点、恢复游侠、阻止或削弱敌方接管。
```

## 十一、游侠技能与节点技能边界

游侠技能可以操作：

```text
手牌牌库
手牌
手牌弃牌区
场上游侠
接管权限节点
越权模块
非实体支援游侠
```

游侠技能不再操作：

```text
节点牌库
节点卡顶牌
节点卡置顶 / 置底
节点牌库重洗
随机节点卡抽取
```

## 十二、核心体验

玩家不是抢物品，而是抢控制现实的权限。

```text
系统不给你权限，你就夺取 Root Access。
```

资源体验：

```text
你不是积攒资源，而是在每个回合燃烧当场偷来的权限窗口。
```
