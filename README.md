# 归零者 / ZERO ACCESS

系统弃民、越权装备与城市权限夺取的战术卡牌项目。

```text
从零权限，到根权限。
From Zero Access to Root Access.
```

---

## 项目定位

《归零者 / ZERO ACCESS》原为《夕妖：抢地盘》的强主题换皮探索，现迁入独立仓库继续维护。

玩家扮演一支零权限小队的首领，带领被城市权限系统删除的人夺取城市系统的 Root Access。

---

## 当前原则

```text
1. 新仓库以“归零者 / ZERO ACCESS”为唯一项目主题。
2. 不在本仓库维护《抢地盘》正式规则。
3. 归零者变体规则只在本仓库维护。
4. 卡面视觉方向为高科技二维封存体卡牌。
5. 后续卡牌制作拆成：卡框、插图、人工文字排版三层。
```
