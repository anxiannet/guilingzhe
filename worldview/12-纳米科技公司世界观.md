# 归零者｜纳米科技公司世界观

> 文件定位：本文件记录《归零者 / ZERO ACCESS》中“纳米科技公司”的世界观设定、视觉标识、相关装备来源与卡牌应用边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`
>
> 说明：本文件是世界观与视觉设定文档，不新增规则字段；具体卡牌费用、ATK、HP 与技能以当前卡表为准。

---

## 一、当前结论

```text
纳米科技公司是《归零者》世界中的企业级生化与微型装备研发势力。
它不是政府机构，也不是普通军火商。
它专长于纳米医疗、微型无人机、生化载荷、节点级诊疗设备与可封存战术装备。
```

项目内定位：

```text
纳米科技公司是生化域、医疗科技和微型战术装备的重要来源之一。
蜂鸟、纳米诊所、生化载荷、微型暗杀设备等内容，可与该公司建立来源关系。
```

## 二、公司性质

```text
纳米科技公司是一家掌握城市纳米医疗、生化载荷封装、微型无人机制造和节点级生命维护技术的超大型技术企业。
它向城市权限系统、安防承包商、医疗节点和灰色权限市场提供设备。
```

它的公开身份：

```text
医疗科技公司。
纳米修复服务商。
城市诊疗节点供应商。
微型设备制造商。
```

它的真实边界：

```text
生化载荷封装。
微型无人机暗杀平台。
权限节点生物数据采集。
可封存战术装备生产。
灰色测试项目外包。
```

## 四、核心技术方向

### 1. 纳米医疗

```text
用于修复、稳定、重构游侠或节点内生物实体的技术。
公开用于医疗，实际也可用于权限实验和战斗恢复。
```

相关内容：

```text
纳米诊所。
复苏舱。
生物档案库。
```

### 2. 微型无人机

```text
用于节点侦查、追踪、暗杀、投送和监控的微型飞行装备。
不同型号负责不同任务。
```

型号分工：

```text
蜂鸟：高速攻击 / 暗杀 / 生化载荷投送。
蜻蜓无人机：低噪巡查 / 追踪 / 悬停监视。
蝶翼侦察机：无感潜入 / 静音侦查 / 环境伪装。
```

### 3. 生化载荷封装

```text
将细菌、病毒、神经干扰剂或其他生化载荷压缩封装进微型设备中。
这些载荷可用于暗杀、驱逐、污染节点或制造战术威慑。
```

注意：

```text
当前标准版规则尚未引入中毒、感染、持续伤害、污染等复杂状态。
因此生化载荷优先作为世界观、插画和卡面风味信息使用。
如后续加入生化状态，应单独建立规则变体或扩展规则。
```

## 五、蜂鸟项目

```text
蜂鸟是纳米科技公司研发的高速微型攻击无人机。
它不是普通侦查无人机，而是暗杀型节点装备。
```

核心特征：

```text
体型极小。
速度极快。
攻击力不输狙击步枪。
可进行节点突袭。
可执行定点清除。
部分型号可装载细菌、病毒等生化载荷。
```

标准版卡表口径：

```text
卡名：蜂鸟。
英文名：Hummingbird。
类型：装备。
子类型：无人机。
定位：高速微型攻击无人机 / 暗杀型节点装备。
ATK：3。
HP：1。
规则技能：无。
技能描述位置：用于简短世界观介绍。
```

技能描述位置建议文案：

```text
纳米科技公司高速暗杀无人机，可执行节点突袭与生化载荷投送。
```

## 六、视觉设计要求

### 纳米科技公司 Logo

```text
蜂鸟卡面必须出现纳米科技公司 Logo。
Logo 是企业来源标识，不是规则图标。
```

Logo 使用位置建议：

```text
机体外壳微型企业标识。
无人机腹部或翼根处的纳米科技公司徽记。
技能描述区附近的小型来源标识。
卡面角落的企业资产码。
```

Logo 风格建议：

```text
冷白、青蓝或医疗绿发光线条。
纳米网格、六边形、微粒环、细胞膜或双螺旋抽象元素。
不能像普通互联网公司 Logo。
不能像传统红十字医疗标志。
不能像普通军队徽章。
```

蜂鸟视觉关键词：

```text
高速残影。
微型机体。
锋利翼片。
针状攻击结构。
透明生化载荷舱。
冷白青蓝纳米光。
纳米科技公司 Logo。
高科技封存装备质感。
```

## 八、规则边界

```text
纳米科技公司设定不自动新增规则字段。
公司 Logo 不作为规则图标。
生化载荷不自动等于中毒、感染或持续伤害。
蜂鸟当前规则技能为空。
蜂鸟的技能描述位置只写世界观介绍，不写规则效果。
```

如后续开发生化机制，应单独定义：

```text
中毒。
感染。
污染节点。
生化载荷。
持续伤害。
免疫或清除。
```

在当前标准版中，以上概念只作为设定储备，不进入默认规则。

## 九、卡牌应用原则

```text
带有纳米科技公司来源的卡牌，可以在视觉上出现企业 Logo、资产编号、纳米网格或医疗 / 生化科技元素。
但卡面不能堆叠大段企业说明。
```

卡面文案应保持短句：

```text
纳米科技公司高速暗杀无人机，可执行节点突袭与生化载荷投送。
```

避免写成：

```text
纳米科技公司研发的第X代微型高机动攻击侦查复合无人机，拥有多种复杂战术模式……
```
