# 归零者｜权限电荷与费用系统

> 文件定位：本文件定义《归零者 / ZERO ACCESS》的资源系统、世界观解释、费用口径与卡面写法。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、资源名称

正式资源名：

```text
权限电荷
```

英文名：

```text
Access Charge
```

卡面短称：

```text
电荷
```

允许英文缩写：

```text
AC
```

---

## 二、世界观定义

权限电荷不是钱，也不是普通能量。

```text
权限电荷，是归零者小队在当前回合临时劫持到的短时权限窗口。
```

它代表：

```text
1. 当回合可用的临时认证额度。
2. 可消耗的越权算力。
3. 被偷来的系统响应窗口。
4. 释放封存卡牌所需的权限脉冲。
5. 发动节点技能所需的短时控制权。
```

核心解释：

```text
归零者不是“存钱发育”，而是在每个回合临时抢到一段权限窗口，并在窗口关闭前把它烧完。
```

---

## 三、资源用途

权限电荷用于支付三类费用：

```text
1. 打出游侠。
2. 打出越权模块。
3. 发动权限节点技能。
```

说明：

```text
没有足够权限电荷时，不能打出对应卡牌，也不能发动对应技能。
```

---

## 四、回合电荷规则

权限电荷不能保留。

```text
每回合开始时，当前玩家获得等同于当前回合数的权限电荷。
回合结束时，未使用的权限电荷全部清空。
第10回合起，每回合获得10电荷。
```

电荷增长表：

| 回合 | 本回合可用电荷 |
|---:|---:|
| 第1回合 | 1电荷 |
| 第2回合 | 2电荷 |
| 第3回合 | 3电荷 |
| 第4回合 | 4电荷 |
| 第5回合 | 5电荷 |
| 第6回合 | 6电荷 |
| 第7回合 | 7电荷 |
| 第8回合 | 8电荷 |
| 第9回合 | 9电荷 |
| 第10回合及以后 | 10电荷 |

简写公式：

```text
本回合电荷 = 当前回合数，最高为10。
```

## 五、默认费用口径

### 游侠费用

游侠打出费用默认等于当前星级。

```text
1星游侠：花费1电荷。
2星游侠：花费2电荷。
3星游侠：花费3电荷。
4星游侠：花费4电荷。
5星游侠：花费5电荷。
```

说明：

```text
星级代表当前装备状态。
装备越完整，释放和维持所需的权限电荷越高。
```

### 越权模块费用

越权模块费用不按星级计算，按模块强度单独标注。

当前测试默认：

```text
普通越权模块：花费2电荷。
反制型轻模块：花费1电荷。
强制清场 / 大范围回滚模块：花费3电荷。
```

### 权限节点技能费用

权限节点技能默认费用：

```text
花费1电荷。
```

说明：

```text
节点已经被控制，因此发动节点技能不需要重新接管，但仍需要消耗短时权限电荷维持指令。
```

## 七、费用与技能的关系

当前口径：

```text
打出游侠需要支付释放费用。
打出越权模块需要支付模块费用。
发动权限节点技能需要支付节点技能费用。
```

游侠入场效果：

```text
游侠的入场效果包含在该游侠的打出费用中，不额外支付技能费用。
```

独立发动的技能：

```text
若某张卡未来拥有可重复发动技能，则卡面必须写明“花费X电荷：……”。
```

权限节点技能：

```text
权限节点技能统一视为主动技能，默认写作“花费1电荷：……”。
```

## 八、卡面写法

推荐卡面短写：

```text
花费1电荷：查看手牌牌库顶1张，可置顶或置底。
```

或使用费用图标后，文字区只写效果：

```text
查看手牌牌库顶1张，可置顶或置底。
```

说明：

```text
若卡框已有费用槽，可不在规则文字中重复写“花费X电荷”。
若没有费用槽，必须在技能文字中写明费用。
```

禁止：

```text
不要使用法力、金币、能量、水晶等通用资源名。
不要把权限电荷写成信用域的货币。
不要让费用说明占满技能区。
不要在同一张卡上同时堆叠费用槽和长费用句。
不要设计需要跨回合存电荷的效果。
```

## 九、设计目的

权限电荷系统要解决三件事：

```text
1. 限制高星游侠过早登场。
2. 让越权模块成为需要取舍的战术资源。
3. 让权限节点技能有使用成本，避免每回合无脑发动。
```

简化后的节奏目标：

```text
第1回合只能做低成本动作。
第3回合开始可以稳定打出3星游侠。
第5回合开始可以直接打出5星游侠。
第10回合后进入满权限对抗阶段。
```

核心体验：

```text
你不是积攒资源，而是在每个回合燃烧当场偷来的权限窗口。
```
