# 归零者｜物品卡术语范围

> 文件定位：本文件定义《归零者 / ZERO ACCESS》中“物品卡 / Gear”的正式术语范围。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、当前结论

```text
正式牌类名：物品卡 / Gear。
旧称“越权模块”废止，不再作为卡面主类型。
```

物品卡是所有非游侠手牌卡的统称。

```text
凡不是游侠的手牌卡，都归入物品卡。
```

---

## 二、为什么改名

```text
“越权模块”偏抽象，玩家不容易理解。
“物品卡”更直接，能自然覆盖装备、设备、工具、武器、权限卡、纳米机器人、电磁手雷等非人物卡。
```

“越权”仍然保留为世界观动作：

```text
越权接入。
越权部署。
越权权限。
越权指令。
```

但不再作为玩家读卡时必须理解的一级牌类名。

---

## 三、正式定义

物品卡，是归零者小队用于接入、部署、侦察、防御、干扰、破坏、恢复、移动、反制或临时获取权限的非游侠物品。

它可以是实体，也可以是一次性物品；可以是硬件，也可以是权限载体；可以是武器，也可以是工具。

---

## 四、包含范围

物品卡包括但不限于：

```text
装备
设备
工具
武器
权限卡
纳米机器人
电磁手雷
程序芯片
数据钥匙
临时权限载体
防御工事
侦察无人机
一次性物品
```

示例：

```text
蜂鸟无人机：物品卡 / 设备 / 锚点。
防御地堡：物品卡 / 设备 / 防御工事。
零日芯片：物品卡 / 程序芯片 / 一次性反制。
电磁手雷：物品卡 / 武器 / 一次性干扰。
权限卡：物品卡 / 权限载体。
纳米机器人：物品卡 / 纳米设备。
```

---

## 五、与游侠的边界

游侠：

```text
人物牌。
行动个体。
有身份、外号、技能、费用、攻击力和生命值。
```

物品卡：

```text
非游侠手牌卡。
可以是装备、设备、工具、武器、权限卡、纳米机器人、电磁手雷或其他物品。
可以有费用、攻击力、生命值和技能。
也可以是一次性效果牌。
```

---

## 六、与可部署实体的关系

不是所有物品卡都必须部署。

物品卡分为两大类：

```text
1. 实体物品：可以部署在权限节点上，拥有攻击力和生命值。
2. 一次性物品：只结算效果，不留在权限节点上，可以没有攻击力和生命值。
```

实体物品例子：

```text
蜂鸟无人机。
防御地堡。
部署式工具。
固定设备。
自动武器。
```

一次性物品例子：

```text
零日芯片。
强制回滚扇区。
电磁手雷。
一次性权限卡。
临时指令。
```

## 七、卡面类型写法

卡面主类型统一写：

```text
物品卡
```

卡面副类型可以写：

```text
设备
装备
工具
武器
权限卡
纳米设备
程序芯片
电磁手雷
防御工事
锚点
一次性
```

示例卡面类型：

```text
物品卡 / 设备 / 锚点
物品卡 / 设备 / 防御工事
物品卡 / 程序芯片 / 一次性
物品卡 / 武器 / 电磁手雷
物品卡 / 权限卡 / 一次性
```

## 八、禁止误解

```text
不要再把“越权模块”作为正式牌类名。
不要让玩家必须理解“模块”才能读懂卡牌。
不要把装备、工具、武器、权限卡另立为与物品卡并列的大类。
不要为每种非游侠物品创建新的一级卡牌类型。
```

统一原则：

```text
一级类型只有：游侠、物品卡、节点。
物品卡内部再区分装备、设备、工具、武器、权限卡、纳米机器人、电磁手雷等副类型。
```
