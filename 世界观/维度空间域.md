## 二、关联卡牌

### 正版储存卡

```text
正版储存卡 / Licensed Storage Card。
技能：二维压缩。
选择场上1个游侠或装备，将其返回其控制者手牌。
```

### 破解储存卡

```text
破解储存卡 / Cracked Storage Card。
技能：越权压缩。
选择场上1个游侠或装备，将其加入你的手牌。
```

### 驱离枪

```text
驱离枪 / Displacement Gun。
类型：装备。
装备分类：游侠装备。
子类型：武器。
费用：3。
ATK 2 / HP 0。
技能：定向驱离。
此装备给游侠装备时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

世界观解释：

```text
驱离枪是由游侠直接操作的定向空间驱离武器，不能脱离使用者独立部署。
它向目标写入单一方向的强制坐标偏移，使目标无法维持当前节点的展开状态，只能在其控制者拥有权限的另一座节点重新展开。
“定向驱离”强调其驱离方向受目标控制者现有节点权限约束，不是任意传送。
驱离枪自身没有独立防护结构，因此HP为0，也不会提高携带者生命值。
它只通过武器输出接口为携带者提供ATK加成。
```

费用定位：

```text
驱离枪费用为3。
它需要场上已有游侠并占用该游侠唯一装备位。
HP为0，只提供+2 ATK。
定向驱离只有在对手存在另一座合法节点时才能执行。
因此属于中费战术装备，而不是独立强战斗实体。
```

技术边界：

```text
驱离枪不能独立占据节点。
驱离枪不能改变目标控制权。
定向驱离不是返回手牌，也不是消灭或销毁。
同节点内转交驱离枪不会重新执行定向驱离。
```

## 三、核心节点：维相枢纽

```text
节点名：维相枢纽 / Phase Nexus。
系统域：维度空间。
技能名：全域重映射。
费用：10电荷。
技能：选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点，包括离线节点。若移动到你控制的节点，你获得该实体的控制权。
```

## 四、设计分层

```text
正版储存卡：合法降维，返回原控制者手牌。
驱离枪：由游侠携带的3费驱离武器，提供+2 ATK并定向驱离敌方目标。
破解储存卡：越权降维，加入发动者手牌。
维相枢纽：任意位置重映射，并可夺取控制权。
```

---

## 五、驱离枪视觉规范

```text
紧凑型定向武器结构，明确由游侠持握或安装在手臂、肩部武器接口。
枪口释放扇形坐标错位波，而非实体弹丸。
命中目标后，目标轮廓发生横向相位撕裂并被推出当前节点坐标。
不得表现为可独立悬浮驻守的炮塔、无人机或节点设备。
```
