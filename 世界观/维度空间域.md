> 文件定位：本文件记录《归零者 / ZERO ACCESS》中“维度空间”系统域的世界观定位、技术边界、节点方向与视觉规范。

## 二、系统域定位

```text
维度空间域负责城市中的空间折叠、实体封存、维度转换、展开许可和高价值资产调度。
它不是普通物流网络，也不是虚拟数据空间。
它直接管理现实实体在不同空间维度状态之间的转换权限。
```

核心能力：

```text
三维实体降维。
二维封存体管理。
空间折叠运输。
实体相位锁定。
合法展开区域认证。
城市资产快速调用。
危险实体隔离。
```

---

## 三、与维相科技的关系

```text
维相科技是维度空间域最重要的技术供应商与基础设施承包商。
维相科技提供维相转换标准、封存介质、展开接口与相位锁定设备。
维度空间域属于城市权限系统，不属于维相科技私人所有。
```

权限关系：

```text
城市系统负责签发维度封存许可、空间展开许可与资产调用权限。
维相科技负责建设和维护相关技术设施。
归零者通过夺取维度空间域节点，获得绕过合法许可的空间调用能力。
```

## 四、与其他系统域的边界

与纳米科技域：

```text
纳米科技域负责纳米医疗、生化载荷和微型无人机。
维度空间域负责这些实体如何被封存、运输和重新展开。
```

与科研域：

```text
科研域负责实验、研发和数据分析。
维度空间域负责空间维度基础设施与实体状态转换。
```

与行政域：

```text
行政域负责许可、身份和城市管理程序。
维度空间域执行具体的封存与展开权限。
```

与安防域：

```text
安防域负责监控、封锁和威胁处置。
维度空间域可以对危险实体执行隔离和降维封存，但不承担常规治安管理。
```

## 五、卡牌关联

当前关联卡牌：

```text
空白储存卡 / Blank Storage Card。
```

关联原因：

```text
空白储存卡使用维相转换技术，将场上游侠或装备强制降维为二维封存体。
目标进入发动者手牌，表示其调用权限被重新绑定。
```

当前规则技能：

```text
二维压缩
选择场上1个游侠或装备，将其加入你的手牌。
```

说明：

```text
空白储存卡属于手牌中的一次性物品，不是维度空间域节点。
卡牌关联系统域只表示技术来源和世界观归属，不新增系统域规则字段。
```

## 六、节点设计方向

维度空间域最终需要3座节点。

节点能力必须体现城市级空间基础设施，可围绕以下方向设计：

```text
实体封存：使游侠或装备离开节点并进入合法封存区域。
空间展开：从特定区域调用游侠或装备进入节点。
相位转移：在己方控制节点之间移动实体。
隔离空间：暂时阻断实体与当前节点的互动。
维度校准：恢复或调整受空间效果影响的实体状态。
```

限制：

```text
本文件暂不确定3座节点名称。
本文件暂不确定节点技能费用。
本文件不允许节点技能直接跳过 Root Access 胜利条件。
不得把维度空间域写成时间旅行、平行宇宙或无限复制系统。
```

## 七、视觉规范

核心视觉：

```text
城市空间被切分为透明薄层。
三维结构向二维平面坍缩。
建筑边缘发生折叠与错层。
实体被固定在透明维度膜中。
冷白、青蓝与青紫空间光。
极度洁净、精确、受控的基础设施感。
```

禁止方向：

```text
普通宇宙星空。
魔法传送门。
玄幻异世界裂缝。
时间机器。
普通数据中心。
霓虹黑客房间。
```

系统域徽章方向：

```text
由立方体压缩成平面的几何符号。
或由多层透明平面交叠形成的空间坐标标识。
徽章必须简洁，可与其他系统域图标并列使用。
```
