> 文件定位：本文件记录《归零者 / ZERO ACCESS》中“维度空间”系统域的世界观定位、技术边界、关联卡牌、节点方向与视觉规范。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`
>
> 说明：具体卡牌费用、属性和技能文本以 `05-标准版卡表.md` 为准。

---

## 一、系统域定位

```text
系统域名称：维度空间。
英文名：Dimensional Space Domain。
主要技术来源：维相科技的维相转换技术。
当前核心节点：维相枢纽 / Phase Nexus。
```

维度空间域负责三维实体降维、二维封存、空间折叠运输、相位锁定、空间驱离以及权限重映射。

---

## 二、企业分工

### 维相科技 / DPT

```text
角色：核心空间技术提供商。
提供内容：维相转换标准、定向坐标偏移技术、相位控制模块、授权接口与安全规范。
不负责：驱离枪的武器化设计、枪体制造、装配和销售。
```

### 阈锋军械 / THRESHOLD ARMAMENTS

```text
简称：TA。
角色：武器装备制造公司。
负责内容：武器平台设计、枪体结构、能源系统、火控接口、生产、装配和销售。
核心能力：将外部授权技术封装为适合游侠直接使用的战术装备。
```

合作关系：

```text
驱离枪由阈锋军械制造。
维相科技只提供空间技术支持和关键模块授权。
技术支持方不等同于制造商。
```

## 三、关联卡牌

### 正版储存卡

```text
正版储存卡 / Licensed Storage Card。
技能：二维压缩。
选择场上1个游侠或装备，将其返回其控制者手牌。
```

### 破解储存卡

```text
破解储存卡 / Cracked Storage Card。
技能：越权压缩。
选择场上1个游侠或装备，将其加入你的手牌。
```

### 驱离枪

```text
驱离枪 / Displacement Gun。
类型：装备。
装备分类：游侠装备。
子类型：武器。
费用：3。
ATK 2 / HP 0。
技能：定向驱离。
制造商：阈锋军械 / TA。
技术支持：维相科技 / DPT。
```

技能：

```text
此装备给游侠装备时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

世界观解释：

```text
驱离枪是阈锋军械基于DPT授权的定向坐标偏移模块开发的战术武器。
阈锋军械负责将维相技术整合进可由游侠直接操作的枪械平台。
维相科技提供核心空间技术，但不生产驱离枪成品。
```

技术边界：

```text
驱离枪不能独立占据节点。
驱离枪不能改变目标控制权。
定向驱离不是返回手牌，也不是消灭或销毁。
同节点内转交驱离枪不会重新执行定向驱离。
```

## 四、核心节点：维相枢纽

```text
节点名：维相枢纽 / Phase Nexus。
系统域：维度空间。
技能名：全域重映射。
费用：10电荷。
技能：选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点，包括离线节点。若移动到你控制的节点，你获得该实体的控制权。
```

## 五、设计分层

```text
正版储存卡：维相科技制造的合法维度封存介质。
驱离枪：阈锋军械制造、维相科技提供技术支持的3费游侠武器。
破解储存卡：基于维相介质破解形成的越权封存工具。
维相枢纽：城市维度空间基础设施节点。
```

## 六、驱离枪视觉规范

```text
紧凑型定向武器结构，明确由游侠持握或安装在手臂、肩部武器接口。
枪体采用阈锋军械的黑灰金属、楔形切面与高对比警戒刻线。
维相科技标识只出现在相位核心、授权模块或接口认证位置。
不得把整件武器表现为维相科技自产设备。
```
