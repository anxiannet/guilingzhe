```text
系统域名称：维度空间。
```

维度空间域当前尚未补足3座节点，因此暂不能作为可完成最高权限的正式系统域投入标准节点牌库。

## 二、系统域定位

```text
维度空间域负责城市中的空间折叠、实体封存、维度转换、展开许可、高价值资产调度与实体所有权重映射。
它直接管理现实实体在不同空间维度状态之间的转换权限。
```

核心能力：

```text
三维实体降维。
二维封存体管理。
空间折叠运输。
实体相位锁定。
合法展开区域认证。
跨节点空间调度。
物质所有权与调用权限重映射。
```

---

## 三、与维相科技的关系

```text
维相科技是维度空间域最重要的技术供应商与基础设施承包商。
维相科技提供维相转换标准、封存介质、展开接口与相位锁定设备。
维度空间域属于城市权限系统，不属于维相科技私人所有。
```

## 四、当前关联卡牌

### 正版储存卡

```text
正版储存卡 / Licensed Storage Card。
技能：二维压缩。
选择场上1个游侠或装备，将其返回其控制者手牌。
```

世界观解释：

```text
正版储存卡执行合法降维，只能把目标压缩回其合法权限持有者的封存序列。
它不改写目标所有权。
```

### 破解储存卡

```text
破解储存卡 / Cracked Storage Card。
技能：越权压缩。
选择场上1个游侠或装备，将其加入你的手牌。
```

世界观解释：

```text
破解储存卡绕过所有权验证，在目标被降维后把调用权限重绑给发动者。
因此目标进入发动者手牌。
```

### 相位转运卡

```text
相位转运卡 / Phase Transfer Card。
技能：空间传送。
选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点，包括离线节点。
```

技术边界：

```text
空间传送不是部署，不触发入场技能。
空间传送不改变目标控制者。
```

## 五、核心节点：维相枢纽

```text
节点名：维相枢纽 / Phase Nexus。
系统域：维度空间。
技能名：全域重映射。
费用：10电荷。
技能：选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点，包括离线节点。若移动到你控制的节点，你获得该实体的控制权。
```

规则边界：

```text
目标节点不得超过3个实体。
若传入发动者控制节点，目标改由发动者控制。
若传入其他节点，控制权不改变。
若传入无主节点，目标原控制者获得该节点控制权。
该效果视为移动，不视为部署，不触发入场技能。
原节点清空时立即结算节点回流。
```

## 六、设计分层

```text
正版储存卡：合法降维，返回原控制者手牌。
破解储存卡：越权降维，加入发动者手牌。
相位转运卡：改变所在节点，不改变控制权。
维相枢纽：改变所在节点，并可在传入己方节点时夺取控制权。
```

## 七、视觉规范

核心视觉：

```text
城市空间被切分为透明薄层。
三维结构向二维平面坍缩。
建筑边缘发生折叠与错层。
实体被固定在透明维度膜中。
冷白、青蓝与青紫空间光。
```

正版储存卡：

```text
完整认证层、合法许可标识、稳定蓝白封存光。
```

破解储存卡：

```text
认证层被篡改、权限锁断裂、青紫错层光与所有权链路重写。
```

禁止方向：

```text
普通宇宙星空。
魔法传送门。
时间机器。
普通数据中心。
霓虹黑客房间。
```
