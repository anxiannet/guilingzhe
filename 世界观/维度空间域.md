## 二、系统域定位

```text
维度空间域负责城市中的空间折叠、实体封存、维度转换、展开许可、高价值资产调度与实体所有权重映射。
它不是普通物流网络，也不是虚拟数据空间。
它直接管理现实实体在不同空间维度状态之间的转换权限。
```

核心能力：

```text
三维实体降维。
二维封存体管理。
空间折叠运输。
实体相位锁定。
合法展开区域认证。
城市资产快速调用。
危险实体隔离。
跨节点空间调度。
物质所有权与调用权限重映射。
```

---

## 三、与维相科技的关系

```text
维相科技是维度空间域最重要的技术供应商与基础设施承包商。
维相科技提供维相转换标准、封存介质、展开接口与相位锁定设备。
维度空间域属于城市权限系统，不属于维相科技私人所有。
```

权限关系：

```text
城市系统负责签发维度封存许可、空间展开许可与资产调用权限。
维相科技负责建设和维护相关技术设施。
归零者通过夺取维度空间域节点，获得绕过合法许可的空间调用能力。
```

## 四、当前关联卡牌

### 空白储存卡

```text
空白储存卡 / Blank Storage Card。
技能：二维压缩。
选择场上1个游侠或装备，将其加入你的手牌。
```

世界观解释：

```text
目标被强制降维为二维封存体，并重新绑定后续调用权。
```

### 相位转运卡

```text
相位转运卡 / Phase Transfer Card。
技能：空间传送。
选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点，包括离线节点。
```

世界观解释：

```text
相位转运卡是维相科技生产的单次空间调度介质。
它可以解除目标原有空间坐标，并在另一座合法节点重新展开。
它不改变目标实体的控制权。
```

技术边界：

```text
空间传送不是部署，不触发入场技能。
空间传送不是复制、消灭、销毁或回手。
相位转运卡只能移动1个实体，不能重写所有权。
```

## 五、核心节点：维相枢纽

节点名：

```text
维相枢纽
```

英文名：

```text
Phase Nexus
```

系统域：

```text
维度空间
```

节点定位：

```text
维相枢纽是城市级空间运输、实体调度与权限重映射网络的核心节点。
它不是普通传送门，也不是维相科技总部。
它连接城市中的空间展开接口、相位锚点、实体调度通道与资产权限数据库。
```

节点技能名：

```text
全域重映射
```

节点技能费用：

```text
10电荷
```

节点技能文本：

```text
选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点，包括离线节点。若移动到你控制的节点，你获得该实体的控制权。
```

世界观解释：

```text
维相枢纽先对目标执行相位锁定，再解除其原有空间坐标。
当目标在发动者控制的节点重新展开时，枢纽同时改写其资产所有权与调用权限。
因此目标在抵达后成为发动者控制的实体。
```

规则边界：

```text
可以选择任意玩家控制的游侠或装备。
目标节点可以是在线节点或离线节点。
目标节点不得超过3个实体。
若目标移动到发动者控制的节点，目标在进入后改由发动者控制，因此不形成混合控制者节点。
若目标移动到其他玩家控制的节点，目标控制权不改变，且目标必须与该节点现有实体属于同一控制者。
若目标移动到无主节点，目标原控制者获得该节点控制权。
全域重映射视为移动，不视为部署，不触发入场技能。
若原节点因目标离开而没有任何实体，立即结算节点回流。
维相枢纽自身必须在线并由发动者控制，才能发动节点技能。
```

设计作用：

```text
相位转运卡提供4费单体空间移动。
维相枢纽提供10费空间移动加控制权夺取。
节点技能因此具有终局级清除、夺取与重新部署价值。
```

## 七、后续节点设计方向

维度空间域最终需要3座节点，目前仍需补充2座。

后续节点能力可围绕：

```text
实体封存。
从特定区域展开实体。
隔离空间。
维度校准。
空间坐标防护。
```

限制：

```text
后续节点不得重复维相枢纽的控制权夺取定位。
节点技能不得直接跳过最高权限胜利条件。
不得加入时间旅行、平行宇宙或无限复制。
```

## 八、视觉规范

核心视觉：

```text
城市空间被切分为透明薄层。
三维结构向二维平面坍缩。
建筑边缘发生折叠与错层。
实体被固定在透明维度膜中。
冷白、青蓝与青紫空间光。
极度洁净、精确、受控的基础设施感。
```

相位转运卡视觉重点：

```text
多个节点坐标投影在透明维相介质上。
一个实体轮廓从原坐标切片消失，并在另一坐标切片重新出现。
不得表现为魔法传送门。
```

维相枢纽视觉重点：

```text
多层城市空间坐标在中央汇聚。
透明空间切片形成大型枢纽结构。
多个展开端口指向不同节点坐标。
核心区域同时显示相位锁定、空间重组与权限重写过程。
不得表现为单一圆形魔法传送门。
```

禁止方向：

```text
普通宇宙星空。
魔法传送门。
玄幻异世界裂缝。
时间机器。
普通数据中心。
霓虹黑客房间。
```
