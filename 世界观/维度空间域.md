## 二、系统域定位

```text
维度空间域负责城市中的空间折叠、实体封存、维度转换、展开许可、高价值资产调度与实体所有权重映射。
它直接管理现实实体在不同空间维度状态之间的转换权限。
```

核心能力：

```text
三维实体降维。
二维封存体管理。
空间折叠运输。
实体相位锁定。
跨节点空间驱离。
物质所有权与调用权限重映射。
```

---

## 三、与维相科技的关系

```text
维相科技是维度空间域最重要的技术供应商与基础设施承包商。
维相科技提供维相转换标准、封存介质、展开接口、相位锁定设备与空间驱离武器。
维度空间域属于城市权限系统，不属于维相科技私人所有。
```

## 四、当前关联卡牌

### 正版储存卡

```text
正版储存卡 / Licensed Storage Card。
技能：二维压缩。
选择场上1个游侠或装备，将其返回其控制者手牌。
```

世界观解释：

```text
正版储存卡执行合法降维，只能把目标压缩回其合法权限持有者的封存序列。
```

### 破解储存卡

```text
破解储存卡 / Cracked Storage Card。
技能：越权压缩。
选择场上1个游侠或装备，将其加入你的手牌。
```

世界观解释：

```text
破解储存卡绕过所有权验证，在目标被降维后把调用权限重绑给发动者。
```

### 驱离枪

```text
驱离枪 / Displacement Gun。
类型：装备。
子类型：武器。
技能：强制驱离。
此装备上场时，选择1个对手游侠或装备，将其移动到其控制者控制的另一座可容纳该实体的节点，包括离线节点。
```

世界观解释：

```text
驱离枪不是传送工具，而是节点清场武器。
它向目标写入强制坐标偏移，使目标无法维持在当前节点的展开状态，只能在其控制者拥有权限的另一座节点重新展开。
```

技术边界：

```text
驱离枪不能把目标送入发动者控制的节点。
驱离枪不能改变目标控制权。
驱离不是返回手牌，也不是消灭或销毁。
```

## 五、核心节点：维相枢纽

```text
节点名：维相枢纽 / Phase Nexus。
系统域：维度空间。
技能名：全域重映射。
费用：10电荷。
技能：选择场上1个游侠或装备，将其移动到任意可容纳该实体的节点，包括离线节点。若移动到你控制的节点，你获得该实体的控制权。
```

规则边界：

```text
目标节点不得超过3个实体。
传入发动者控制节点时，目标改由发动者控制。
传入其他节点时不改变控制权。
移动不视为部署，不触发入场技能。
原节点清空时立即结算节点回流。
```

## 六、设计分层

```text
正版储存卡：合法降维，返回原控制者手牌。
驱离枪：以武器效果强制目标离开当前节点，但不改变所有权。
破解储存卡：越权降维，加入发动者手牌。
维相枢纽：任意位置重映射，并可在传入己方节点时夺取控制权。
```

## 七、视觉规范

核心视觉：

```text
城市空间被切分为透明薄层。
三维结构向二维平面坍缩。
建筑边缘发生折叠与错层。
实体被固定在透明维度膜中。
冷白、青蓝与青紫空间光。
```

驱离枪视觉重点：

```text
短枪或紧凑型定向武器结构。
枪口不是发射实体弹丸，而是释放扇形坐标错位波。
命中目标后，目标轮廓发生横向相位撕裂并被推出当前节点坐标。
必须像战术武器，不能像储存卡、门禁卡或普通传送设备。
```

禁止方向：

```text
普通宇宙星空。
魔法传送门。
时间机器。
普通数据中心。
霓虹黑客房间。
```
