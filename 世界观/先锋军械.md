# 归零者｜先锋军械世界观

> 文件定位：本文件记录《归零者 / ZERO ACCESS》中武器装备制造企业“先锋军械”的企业定位、技术边界、产品方向与视觉语言。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、正式名称

```text
中文名：先锋军械
英文名：VANGUARD ARMAMENTS
简称：VA
```

名称含义：

```text
“先锋”代表率先进入冲突区域、突破防线并建立战术优势。
“军械”直接说明其武器、装甲与节点防御装备制造商身份。
```

---

## 二、企业定位

```text
先锋军械是一家专门制造游侠武器、节点武装与战术装备的军械企业。
它不以发明底层科技见长，而以快速吸收、封装和武器化外部技术著称。
```

核心能力：

```text
武器平台设计。
能源与火控系统整合。
外部技术模块武器化。
游侠装备人体工学适配。
节点防御设备量产。
城市权限接口兼容。
```

产品逻辑：

```text
技术公司提供核心模块、标准与授权接口。
先锋军械负责将模块整合成可部署、可维护、可量产的战术装备。
制造商与技术支持方必须分开登记。
```

英文名：ACCESS BREAKER
产品代号：VA-AB1
类型：游侠武器
制造商：先锋军械 / VA

结构简单，可在零权限环境启动，容易拆修和改装。

## 四、轻动力甲

```text
中文名：轻动力甲
英文名：LIGHT POWERED ARMOR
产品代号：待定
类型：游侠装备
子类型：装甲
制造商：先锋军械 / VA
```

产品定位：

```text
轻动力甲是先锋军械面向游侠推出的轻型动力防护装备。
它通过腿部助力、脊柱支撑和轻量装甲提高携带者的生存能力与普通移动效率。
```

标准版候选属性：

```text
编号：G-S04。
唯一：否。
复制数：2。
费用：3。
ATK 0 / HP 2。
```

装备功能：

```text
此游侠进行普通移动时，不支付电荷。
```

功能边界：

```text
该文本是装备基础功能，不设置技能名。
不增加游侠每回合可普通移动的次数。
刚上场状态仍然禁止普通移动。
技能移动不属于普通移动，不适用该功能。
```

视觉方向：

```text
轻量外骨骼与局部装甲结合。
腿部助力结构必须清晰可见。
保持游侠机动轮廓，不做重型全身装甲。
使用先锋军械的黑灰金属、楔形切面和警戒刻线。
```

## 七、驱离枪

```text
产品名：驱离枪 / Displacement Gun。
类型：游侠装备。
制造商：先锋军械 / VA。
联合研发：维相科技 / DPT。
```

```text
维相科技提供定向坐标偏移模块、维相转换标准和授权接口。
先锋军械负责枪体、能源系统、火控系统、模块封装、生产与销售。
驱离枪利用定向相位偏移迫使目标退出当前节点。
```

## 八、产品层级

```text
断权枪：最知名的普及型产品。
轻动力甲：常规防护与机动型游侠装备。
防御地堡：高强度节点防御装备。
驱离枪：与维相科技合作的技术联名产品。
终阈武装：最强的唯一旗舰装备。
```

## 九、视觉语言

```text
黑灰深色金属主体。
楔形切面与向前收束的锋利轮廓。
短促、高对比的警戒刻线。
模块化插槽与明确的技术授权接口。
结构克制、工业化、适合量产，不做夸张霓虹装饰。
```

卡面标识：

```text
制造商主标识：VA / VANGUARD ARMAMENTS。
合作技术标识置于模块或认证区域，不与制造商标识同级。
```
