# 归零者｜阈锋军械世界观

> 文件定位：本文件记录《归零者 / ZERO ACCESS》中武器装备制造企业“阈锋军械”的企业定位、技术边界、产品方向与视觉语言。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、正式名称

```text
中文名：阈锋军械
英文名：THRESHOLD ARMAMENTS
简称：TA
```

名称含义：

```text
“阈”代表权限阈值、系统边界与武器启动条件。
“锋”代表将技术转化为直接战术输出的武器化能力。
```

---

## 二、企业定位

```text
阈锋军械是一家专门制造游侠武器、节点武装与战术装备的军械企业。
它不以发明底层科技见长，而以快速吸收、封装和武器化外部技术著称。
```

核心能力：

```text
武器平台设计。
能源与火控系统整合。
外部技术模块武器化。
游侠装备人体工学适配。
节点防御设备量产。
城市权限接口兼容。
```

---

## 三、企业特征

阈锋军械的产品逻辑：

```text
不讨论技术是否应该被武器化，只判断它是否能在权限冲突中形成优势。
```

它经常与掌握核心技术的企业合作：

```text
技术公司提供核心模块、标准与授权接口。
阈锋军械负责将模块整合成可部署、可维护、可量产的战术装备。
```

因此：

```text
阈锋军械是制造商。
合作企业可以是技术支持方。
制造商与技术支持方必须在卡表中分开登记。
```

---

## 四、驱离枪

```text
产品名：驱离枪 / Displacement Gun。
类型：游侠装备。
制造商：阈锋军械 / TA。
技术支持：维相科技 / DPT。
```

分工：

```text
维相科技提供定向坐标偏移模块、维相转换标准和授权接口。
阈锋军械负责枪体、能源系统、火控系统、模块封装、生产与销售。
```

产品定位：

```text
驱离枪不是传送设备，而是利用定向相位偏移迫使目标退出当前节点的战术武器。
```

---

## 五、产品方向

阈锋军械后续产品可覆盖：

```text
游侠武器。
节点防御装备。
空间驱离武器。
权限干扰弹药。
火控与目标锁定模块。
基于其他企业技术授权的联合装备。
```

不建议方向：

```text
普通现实枪械复刻。
无技术来源说明的万能武器。
纯雇佣兵风格军火公司。
只强调杀伤、不体现权限系统和节点争夺的装备。
```

## 六、视觉语言

```text
黑灰深色金属主体。
楔形切面与向前收束的锋利轮廓。
短促、高对比的警戒刻线。
模块化插槽与明确的技术授权接口。
结构克制、工业化、适合量产，不做夸张霓虹装饰。
```

品牌标识方向：

```text
一个由门槛线与枪口楔形组合而成的符号。
图形表现“越过阈值后立即形成武力输出”。
```

卡面标识：

```text
制造商主标识：TA / THRESHOLD ARMAMENTS。
合作技术标识置于模块或认证区域，不与制造商标识同级。
```

阈锋军械已确认为驱离枪制造商。

阈锋军械负责制造，维相科技负责空间技术支持。
