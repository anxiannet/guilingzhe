# 归零者｜阈锋军械世界观

> 文件定位：本文件记录《归零者 / ZERO ACCESS》中武器装备制造企业“阈锋军械”的企业定位、技术边界、产品方向与视觉语言。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、正式名称

```text
中文名：阈锋军械
英文名：THRESHOLD ARMAMENTS
简称：TA
```

名称含义：

```text
“阈”代表权限阈值、系统边界与武器启动条件。
“锋”代表将技术转化为直接战术输出的武器化能力。
```

---

## 二、企业定位

```text
阈锋军械是一家专门制造游侠武器、节点武装与战术装备的军械企业。
它不以发明底层科技见长，而以快速吸收、封装和武器化外部技术著称。
```

核心能力：

```text
武器平台设计。
能源与火控系统整合。
外部技术模块武器化。
游侠装备人体工学适配。
节点防御设备量产。
城市权限接口兼容。
```

---

## 三、企业特征

阈锋军械的产品逻辑：

```text
不讨论技术是否应该被武器化，只判断它是否能在权限冲突中形成优势。
```

它经常与掌握核心技术的企业合作：

```text
技术公司提供核心模块、标准与授权接口。
阈锋军械负责将模块整合成可部署、可维护、可量产的战术装备。
```

因此：

```text
阈锋军械是制造商。
合作企业可以是技术支持方。
制造商与技术支持方必须在卡表中分开登记。
```

## 四、拳头明星产品：断权枪

```text
中文名：断权枪
英文名：ACCESS BREAKER
产品代号：TA-AB1
类型：游侠武器
制造商：阈锋军械 / TA
```

产品定位：

```text
阈锋军械最知名、产量最大、流通时间最长的基础战术武器。
它不是为了单纯击穿装甲，而是向目标设备或防御系统注入短时权限干扰脉冲，使其授权链路失效。
```

归零者为什么熟悉它：

```text
结构简单，零权限环境下仍可启动。
不依赖持续城市网络认证。
模块通用，容易拆修和改装。
旧型号大量流入城市底层与回收市场。
绝大多数归零者小队都见过、用过或拆过断权枪。
```

在归零者中的地位：

```text
断权枪不是最强的武器，却是最像“归零者武器”的产品。
它让没有系统授权的人，也能暂时让有权限的设备失去响应。
```

常见俗称：

```text
断权。
老阈锋。
AB-1。
```

当前规则状态：

```text
断权枪已确定为阈锋军械的世界观明星产品。
尚未登记为标准版卡牌。
费用、ATK、HP、装备分类与技能尚未设定。
```

## 五、最强游侠装备：终阈武装

```text
中文名：终阈武装
英文名：FINAL THRESHOLD RIG
产品代号：TA-FTX
类型：旗舰级游侠装备
制造商：阈锋军械 / TA
唯一属性：是
```

产品定位：

```text
终阈武装是阈锋军械制造过的最强游侠装备。
它不是单一枪械，而是一套覆盖游侠全身的模块化战斗外骨骼、火控核心、能源系统与武器接口总成。
它把游侠本人的行动能力、武器输出、防护结构和权限干扰能力整合为一个战斗平台。
```

为什么叫“终阈”：

```text
普通阈锋产品用于跨越某一道权限阈值。
终阈武装被设计为跨越最后一道战术阈值：让单个游侠具备正面突破重防节点的能力。
```

世界观地位：

```text
断权枪是阈锋军械最出名的产品。
终阈武装是阈锋军械最强的产品。
驱离枪是阈锋军械与维相科技合作的技术联名产品。
三者定位不能混淆。
```

稀有性：

```text
终阈武装无法稳定量产。
现存完整套件极少，每套都拥有独立序列与维护记录。
它应具有“唯一”属性，标准牌组中最多1张。
```

外观方向：

```text
全身式深色金属外骨骼。
肩部与前臂具有可替换武器接口。
胸口为高密度权限能源核心。
轮廓应像一名游侠被升级为移动战术平台，而不是普通动力装甲。
保持阈锋军械的楔形切面、警戒刻线和模块化结构。
```

当前规则状态：

```text
终阈武装已确定为阈锋军械最强游侠装备。
尚未登记为标准版卡牌。
费用、ATK、HP与技能尚未设定。
若进入标准版卡表，应具有“唯一”属性并只配置1张。
```

## 六、驱离枪

```text
产品名：驱离枪 / Displacement Gun。
类型：游侠装备。
制造商：阈锋军械 / TA。
技术支持：维相科技 / DPT。
```

分工：

```text
维相科技提供定向坐标偏移模块、维相转换标准和授权接口。
阈锋军械负责枪体、能源系统、火控系统、模块封装、生产与销售。
```

产品定位：

```text
驱离枪不是传送设备，而是利用定向相位偏移迫使目标退出当前节点的战术武器。
它属于阈锋军械在断权枪之后推出的技术联名型战术产品。
```

---

## 七、产品方向

阈锋军械后续产品可覆盖：

```text
游侠武器。
节点防御装备。
空间驱离武器。
权限干扰弹药。
火控与目标锁定模块。
基于其他企业技术授权的联合装备。
```

不建议方向：

```text
普通现实枪械复刻。
无技术来源说明的万能武器。
纯雇佣兵风格军火公司。
只强调杀伤、不体现权限系统和节点争夺的装备。
```

## 八、视觉语言

```text
黑灰深色金属主体。
楔形切面与向前收束的锋利轮廓。
短促、高对比的警戒刻线。
模块化插槽与明确的技术授权接口。
结构克制、工业化、适合量产，不做夸张霓虹装饰。
```

品牌标识方向：

```text
一个由门槛线与枪口楔形组合而成的符号。
图形表现“越过阈值后立即形成武力输出”。
```

断权枪视觉方向：

```text
紧凑、厚实、耐拆修的短枪结构。
外露权限脉冲线圈与可替换能源匣。
枪体有大量旧型号维修标记和非官方改装接口。
不得表现为豪华高端武器或普通实弹手枪。
```

卡面标识：

```text
制造商主标识：TA / THRESHOLD ARMAMENTS。
合作技术标识置于模块或认证区域，不与制造商标识同级。
```
