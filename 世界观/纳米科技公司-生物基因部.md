## 一、部门定位

```text
部门名称：生物基因部
所属企业：纳米科技公司
职能：研发游侠使用的短时生理强化制剂、基因调节剂、战术激素与基因备份技术。
```

生物基因部负责研发，不作为独立制造商登记。产品企业来源统一写“纳米科技公司”，研发部门写“生物基因部”。

## 二、战术激素统一口径

```text
类型：一次性物品
子类型：激素
使用对象：己方游侠
持续时间：当前回合
研发：纳米科技公司·生物基因部
```

统一限制：

```text
每个游侠每回合最多使用1张激素。
```

该限制同时适用于同种与不同种激素。一个游侠在同一回合内不能同时使用两张激素，激素效果不能叠加。

当前产品：

```text
力量激素：选择1个己方游侠。本回合中，其获得+2 ATK。
耐力激素：选择1个己方游侠。本回合中，其获得+2 HP。
强化激素：选择1个己方游侠。本回合中，其获得+1 ATK和+1 HP。
```

## 三、基因备份项目

```text
产品名：基因备份卡
英文名：GENE BACKUP CARD
制造商：纳米科技公司
研发部门：生物基因部
联合研发：维相科技 / DPT
```

产品定位：

```text
基因备份卡是面向游侠的单次生命数据保护设备。
它在装备时持续记录游侠的基因结构、生物身份与有效封存索引。
当游侠即将被消灭时，备份卡调用维相接口，将游侠重新压缩回其控制者的合法手牌序列。
```

生物基因部负责：

```text
游侠基因提取。
生物身份编码。
战斗期间的基因状态校验。
备份数据完整性验证。
```

维相科技负责：

```text
二维封存介质。
紧急数据回传接口。
合法权限持有者识别。
实体重构与重新展开接口。
```

技术边界：

```text
基因备份卡不是复制游侠。
同一游侠不会同时存在两个有效实体。
技能触发后，原游侠返回其控制者手牌，备份卡因完成单次回传而报废并进入公共弃牌区。
```

标准版候选规则：

```text
此游侠将被消灭时，改为将该游侠返回其控制者手牌，并将此装备置入公共弃牌区。
```
