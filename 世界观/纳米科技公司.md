> 说明：本文件是世界观与视觉设定文档，不新增规则字段；具体卡牌费用、ATK、HP 与技能以当前卡表为准。

## 一、当前结论

```text
纳米科技公司是《归零者》世界中的企业级生化与微型装备研发势力。
它不是政府机构，也不是普通军火商。
它专长于纳米医疗、微型无人机、生化载荷、节点级诊疗设备与可封存战术装备。
蜂鸟是该公司量产成熟的商业战略产品。
```

项目内定位：

```text
纳米科技公司是生化域、医疗科技、微型战术装备与纳米科技域的重要来源之一。
蜂鸟、蜂鸟巢、纳米诊所、生化载荷、微型暗杀设备等内容，可与该公司建立来源关系。
```

## 四、核心技术方向

### 1. 纳米医疗

```text
用于修复、稳定、重构游侠或节点内生物实体的技术。
公开用于医疗，实际也可用于权限实验和战斗恢复。
```

相关内容：

```text
纳米诊所。
复苏舱。
生物档案库。
```

### 2. 微型无人机

```text
用于节点侦查、追踪、暗杀、投送和监控的微型飞行装备。
不同型号负责不同任务。
```

型号分工：

```text
蜂鸟：高速攻击 / 暗杀 / 生化载荷投送 / 量产补给。
蜻蜓无人机：低噪巡查 / 追踪 / 悬停监视。
蝶翼侦察机：无感潜入 / 静音侦查 / 环境伪装。
```

### 3. 生化载荷封装

```text
将细菌、病毒、神经干扰剂或其他生化载荷压缩封装进微型设备中。
这些载荷可用于暗杀、驱逐、污染节点或制造战术威慑。
```

注意：

```text
当前标准版规则尚未引入中毒、感染、持续伤害、污染等复杂状态。
因此生化载荷优先作为世界观、插画和卡面风味信息使用。
如后续加入生化状态，应单独建立规则变体或扩展规则。
```

### 4. 蜂鸟巢指挥网络

```text
蜂鸟巢是纳米科技公司用于管理蜂鸟无人机群的节点级指挥设施。
它不是普通无人机停机坪，而是具备权限调度、战术回收、载荷重组和群体召回能力的纳米科技节点。
```

功能定位：

```text
召回分散蜂鸟。
集中终局战力。
作为10费用锚点。
加快游戏收束。
帮助玩家完成胜利条件。
超过节点上限的蜂鸟回到蜂鸟巢控制者手牌。
```

### 5. 商业战略产品补给链

```text
蜂鸟不是一次性原型机，而是纳米科技公司成熟量产的商业战略产品。
当前部署体被销毁，只代表本次部署的机体损毁。
纳米科技公司的供应链会将新的同型号蜂鸟封存体重新补入手牌牌库。
```

规则对应：

```text
此装备被销毁时，将其放到手牌牌库底。
```

世界观含义：

```text
被销毁的是当前部署体。
回到手牌牌库的是后续补入供应序列的新同型号产品。
这不是同一架蜂鸟复活，而是量产战略产品的持续投放。
```

## 五、蜂鸟项目

```text
蜂鸟是纳米科技公司研发的高速微型攻击无人机。
它不是普通侦查无人机，而是暗杀型节点装备。
它也是纳米科技公司成熟的商业战略产品。
```

核心特征：

```text
体型极小。
速度极快。
攻击力不输狙击步枪。
可进行节点突袭。
可执行定点清除。
部分型号可装载细菌、病毒等生化载荷。
被销毁后可由供应链补入新的同型号产品。
```

标准版卡表口径：

```text
卡名：蜂鸟。
英文名：Hummingbird。
类型：装备。
子类型：无人机。
复制数：4张。
定位：高速微型攻击无人机 / 暗杀型节点装备。
ATK：3。
HP：1。
规则技能：此装备被销毁时，将其放到手牌牌库底。
技能描述位置：用于简短世界观介绍。
```

技能描述位置建议文案：

```text
纳米科技公司高速暗杀无人机，可执行节点突袭与生化载荷投送。
```

设定说明建议文案：

```text
蜂鸟是纳米科技公司量产成熟的商业战略产品。当前部署体被销毁后，新的同型号产品会重新进入供应序列。
```

## 七、蜂鸟巢节点

```text
蜂鸟巢是纳米科技公司管理蜂鸟无人机群的节点设施。
它可以在终局阶段发出紧急召回指令，将分散在不同节点的蜂鸟重新聚集到蜂鸟巢。
若蜂鸟巢无法容纳全部蜂鸟，超出上限的蜂鸟会回到蜂鸟巢控制者手牌。
```

标准版候选口径：

```text
节点名：蜂鸟巢。
英文名：Hummingbird Hive。
系统域：纳米科技。
节点技能名：紧急召回。
节点技能费用：10。
节点技能：将场上所有“蜂鸟”移动到蜂鸟巢。若蜂鸟巢超过实体上限，超出的“蜂鸟”返回蜂鸟巢控制者手牌。
```

设计作用：

```text
作为10费用锚点。
用于终局阶段集中蜂鸟战力。
加快游戏收束。
帮助玩家完成胜利条件。
```

规则边界：

```text
紧急召回不直接宣告胜利。
技能结算后仍需按标准 Root Access 胜利条件检查。
蜂鸟巢最多保留3个实体。
若召回后超过实体上限，由发动技能的玩家选择保留在蜂鸟巢的蜂鸟。
超出的蜂鸟返回蜂鸟巢控制者手牌。
```

## 八、术语边界

```text
游侠 / 人物单位：被消灭。
装备 / 物品实体：被销毁。
一次性物品：被使用后进入弃牌区。
```

说明：

```text
游侠被消灭，表示现场行动个体失去战斗能力，部署状态结束。
装备被销毁，表示当前部署设备损毁，实体结构不可继续使用。
蜂鸟被销毁后回到手牌牌库，不代表同一架蜂鸟复活，而代表纳米科技公司补入新的同型号产品。
```

## 十一、规则边界

```text
纳米科技公司设定不自动新增规则字段。
公司 Logo 不作为规则图标。
生化载荷不自动等于中毒、感染或持续伤害。
蜂鸟的规则技能只处理自身被销毁后的牌库去向。
蜂鸟巢的紧急召回是节点技能，不是蜂鸟自身技能。
紧急召回不直接宣告胜利，仍需按 Root Access 胜利条件检查。
超出蜂鸟巢实体上限的蜂鸟返回蜂鸟巢控制者手牌。
```

如后续开发生化机制，应单独定义：

```text
中毒。
感染。
污染节点。
生化载荷。
持续伤害。
免疫或清除。
```

在当前标准版中，以上概念只作为设定储备，不进入默认规则。

```text
纳米科技公司高速暗杀无人机，可执行节点突袭与生化载荷投送。
```

```text
紧急召回：将场上所有“蜂鸟”移动到蜂鸟巢。超过上限的返回蜂鸟巢控制者手牌。
```
