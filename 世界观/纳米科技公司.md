# 归零者｜纳米科技公司世界观

> 文件定位：本文件记录《归零者 / ZERO ACCESS》中“纳米科技公司”的世界观设定、视觉标识、相关装备来源与卡牌应用边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`
>
> 说明：本文件是世界观与视觉设定文档，不新增规则字段；具体卡牌费用、ATK、HP 与技能以当前卡表为准。

---

## 一、当前结论

```text
纳米科技公司是《归零者》世界中的企业级生化与微型装备研发势力。
它不是政府机构，也不是普通军火商。
它专长于纳米医疗、微型无人机、生化载荷、节点级诊疗设备与可封存战术装备。
```

项目内定位：

```text
纳米科技公司是生化域、医疗科技、微型战术装备与纳米科技域的重要来源之一。
蜂鸟、蜂鸟巢、纳米诊所、生化载荷、微型暗杀设备等内容，可与该公司建立来源关系。
```

## 二、公司性质

```text

它向城市权限系统、安防承包商、医疗节点和灰色权限市场提供设备。
```

它的公开身份：

```text
医疗科技公司。
纳米修复服务商。
城市诊疗节点供应商。
微型设备制造商。
```

它的真实边界：

```text
生化载荷封装。
微型无人机暗杀平台。
权限节点生物数据采集。
可封存战术装备生产。
灰色测试项目外包。
```

## 四、核心技术方向

### 1. 纳米医疗

```text
用于修复、稳定、重构游侠或节点内生物实体的技术。
公开用于医疗，实际也可用于权限实验和战斗恢复。
```

相关内容：

```text
纳米诊所。
复苏舱。
生物档案库。
```

### 2. 微型无人机

```text
用于节点侦查、追踪、暗杀、投送和监控的微型飞行装备。
不同型号负责不同任务。
```

型号分工：

```text
蜂鸟：高速攻击 / 暗杀 / 生化载荷投送。
蜻蜓无人机：低噪巡查 / 追踪 / 悬停监视。
蝶翼侦察机：无感潜入 / 静音侦查 / 环境伪装。
```

### 3. 生化载荷封装

```text
将细菌、病毒、神经干扰剂或其他生化载荷压缩封装进微型设备中。
这些载荷可用于暗杀、驱逐、污染节点或制造战术威慑。
```

注意：

```text
当前标准版规则尚未引入中毒、感染、持续伤害、污染等复杂状态。
因此生化载荷优先作为世界观、插画和卡面风味信息使用。
如后续加入生化状态，应单独建立规则变体或扩展规则。
```

### 4. 蜂鸟巢指挥网络

```text
蜂鸟巢是纳米科技公司用于管理蜂鸟无人机群的节点级指挥设施。
它不是普通无人机停机坪，而是具备权限调度、战术回收、载荷重组和群体召回能力的纳米科技节点。
```

功能定位：

```text
召回分散蜂鸟。
集中终局战力。
作为10费用锚点。
加快游戏收束。
帮助玩家完成胜利条件。
超过节点上限的蜂鸟回到蜂鸟巢控制者手牌。
```

## 五、蜂鸟项目

```text
蜂鸟是纳米科技公司研发的高速微型攻击无人机。
它不是普通侦查无人机，而是暗杀型节点装备。
```

核心特征：

```text
体型极小。
速度极快。
攻击力不输狙击步枪。
可进行节点突袭。
可执行定点清除。
部分型号可装载细菌、病毒等生化载荷。
```

标准版卡表口径：

```text
卡名：蜂鸟。
英文名：Hummingbird。
类型：装备。
子类型：无人机。
复制数：4张。
定位：高速微型攻击无人机 / 暗杀型节点装备。
ATK：3。
HP：1。
规则技能：无。
技能描述位置：用于简短世界观介绍。
```

技能描述位置建议文案：

```text
纳米科技公司高速暗杀无人机，可执行节点突袭与生化载荷投送。
```

## 七、蜂鸟巢节点

```text
蜂鸟巢是纳米科技公司管理蜂鸟无人机群的节点设施。
它可以在终局阶段发出紧急召回指令，将分散在不同节点的蜂鸟重新聚集到蜂鸟巢。
若蜂鸟巢无法容纳全部蜂鸟，超出上限的蜂鸟会回到蜂鸟巢控制者手牌。
```

标准版候选口径：

```text
节点名：蜂鸟巢。
英文名：Hummingbird Hive。
系统域：纳米科技。
节点技能名：紧急召回。
节点技能费用：10。
节点技能：将场上所有“蜂鸟”移动到蜂鸟巢。若蜂鸟巢超过实体上限，超出的“蜂鸟”返回蜂鸟巢控制者手牌。
```

设计作用：

```text
作为10费用锚点。
用于终局阶段集中蜂鸟战力。
加快游戏收束。
帮助玩家完成胜利条件。
```

规则边界：

```text
紧急召回不直接宣告胜利。
技能结算后仍需按标准 Root Access 胜利条件检查。
蜂鸟巢最多保留3个实体。
若召回后超过实体上限，由发动技能的玩家选择保留在蜂鸟巢的蜂鸟。
超出的蜂鸟返回蜂鸟巢控制者手牌。
```

## 十、规则边界

```text
纳米科技公司设定不自动新增规则字段。
公司 Logo 不作为规则图标。
生化载荷不自动等于中毒、感染或持续伤害。
蜂鸟当前规则技能为空。
蜂鸟的技能描述位置只写世界观介绍，不写规则效果。
蜂鸟巢的紧急召回是节点技能，不是蜂鸟自身技能。
紧急召回不直接宣告胜利，仍需按 Root Access 胜利条件检查。
超出蜂鸟巢实体上限的蜂鸟返回蜂鸟巢控制者手牌。
```

如后续开发生化机制，应单独定义：

```text
中毒。
感染。
污染节点。
生化载荷。
持续伤害。
免疫或清除。
```

在当前标准版中，以上概念只作为设定储备，不进入默认规则。

## 十一、卡牌应用原则

```text
带有纳米科技公司来源的卡牌，可以在视觉上出现企业 Logo、资产编号、纳米网格或医疗 / 生化科技元素。
但卡面不能堆叠大段企业说明。
```

卡面文案应保持短句：

```text
纳米科技公司高速暗杀无人机，可执行节点突袭与生化载荷投送。
```

蜂鸟巢节点技能建议短句：

```text
紧急召回：将场上所有“蜂鸟”移动到蜂鸟巢。超过上限的返回蜂鸟巢控制者手牌。
```

避免写成：

```text
纳米科技公司研发的第X代微型高机动攻击侦查复合无人机，拥有多种复杂战术模式……
```
