## 一、正式名称

```text
中文名：维相科技
英文名：DIMENSIONAL PHASE TECHNOLOGIES
简称：DPT
```

名称含义：

```text
“维”代表空间维度、二维压缩与高维展开。
“相”代表物质形态、空间状态与实体相位。
```

维相科技不是普通数据存储公司，而是掌握实体在不同空间维度之间转换的核心企业。

---

## 二、公司定位

```text
维相科技是城市中最早掌握空间维度工程的超大型技术企业。
它通过改变实体占据空间的方式，将三维实体降维封存为二维权限载体，并在获得合法调用权限后重新展开。
```

主要技术：

```text
三维实体二维化。
空间折叠与展开。
实体相位锁定。
维度封存。
空间容积重构。
远距离实体调用。
物质所有权与权限绑定。
```

---

## 三、核心科技：维相转换技术

```text
中文名：维相转换技术
英文名：Dimensional Phase Conversion
```

标准流程：

```text
1. 降维：将三维实体压缩为稳定的二维封存状态。
2. 锁相：固定实体结构、身份信息和调用权限。
3. 展开：在指定空间中恢复实体的三维形态。
```

技术边界：

```text
不是缩小。
不是数据上传。
不是复制。
不是时间旅行。
同一实体在同一时刻只能保留一个有效维相状态。
```

## 四、正版储存卡

卡牌名称：

```text
正版储存卡
```

英文建议：

```text
Licensed Storage Card
```

产品性质：

```text
维相科技合法生产并通过城市权限系统认证的通用维度封存介质。
```

正常版本必须验证：

```text
目标封存许可。
目标所有权。
目标生物身份。
使用者调用权限。
展开区域安全等级。
```

对应技能：

```text
二维压缩
选择场上1个游侠或装备，将其返回其控制者手牌。
```

世界观解释：

```text
正版储存卡只能把目标压缩回其合法权限持有者的二维封存序列。
它不能夺取目标所有权，也不能把目标加入其他玩家手牌。
```

## 五、破解储存卡

卡牌名称：

```text
破解储存卡
```

英文建议：

```text
Cracked Storage Card
```

内部来源：

```text
以维相科技正版储存介质为基础，破解所有权验证、目标白名单和调用权限校验。
```

公司内部原型型号：

```text
DPT-P0
BLANK DIMENSIONAL MEDIUM
```

对应技能：

```text
越权压缩
选择场上1个游侠或装备，将其加入你的手牌。
```

世界观解释：

```text
目标被强制降维后，调用权限被重绑到发动者名下。
目标原控制者失去该实体，发动者获得后续重新展开权。
```

规则边界：

```text
不是传送。
不是复制。
不是消灭或销毁。
目标以二维封存体状态进入发动者手牌。
```

## 六、两种储存卡的区别

```text
正版储存卡：合法压缩，返回原控制者手牌。
破解储存卡：越权压缩，加入发动者手牌。
```

核心差异：

```text
二维压缩只改变实体状态与所在区域，不改变合法所有权。
越权压缩同时改写调用权限，因此构成夺取。
```



```text

不得表现为普通黑客代码界面。
```
