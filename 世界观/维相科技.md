## 三、核心技术：维相转换

中文技术名：

```text
维相转换技术
```

英文技术名：

```text
Dimensional Phase Conversion
```

技术定义：

```text
维相转换不是将实体缩小，也不是将实体复制为数据。
它通过改变实体的空间维度状态，把三维实体压缩为稳定的二维封存体。
封存体仍保存原实体的结构、身份、状态与调用权限。
获得合法权限后，可以在指定空间中重新展开为三维实体。
```

标准流程：

```text
1. 降维：将三维实体转换为二维封存状态。
2. 锁相：固定实体结构、身份信息与调用权限。
3. 封存：把二维封存体写入权限载体。
4. 展开：在合法空间内恢复实体的三维形态。
```

技术边界：

```text
维相科技掌握的是空间维度转换，不是时间旅行。
维相转换不创造新的实体，也不复制目标。
同一实体在同一时刻只能保留一个有效维相状态。
未经授权的强制封存属于严重权限犯罪。
```

```text

```

暂不自动归入维相科技的内容：

```text
蜂鸟、蜻蜓无人机等微型无人机，仍以纳米科技公司为主要制造来源。
普通武器、回收装备和归零者改装物，不因使用二维卡牌载体而自动视为维相科技产品。
节点卡本身是游戏中的权限载体表现，不代表所有节点都由维相科技制造。
```

技术授权关系：

```text
其他制造商可以使用维相科技授权的封存接口，把自己的产品压缩为二维权限载体。
因此“卡牌是二维封存体”不等于“所有卡牌都由维相科技制造”。
维相科技提供的是底层维度封存标准，不必成为所有物品的品牌来源。
```

## 八、与纳米科技公司的边界

```text
维相科技负责空间维度、实体降维、维相锁定和展开接口。
纳米科技公司负责纳米医疗、生化封装、微型无人机和相关战术产品。
```

合作关系：

```text
纳米科技公司的蜂鸟等产品，可以使用维相科技授权的二维封存标准进入手牌牌库和战术调用系统。
维相科技不负责蜂鸟的武器系统、生化载荷或无人机制造。
纳米科技公司不掌握维相科技完整的城市级空间维度基础设施。
```

冲突边界：

```text
不得将空白储存卡写成纳米科技公司的普通存储产品。
不得将维相科技写成制造所有卡牌与所有装备的万能企业。
不得把二维压缩解释成数据上传、意识上传或普通数字化。
```
