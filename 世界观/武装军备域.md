# 归零者｜武装军备域

> 文件定位：本文件记录《归零者 / ZERO ACCESS》中“武装军备”系统域、核心节点“指挥中心”及其模式规则边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`
>
> 企业来源：先锋军械 / VANGUARD ARMAMENTS（VA）。

---

## 一、系统域定位

```text
系统域名称：武装军备。
英文名：Armaments Domain。
主要企业来源：先锋军械 / VA。
当前核心节点：指挥中心 / Command Center。
```

武装军备域负责城市武器、装甲、节点装备、火控权限与战时军备的登记、调度和紧急征用。

先锋军械是该域的重要装备制造商和运营承包商，但不等同于整个系统域。

---

## 二、核心节点：指挥中心

```text
节点名：指挥中心。
英文名：Command Center。
系统域：武装军备。
唯一：是。
复制数：1。
节点技能名：紧急征用。
节点技能费用：10电荷。
```

节点技能：

```text
选择另一座节点。将其中所有独立部署的装备移动到指挥中心，并获得其控制权。超过实体上限的装备返回你的手牌。
```

---

## 三、规则口径

```text
“所有装备”仅指目标节点内独立部署的装备。
游侠携带的装备不受紧急征用影响。
被移动的装备进入指挥中心后，由技能发动者获得控制权。
指挥中心最多保留3个部署实体。
若超过实体上限，由技能发动者选择保留在指挥中心的装备。
超出的装备返回技能发动者手牌。
```

节点控制结算：

```text
目标节点失去装备后，立即重新检查节点控制权。
若目标节点没有任何有效实体，该节点变为无主。
若目标节点仍有游侠或其他有效实体，则按标准控制规则判断归属。
紧急征用本身不直接宣告胜利。
技能结算完成后，再检查最高权限胜利条件。
```

---

## 四、模式规则

```text
所有权限节点统一用于2至4人的混战模式。
所有权限节点均不用于2人的对战模式。
指挥中心不设置额外的混牌限制、所有权记录或归还规则。
```

---

## 五、设计作用

```text
作为10费用终局锚点。
集中另一节点的独立部署装备。
改变装备控制权。
使失去全部实体的目标节点变为无主。
制造节点重新争夺窗口。
与蜂鸟巢、维相枢纽形成同级的全场格局改变能力。
```
