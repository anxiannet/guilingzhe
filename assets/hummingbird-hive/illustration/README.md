# 蜂鸟巢插图目录

本目录用于保存 `蜂鸟巢 / Hummingbird Hive` 的无文字、无卡框插图候选。

---

## 当前用途

```text
存放后续用于样卡合成的插图资产。
插画层只表现被封存的蜂鸟巢节点设施本体。
插画层不承担规则阅读、系统域识别、封存卡号识别和品牌标识。
```

---

## 插画层不包含

```text
卡名
技能文字
系统域标签
封存卡号
节点位址
归零者 / ZERO ACCESS Logo
制作版本号
```

说明：

```text
系统域标签、封存卡号、节点位址属于完整卡牌层的世界观内正式信息。
这些内容必须在设计工具中人工排版，不能由 AI 插画生成。
```
