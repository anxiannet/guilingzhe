## 资产处理原则

```text
1. 历史成品卡只作为参考，不直接作为最终印刷文件。
2. 卡框、插图、文字排版分层处理。
3. 技能文案、编号、区域信息和 Logo 必须人工排版。
4. 不再直接生成完整成卡来赌文字细节。
5. 先整理历史成品卡，再决定是否重新提取卡框、重做插图或重新合成样卡。
```

```text

7. 导出中文主版样卡。
```
