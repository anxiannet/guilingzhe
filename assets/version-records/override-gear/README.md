# 越权模块｜版本记录索引

本目录用于记录 `归零者 / ZERO ACCESS` 中所有唯一越权模块卡的版本记录。

---

## 原则

```text
每张唯一越权模块卡只有一个卡号。
重复复制不新增卡号。
版本记录归属到唯一卡号下面。
版本号不拼进卡号。
```

---

## 术语范围

```text
越权模块 / Override Gear，是所有非游侠卡的统称。
```

越权模块包括但不限于：

```text
装备
设备
工具
武器
权限卡
纳米机器人
电磁手雷
程序芯片
数据钥匙
临时权限载体
防御工事
侦察无人机
一次性越权物品
```

说明：

```text
越权模块不是狭义电子模块。
装备、设备、工具、武器、权限卡、纳米机器人、电磁手雷等非游侠物品，都归入越权模块。
```
