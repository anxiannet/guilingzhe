# 物品卡｜版本记录索引

本目录用于记录 `归零者 / ZERO ACCESS` 中所有唯一物品卡的版本记录。

---

## 原则

```text
每张唯一物品卡只有一个卡号。
重复复制不新增卡号。
版本记录归属到唯一卡号下面。
版本号不拼进卡号。
```

---

## 术语范围

```text
物品卡 / Gear，是所有非游侠手牌卡的统称。
```

物品卡包括但不限于：

```text
装备
设备
工具
武器
权限卡
纳米机器人
电磁手雷
程序芯片
数据钥匙
临时权限载体
防御工事
侦察无人机
一次性物品
```

说明：

```text
旧称“越权模块”废止，不再作为卡面主类型。
装备、设备、工具、武器、权限卡、纳米机器人、电磁手雷等非游侠手牌卡，都归入物品卡。
```
