# 游侠｜版本记录索引

本目录用于记录 `归零者 / ZERO ACCESS` 中所有唯一游侠卡的版本记录。

目录名 `operators/` 暂时保留，表示英文侧 `Operative` 资产记录目录；中文正式术语以“游侠”为准。

---

## 原则

```text
每张唯一游侠卡只有一个卡号。
重复复制不新增卡号。
版本记录归属到唯一卡号下面。
版本号不拼进卡号。
游侠中文名称按技能命名。
游侠中文名称不参考英文名、不参考星级、不参考装备等级。
```

---

## 当前编号范围

```text
R-001 至 R-020：标准版唯一游侠。
R-021 至 R-023：扩展包新增唯一游侠。
```
