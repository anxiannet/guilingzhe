# 执行者｜版本记录索引

本目录用于记录 `归零者 / ZERO ACCESS` 中所有唯一执行者卡的版本记录。

---

## 原则

```text
每张唯一执行者卡只有一个卡号。
重复复制不新增卡号。
版本记录归属到唯一卡号下面。
版本号不拼进卡号。
```

---

## 当前编号范围

```text
R-001 至 R-020：标准版唯一执行者。
R-021 至 R-023：扩展包新增唯一执行者。
```

---

## 当前重点记录


```text
完整卡名、复制数、星级和技能映射见 12-公共牌编号与版本记录.md。
```
