# 归零者｜卡背设计规范

> 文件定位：本文件维护《归零者 / ZERO ACCESS》全部卡牌背面的统一世界观解释、品牌标识要求、牌库区分方式与视觉边界。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`
>
> 说明：卡背上的维相科技标识表示卡牌使用维相转换与二维封存标准，不代表卡牌正面的游侠、物品或节点均由维相科技制造。

---

## 一、当前结论

```text
所有正式卡牌背面必须带有维相科技（DPT）标识。
```

适用范围：

```text
游侠卡背。
物品卡背。
节点卡背。
标准版卡牌。
并入标准版的扩展包卡牌。
后续采用同一二维封存载体体系的扩展卡牌。
```

统一原则：

```text
所有卡牌都是使用维相科技维相转换标准形成的二维权限载体。
卡背上的 DPT 标识是封存载体标准标识。
卡牌正面的公司 Logo、型号或来源信息是内容制造来源标识。
两者不得混淆。
```

---

## 二、世界观解释

```text
维相科技为城市权限系统提供维相转换、实体降维、二维封存、相位锁定和重新展开的底层技术标准。
游侠、物品与节点信息可以通过这一标准封存在二维权限载体中。
调用一张卡，表示释放、展开或调用其中封存的战术实体或系统权限。
```

因此：

```text
蜂鸟正面仍可标记纳米科技公司，表示蜂鸟的产品制造来源。
蜂鸟背面仍必须标记维相科技，表示该卡使用 DPT 二维封存标准。
空白储存卡正面和背面都可以出现维相科技标识，因为其产品本身也由维相科技制造。
游侠不是维相科技制造的人造产品，但游侠卡背仍使用 DPT 封存标准标识。
节点不属于维相科技私人所有，但节点卡背仍使用 DPT 权限载体标准标识。
```

禁止解释：

```text
不得声称所有游侠由维相科技制造。
不得声称所有装备由维相科技制造。
不得声称所有节点归维相科技所有。
不得将 DPT 卡背标识解释为卡牌正面内容的统一制造商。
不得将二维封存解释为普通数据上传或数字复制。
```

## 三、牌库区分

《归零者》至少存在两套需要从背面直接区分的牌库：

```text
手牌牌库：游侠与物品卡。
节点牌库：节点卡。
```

因此：

```text
所有卡背统一使用维相科技 DPT 品牌母设定。
手牌卡背与节点卡背必须能够从视觉上立即区分。
两种卡背可以使用不同构图、徽章、编号系统或空间结构。
两种卡背都必须保留同一套 DPT Logo 体系与维相转换视觉语言。
```

不得采用：

```text
游侠卡与物品卡使用不同背面，从而在手牌牌库中泄露卡牌类型。
节点卡背与手牌卡背完全相同，导致玩家无法快速区分牌库。
通过文字长句区分牌库。
仅依靠卡牌尺寸区分而不做背面视觉区分。
```

## 四、手牌卡背

适用卡牌：

```text
游侠。
装备。
一次性物品。
```

世界观定位：

```text
战术实体与物品的通用二维封存载体。
```

视觉重点：

```text
中央维相封存核心。
三维结构向二维平面压缩。
透明数据膜与空间折叠边缘。
冷白、青蓝与青紫维度光。
维相科技 DPT 标识。
归零者 / ZERO ACCESS 品牌标识。
```

建议小字：

```text
DPT
DIMENSIONAL PHASE
TACTICAL MEDIUM
```

限制：

```text
卡背不得显示游侠、装备或一次性物品类型。
卡背不得显示卡名、费用、ATK、HP 或技能信息。
不得通过不同颜色、不同 Logo 或不同纹理泄露正面卡牌类别。
```

## 五、节点卡背

适用卡牌：

```text
所有系统域节点卡。
```

世界观定位：

```text
城市节点权限信息与基础设施调用权的二维权限载体。
```

视觉重点：

```text
城市空间网格。
多个透明平面交叠形成的系统坐标。
节点权限核心。
维相科技 DPT 标识。
归零者 / ZERO ACCESS 品牌标识。
与手牌卡背明显不同的横向构图。
```

建议小字：

```text
DPT
DIMENSIONAL PHASE
NODE ACCESS MEDIUM
```

限制：

```text
卡背不得显示具体系统域。
卡背不得显示节点名称、节点技能或节点编号。
不同系统域的节点不得使用不同卡背，以免泄露节点牌库信息。
```

## 六、DPT 标识规范

公司中文名：

```text
维相科技
```

英文名：

```text
Dimensional Phase Technologies
```

简称：

```text
DPT
```

Logo方向：

```text
一个由三维立方体逐渐压缩成二维平面的简洁几何符号。
一侧保留空间体积，另一侧收束为单一薄面。
标识必须适合在卡背中央、底边铭牌或安全认证区使用。
```

Logo层级：

```text
DPT 标识是卡背世界观与技术标准识别元素。
归零者 / ZERO ACCESS 是产品品牌标识。
两者可以同时出现，但不得互相取代。
DPT 不得大到让卡背看起来像企业广告。
```

## 七、统一视觉边界

必须体现：

```text
高科技二维封存载体。
空间折叠。
实体降维。
透明数据膜。
精密工业标准。
冷白、青蓝或青紫 Root Access 光。
收藏卡质感。
```

禁止方向：

```text
普通企业名片。
现实银行卡或信用卡。
SD卡、U盘或普通存储芯片。
普通科幻门禁卡。
霓虹黑客界面。
宇宙星空背景。
魔法阵或玄幻传送门。
大面积说明文字。
教学说明页。
复杂花纹覆盖核心空间结构。
```

## 八、印刷与保密要求

```text
同一牌库中的卡牌背面必须完全一致。
手牌牌库中的游侠与所有物品卡不得出现可识别差异。
节点牌库中的不同系统域和不同节点不得出现可识别差异。
扩展卡并入标准版牌库时，其背面颜色、亮度、裁切、安全边距和 Logo 位置必须与对应牌库保持一致。
不得因印刷批次、扩展包或稀有度差异造成背面可识别。
```

建议：

```text
标准版和扩展包使用同一份卡背母版文件。
如需调整印刷参数，只允许做不影响肉眼识别的一致性校正。
```

---

## 九、当前执行口径

```text
所有卡牌背面统一带有维相科技 DPT 标识。
手牌卡背与节点卡背分别设计。
同一牌库内部不得通过卡背泄露卡牌信息。
DPT 标识表示二维封存技术标准，不表示正面内容统一由维相科技制造。
卡牌正面的制造商标识继续按具体卡牌来源单独处理。
```
