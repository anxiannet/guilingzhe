# 归零者｜游侠小传与插画参考

> 文件定位：本文件用于记录 23 名唯一游侠的性别、年龄、性格、习惯、背景故事与插画参考。
>
> 本文件服务于插画提示词、样卡制作与角色视觉审核，不直接进入卡面文字。

---

## 一、使用原则

游侠小传用于帮助插画建立人物感。

```text
卡名负责代号感。
技能负责动作。
当前星级负责装备层级。
小传负责脸、性格、姿态、道具和生活痕迹。
```

---

## 二、角色颜值与视觉吸引力标准

游侠人物可以颜值在线，但必须符合《归零者》的世界观气质。

```text
颜值在线 = 脸部有吸引力、辨识度强、精神状态有戏、适合收藏卡。
```

允许：

```text
五官清晰。
脸部轮廓有记忆点。
眼神有故事。
发型、伤痕、义眼、纹身、工具痕迹能强化代号。
人物看起来有魅力、有戏、有收藏价值。
不同年龄、性别和气质都可以有吸引力。
```

禁止：

```text
不要网红脸。
不要同质化帅哥美女。
不要擦边。
不要性感化。
不要低俗服装设计。
不要为了颜值削弱废城边缘生存感。
不要把所有人画成年轻偶像。
不要用单一审美标准统一所有游侠。
```

审核口径：

```text
第一眼要好看。
第二眼要能记住是谁。
第三眼要能看出他 / 她 / 他们为什么会变成这个代号。
```

## 三、世界观参考

游侠是被城市权限系统删除后，在城市权限边缘行动的人。

他们不是正规军，不是官方探员，也不是普通雇佣兵。

他们靠捡来的装备、抢来的武器、拼出来的护甲、改出来的模块和越权接入的系统资源活下去。

插画应体现：

```text
被系统删除后的生存痕迹。
城市权限边缘的行动经验。
代号带来的脸部识别点。
当前星级对应的装备状态。
高科技二维封存体的卡内压缩空间感。
```
