# 归零者｜游侠基础技能池与权限驱逐定义

> 文件定位：本文件用于记录《归零者 / ZERO ACCESS》当前通用游侠基础技能池，以及“权限驱逐 / 返回手牌”类效果的正式规则定义。
>
> 当前唯一主仓库：`anxiannet/guilingzhe`

---

## 一、设计目标

当前游侠技能池用于 2 人对战与 4 人共用牌库模式。

游侠技能不应只服务双人对战，也不应依赖节点卡才能成立。

本文件只记录基础游侠技能池，不替代：

```text
03-规则变体.md
04-卡表.md
19-技能自动测试规范.md
20-技能载体审核与世界观解释.md
21-技能规则意义审核.md
```

---

## 二、基础游侠技能筛选标准

游侠基础技能必须满足：

```text
1. 适用于 2 人对战与 4 人共用牌库模式。
2. 不依赖节点卡。
3. 不直接提及节点牌库。
4. 不与 12 座节点技能重复或类似。
5. 单一基本动作。
6. 同类技能只保留 1 个代表。
7. 不包含复杂条件限制。
8. 不包含连锁触发。
9. 不包含跨回合长期状态记录。
10. 不把可以通过费用、攻击力、生命值直接调整解决的内容写成技能。
```

## 三、当前排除方向

### 1. 排除节点相关技能

```text
查看节点信息。
使节点在线。
使节点离线。
保护节点。
移动游侠到节点。
移动装备到节点。
接管节点相关额外动作。
```

### 2. 排除与节点技能重复的技能

```text
选择1个己方游侠，本回合攻击力+1。
选择1个己方游侠，本回合生命值+1。
从弃牌区将1张游侠加入手牌。
选择1个己方装备，本回合攻击力+1。
选择1个己方装备，本回合生命值+1。
从弃牌区将1张装备加入手牌。
```

### 3. 排除可由数值直接替代的技能

```text
此游侠攻击力+1。
此游侠生命值+1。
获得1电荷。
```

说明：

```text
如果某张游侠打出后只是获得1电荷，本质等同于降低该游侠实际费用，应通过修改费用解决。
如果某张游侠只是给自己加攻或加生命，应通过修改该游侠自身 ATK / HP 解决。
```

### 4. 排除组合动作

```text
抽1张牌，然后弃1张牌。
弃1张牌，抽2张牌。
查看牌库顶后置底。
伤害后抽牌。
返回手牌后获得电荷。
```

说明：

```text
这些不是单一基本动作，暂不进入基础游侠技能池。
```

## 四、按作用对象整理的基础游侠技能池

### 作用对象：手牌牌库

| 技能作用方式 | 技能 | 保留状态 | 说明 |
|---|---|---|---|
| 抽取 | 抽1张牌。 | 主池 | 从手牌牌库抽取1张牌，加入当前玩家手牌。 |
| 查看 | 查看手牌牌库顶1张。 | 主池 | 查看手牌牌库顶端信息，不改变牌库顺序。 |
| 重洗 | 重洗手牌牌库。 | 备选 | 单一动作，但体感较弱，暂不进主池。 |
| 置底 | 将手牌牌库顶1张置底。 | 备选 | 单一动作，但通常需要配合查看才有价值，暂不进主池。 |

### 作用对象：对手手牌

| 技能作用方式 | 技能 | 保留状态 | 说明 |
|---|---|---|---|
| 查看 | 查看1名对手的手牌。 | 主池 | 获取1名对手的隐藏手牌信息。 |
| 弃置 | 选择1名对手，其弃1张牌。 | 主池 | 由目标对手选择并弃置其1张手牌。 |
| 公开 | 选择1名对手，其公开1张手牌。 | 备选 | 单一动作，但信息强度较弱，暂不进主池。 |
| 回牌库 | 选择1名对手，其将1张手牌放到手牌牌库底。 | 备选 | 单一动作，但比弃置更难评估强度，暂不进主池。 |

### 作用对象：对手场上游侠

| 技能作用方式 | 技能 | 保留状态 | 说明 |
|---|---|---|---|
| 伤害 | 对1个对手场上游侠造成1点伤害。 | 主池 | 影响对手游侠生命值与场上存活。 |
| 降攻 | 选择1个对手场上游侠，本回合攻击力-1。 | 主池 | 本回合内降低目标游侠攻击表现。 |
| 禁攻 | 选择1个对手场上游侠，本回合不能攻击。 | 主池 | 本回合内限制目标游侠攻击动作。 |
| 返回手牌 | 将1个对手场上游侠返回其控制者手牌。 | 主池 | 使目标游侠从当前场上部署状态退出，并返回其控制者手牌。 |
| 技能失效 | 选择1个对手场上游侠，本回合技能失效。 | 备选 | 需要先定义“技能失效”边界，暂不进主池。 |
| 处理 | 处理1个对手场上游侠。 | 不建议 | 过强，不作为基础游侠技能。 |

## 六、“权限驱逐 / 返回手牌”定义

### 术语分层

```text
规则俗称：回手。
内部主题名：权限驱逐。
卡面规则文字：返回手牌。
```

卡面不写“权限驱逐”。

卡面不写“吹回”。

卡面统一写成：

```text
将1个目标返回其控制者手牌。
```

### 定义

```text
返回手牌，是指将1个场上的可部署实体从当前场上部署状态移回其控制者手牌。
```

权限驱逐是“返回手牌”类技能的世界观主题名。

### 可被返回手牌的对象

当前可被返回手牌的对象：

```text
场上游侠。
场上装备。
```

不可被返回手牌的对象：

```text
节点。
一次性物品。
手牌中的牌。
牌库中的牌。
弃牌区中的牌。
已经被处理、隔离或不在场上的牌。
```

### 区域变化

```text
zone_from = battlefield / field / node_slot
zone_to = controller_hand
```

说明：

```text
虽然可部署实体在 4 人模式中会部署在节点上，但“返回手牌”不以节点为技能作用对象。
返回手牌的作用对象是场上的可部署实体。
节点只是该实体当前所在的位置。
```

### 控制者与拥有者

正式口径：

```text
返回其控制者手牌。
```

当前默认规则：

```text
控制者 = 该牌当前控制者。
```

如未来出现“临时夺取控制权”的效果，需要额外定义：

```text
返回当前控制者手牌，还是原拥有者手牌。
```

在当前首版规则中，暂不加入临时夺取控制权，因此默认写作“返回其控制者手牌”。

### 与处理的区别

```text
处理：目标离开场上并进入弃牌区。
返回手牌：目标离开场上并进入其控制者手牌。
```

返回手牌不是伤害，不触发“受到伤害”相关判断。

返回手牌不是处理，不触发“被处理”相关判断。

返回手牌不是移动，不等同于“移动到节点”。

### 状态清除

返回手牌的可部署实体离开场上时，清除其本回合临时状态。

包括但不限于：

```text
本回合攻击力-1。
本回合不能攻击。
本回合生命值变化。
本回合不能参与某动作。
```

说明：

```text
这些状态不带回手牌。
重新打出时，按新实体进入场上处理。
```

### 费用与重新打出

```text
返回手牌后的牌，之后重新打出时，必须重新支付其打出费用。
```

返回手牌本身不返还已支付费用。

### 合法目标

返回手牌技能发动时，必须选择合法目标。

合法目标必须满足：

```text
1. 是场上的可部署实体。
2. 符合技能指定归属，如“对手场上游侠”“己方场上装备”。
3. 当前没有被规则标记为不能返回手牌。
```

如果没有合法目标，该技能不能发动。

## 七、返回手牌标准写法

### 对手目标

```text
将1个对手场上游侠返回其控制者手牌。
将1个对手场上装备返回其控制者手牌。
```

### 己方目标

```text
将1个己方场上游侠返回你的手牌。
将1个己方场上装备返回你的手牌。
```

### 任意目标

```text
将1个场上游侠返回其控制者手牌。
将1个场上装备返回其控制者手牌。
```

首版建议优先使用明确归属：

```text
对手场上游侠。
对手场上装备。
己方场上游侠。
己方场上装备。
```

减少“任意目标”造成的自选收益和争议。

## 八、自动测试字段

返回手牌技能可转为以下测试字段：

```text
actor_player_id
source_card_id
target_card_id
target_card_type = operative / equipment
target_controller_id
zone_from = field
zone_to = target_controller_hand
clear_temporary_status = true
cost_refund = false
trigger_destroyed_effect = false
trigger_damage_effect = false
```

测试点：

```text
1. 目标离开场上。
2. 目标进入其控制者手牌。
3. 目标不进入弃牌区。
4. 目标临时状态被清除。
5. 不触发受到伤害效果。
6. 不触发被处理效果。
7. 重新打出时需要重新支付费用。
```

## 九、命名与世界观解释

“吹回”本身是规则俗称，不适合直接作为《归零者》的世界观词。

在《归零者》中，“返回手牌”类效果可以解释为：

```text
权限驱逐。
强制退场。
连接断开。
节点排斥。
```

也就是：

```text
目标不是被物理吹飞，而是被节点系统从当前接入点踢出，回到控制者手牌，代表撤回到待部署状态。
```

推荐世界观解释：

```text
节点释放一次权限驱逐，把目标实体从当前节点的接入层踢出。
该实体没有死亡，也没有被处理，而是连接中断，退回控制者手牌，等待重新部署。
```

规则意义：

```text
移除当前节点上的压力。
打断对方接管布局。
让目标控制者需要重新支付费用部署。
```

游侠使用“返回手牌”类技能时，世界观解释应偏向：

```text
近战驱离。
现场撤人。
掩护撤回。
人力干扰接入。
威慑目标退出接入。
```

物品卡使用“返回手牌”类技能时，世界观解释可偏向：

```text
牵引弹。
回滚扇区。
弹射装置。
断开接口。
权限驱逐。
```

节点技能当前不使用“返回手牌”作为固定 12 节点技能。
