# 归零者｜游侠攻击力与装备锚点设定

> 文件定位：本文件记录 `归零者 / ZERO ACCESS` 中人物牌类型“游侠”的攻击力视觉锚点、装备来源、生命值 / 防御视觉锚点与插画审核口径。
>
> 本文件用于统一世界观、卡表命名、插画提示词和样卡审核口径。
>
> 说明：本文件不新增规则字段。具体费用、ATK、HP 与技能，以当前卡表为准。

---

## 一、当前结论

```text
人物牌中文类型：游侠。
英文类型：Operative。
旧称“执行者”废弃。
“队员”不作为正式术语。
游侠没有星级规则字段。
旧星级体系保留为 ATK 的视觉锚点，不作为费用、等级、稀有度或技能目标条件。
卡牌总价值主要由费用体现，也就是打出或发动时消耗的权限电荷数。
```

术语关系：

```text
归零者 / ZERO ACCESS：游戏名、阵营名、大身份。
游侠 / Operative：人物牌类型、行动个体。
物品卡 / Gear：所有非游侠手牌卡的统称。
装备：可以作为可部署实体进入节点的物品卡。
一次性物品：只结算效果，不留在节点上的物品卡。
节点：争夺地点。
系统域 / System Domain：节点区域。
Root Access：胜利目标。
```

说明：

```text
旧称“越权模块 / Override Gear”不再作为卡面主类型。
物品卡可以包括装备、设备、工具、武器、权限卡、纳米机器人、电磁手雷、程序芯片和一次性越权物品。
```

## 二、游侠定义

```text
游侠，是被城市权限系统删除后，脱离官方身份，在城市权限边缘行动的人。
他们不是正规军，不是官方探员，也不是普通雇佣兵。
他们靠捡、抢、改、拼装备逐步武装自己，并通过越权接入、潜入、攻击节点、占领节点和小队协作争夺系统权限。
```

一句话定义：

```text
游侠是被系统删除后，靠捡来的装备和抢来的权限活下去的人。
```

---

## 三、装备来源原则

游侠的战力不同，不来自官方授权、军衔或固定职业，而来自：

```text
1. 捡来的装备。
2. 抢来的武器。
3. 拼出来的护甲。
4. 改出来的模块。
5. 越权接入的系统资源。
```

因此，游侠视觉不应做成正规军制式小队，也不应做成普通黑客或普通雇佣兵。

## 四、攻击力 ATK 视觉锚点

```text
ATK 是游侠或装备在攻击节点战斗中造成伤害的数值。
旧星级体系不再作为规则字段，而是降级为 ATK 的视觉锚点。
视觉锚点用于帮助美术判断：这个实体的攻击力应该表现出什么装备完整度、火力水平和战斗压迫感。
```

### ATK 0｜无攻击锚点

规则含义：

```text
不能主动攻击节点。
可以驻守节点。
防守时造成 0 点伤害。
```

视觉含义：

```text
侦察、支援、工具、无人设备、后勤设备。
几乎没有主动进攻能力。
有功能，但不负责突破节点。
```

适合表现：

```text
探针。
侦察无人机。
信标。
数据设备。
临时工具。
弱火力装备。
```

### ATK 1｜低攻击锚点

对应旧视觉层级：

```text
旧1星游侠感。
```

装备描述：

```text
只有匕首、短刀、撬棍、旧工具、临时护具。
几乎没有稳定火力。
依靠灵活、偷袭、近身、破门、拆锁、绕行行动。
```

战斗气质：

```text
能参与战斗，但不是主攻。
底层、灵活、危险但脆弱。
```

视觉关键词：

```text
匕首。
撬棍。
短刀。
旧工具。
轻便包。
厚外套。
拼接护具。
临时面罩。
```

### ATK 2｜标准攻击锚点

对应旧视觉层级：

```text
旧2星至3星游侠感。
```

装备描述：

```text
拥有手枪、自制枪械、土改武器，或残缺低配制式装备。
开始具备稳定火力，但还不是完整武装。
有局部护甲、旧制式背心、二手战术挂具。
```

战斗气质：

```text
具备明确进攻压力。
可以承担普通节点争夺。
像是已经从底层幸存者变成真正的战术行动单位。
```

视觉关键词：

```text
手枪。
短枪。
自制枪。
改装弹匣。
简易枪套。
破旧护甲片。
旧制式背心。
低配头盔。
二手战术挂具。
拼装通讯器。
```

### ATK 3｜强攻击锚点

对应旧视觉层级：

```text
旧4星至5星游侠感。
```

装备描述：

```text
拥有完整制式装备、稳定火力，甚至接近轻型外骨骼或机甲化装备。
不代表身份更高级，而是装备更完整、火力更稳定、接入能力更强。
```

战斗气质：

```text
小队核心战力。
正面突破能力强。
适合承担关键节点攻坚。
视觉上应明显比普通游侠更有压迫感。
```

视觉关键词：

```text
完整头盔。
防弹衣。
战术背心。
制式步枪。
充足弹匣。
战术护目镜。
外骨骼。
机械臂。
装甲脊柱。
能量背架。
Root Access 冷光。
```

## 五、生命值 HP / 防御视觉锚点

```text
HP 是游侠或装备在节点战斗中能承受多少伤害的数值。
“防御力”可以作为玩家理解词，但正式规则字段仍写 HP / 生命值。
HP 表示驻守能力、承伤能力和留场能力。
```

### HP 1｜脆弱

规则含义：

```text
受到 1 点伤害就被消灭。
```

视觉锚点：

```text
无护甲。
轻便衣物。
临时护具。
工具型设备。
```

设计感觉：

```text
容易被清除，但通常费用低、攻击高或技能直接。
```

### HP 2｜基础防御

规则含义：

```text
能承受 1 点伤害。
受到 2 点伤害会被消灭。
```

视觉锚点：

```text
厚外套。
拼接护具。
局部护甲。
简易战术背心。
```

设计感觉：

```text
普通耐久，可以站住一段时间，但不算硬。
```

### HP 3｜稳定驻守

规则含义：

```text
能承受 1 到 2 点伤害。
需要 3 点伤害才能被消灭。
```

视觉锚点：

```text
低配制式护甲。
战术背心。
护肩。
头盔。
强化护具。
```

设计感觉：

```text
能守节点，有明显留场能力。
```

### HP 4｜强防御

规则含义：

```text
普通 1 到 2 ATK 单位很难单独清掉。
需要强攻或多次战斗处理。
```

视觉锚点：

```text
完整防弹衣。
重型护甲。
盾牌。
强化外套。
节点守卫型装备。
```

设计感觉：

```text
主要价值是守点，不一定攻击高。
```

### HP 5｜重型防御 / 工事

规则含义：

```text
极难被单个普通单位清除。
需要高 ATK 或多实体围攻。
```

视觉锚点：

```text
重型装甲。
固定工事。
防御地堡。
外骨骼防护结构。
```

设计感觉：

```text
不是用来打人的，是用来占住节点的。
```

## 六、ATK / HP 组合锚点

### 1 / 1｜低费基础单位

```text
轻量、便宜、功能简单。
能打，但很脆。
适合侦察员、信使、扒手等低费游侠。
```

### 1 / 2｜辅助驻守单位

```text
攻击弱，但比 1 / 1 稍能站。
常见于信息、回手、辅助技能。
适合烟幕手、预读者、广播员、扳手、中间人等。
```

### 2 / 1｜快攻单位

```text
攻击高于生命。
能打掉目标，但自己容易死。
适合甩棍、快手、突击型游侠。
```

### 1 / 3｜防守 / 续航单位

```text
攻击弱，留场强。
适合守节点或承接被动技能。
适合不倒翁、防守型游侠。
```

### 3 / 2｜高压攻击单位

```text
能突破多数普通守点实体。
但自身不是特别硬。
适合清除者、爆破兵、强攻型游侠。
```

### 3 / 3｜核心战斗单位

```text
攻击强，生命也稳定。
费用应明显偏高。
适合推土机、清道夫、高费主战单位。
```

### 1 / 4｜重防守单位

```text
攻击弱，但极难清掉。
主要价值是守节点。
适合门神、防御型游侠。
```

### 1 / 5｜重型装备 / 工事

```text
几乎不是进攻牌。
核心价值是节点驻守和拖延。
适合回收地堡、重型防守装备。
```

## 七、费用、电荷与总价值锚点

```text
ATK 表示攻击压力。
HP 表示防御与留场能力。
费用表示卡牌总价值。
```

费用分层参考：

```text
1费：轻量单位，低 ATK / 低 HP，技能简单。
2费：标准低费牌，可以有基础技能，或中等身材。
3费：中费牌，可以有组合技能、较好留场，或特殊被动。
4费：强牌，要么攻击强，要么防御强，要么技能强。
5费：高价值核心牌，应具备强清除、万能解、强攻坚或关键节奏能力。
```

说明：

```text
高 ATK 不一定代表卡牌总价值高。
高 HP 不一定代表卡牌总价值高。
卡牌价值主要通过费用，也就是权限电荷消耗体现。
```

## 八、插画与卡面应用

```text
旧星级越高，不是人物更“高级职业”，而是装备越来越完整、火力越来越稳定、接入能力越来越强。
旧星级现在只服务 ATK 视觉锚点，不进入规则文本。
```

插画必须体现：

```text
ATK 0：没有主动攻击力，偏工具、侦察、后勤或无人设备。
ATK 1：活下去，低攻击，临时武装，近身工具。
ATK 2：有火力，标准攻击，自制枪械或低配战术装备。
ATK 3：强攻坚，完整武装，稳定火力或外骨骼压迫感。
HP 1：脆弱，几乎无护甲。
HP 2：基础防护，局部护具。
HP 3：稳定驻守，低配制式护甲。
HP 4：强防御，完整护甲或盾牌。
HP 5：重型防御，工事、地堡或外骨骼防护结构。
```

禁止方向：

```text
不要做成正规军晋升军衔。
不要做成普通雇佣兵枪战。
不要做成传统武侠游侠。
不要把游侠画成弓箭手职业。
不要让攻击力差异只靠颜色或星星表现。
不要把星级画成稀有度、等级或身份阶层。
```

## 九、规则文本使用

规则中推荐使用：

```text
打出1张游侠。
选择1名游侠。
该游侠参与攻击节点。
将1名游侠移到另一座节点。
选择1个 ATK 1 的游侠。
选择1个 ATK 3 以下的游侠。
打出1张物品卡。
选择1个可部署实体。
```

不再使用：

```text
执行者。
队员。
干员。
探员。
行动员。
1星游侠。
4星以下游侠。
处理1个1星游侠。
吹回1个4星以下游侠。
```

## 十、最终口径

```text
旧星级不作为规则字段，只作为 ATK 的视觉锚点。
生命值可以在玩家说明中理解为防御力，但正式字段仍写 HP / 生命值。
ATK 是主动突破能力。
HP 是防御与驻守能力。
费用是卡牌总价值。
```
