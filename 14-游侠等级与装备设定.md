# 归零者｜游侠等级与装备设定

> 文件定位：本文件记录 `归零者 / ZERO ACCESS` 中人物牌类型“游侠”的等级、装备来源与战力层级。
>
> 本文件用于统一世界观、卡表命名、插画提示词和样卡审核口径。

---

## 一、当前结论

```text
人物牌中文类型：游侠。
英文类型：Operative。
旧称“执行者”废弃。
“队员”不作为正式术语。
```

术语关系：

```text
归零者 / ZERO ACCESS：游戏名、阵营名、大身份。
游侠 / Operative：人物牌类型、行动个体。
越权模块 / Override Gear：所有非游侠卡的统称。
权限节点 / Access Node：争夺地点。
系统域 / System Domain：节点区域。
Root Access：胜利目标。
```

说明：

```text
越权模块不是狭义装备牌。
越权模块可以包括装备、设备、工具、武器、权限卡、纳米机器人、电磁手雷、程序芯片和一次性越权物品。
```

---

## 二、游侠定义

```text
游侠，是被城市权限系统删除后，脱离官方身份，在城市权限边缘行动的人。
他们不是正规军，不是官方探员，也不是普通雇佣兵。
他们靠捡、抢、改、拼装备逐步武装自己，并通过越权接入、潜入、接管和小队协作争夺系统权限。
```

一句话定义：

```text
游侠是被系统删除后，靠捡来的装备和抢来的权限活下去的人。
```

---

## 三、装备来源原则

游侠的战力不同，不来自官方授权、军衔或固定职业，而来自：

```text
1. 捡来的装备。
2. 抢来的武器。
3. 拼出来的护甲。
4. 改出来的模块。
5. 越权接入的系统资源。
```

因此，游侠视觉不应做成正规军制式小队，也不应做成普通黑客或普通雇佣兵。

## 四、1星至5星装备层级

### 1星游侠

```text
装备水平：只有匕首类冷兵器和简易工具。
防护水平：几乎无护甲，最多是拼接护具或厚外套。
战斗气质：底层、灵活、危险但脆弱。
视觉关键词：匕首、撬棍、短刀、旧工具、轻便包、临时护具。
```

### 2星游侠

```text
装备水平：拥有手枪、自制枪械或土改武器。
防护水平：低质护具，可能有局部防护。
战斗气质：开始具备火力，但弹药和稳定性不足。
视觉关键词：手枪、短枪、自制枪、改装弹匣、简易枪套、破旧护甲片。
```

### 3星游侠

```text
装备水平：拥有残缺或低配制式装备。
防护水平：部分制式护甲，品质不高，弹药不足。
战斗气质：已接近专业行动人员，但资源仍紧张。
视觉关键词：旧制式背心、低配头盔、二手战术挂具、短缺弹药、拼装通讯器。
```

### 4星游侠

```text
装备水平：拥有全套制式装备。
防护水平：头盔、防弹衣、战术背心齐全。
战斗气质：装备精良，弹药充足，具备稳定接管战斗能力。
视觉关键词：完整头盔、全套防弹衣、制式步枪、充足弹匣、战术护目镜、整洁但非官方的装备改色。
```

### 5星游侠

```text
装备水平：拥有轻型动力机甲或高度集成的外骨骼装备。
防护水平：高防护、高机动、高火力。
战斗气质：小队核心战力，接近 Root 级行动单位。
视觉关键词：轻型动力机甲、外骨骼、装甲脊柱、机械臂、能量背架、Root Access 冷光。
```

## 五、插画与卡面应用

```text
星级越高，不是人物更“高级职业”，而是装备越来越完整、火力越来越稳定、接入能力越来越强。
```

插画必须体现：

```text
1星：活下去。
2星：有火力。
3星：拼出专业感。
4星：完整武装。
5星：轻型动力机甲。
```

禁止方向：

```text
不要做成正规军晋升军衔。
不要做成普通雇佣兵枪战。
不要做成传统武侠游侠。
不要把游侠画成弓箭手职业。
不要让等级差异只靠颜色或星星表现。
```

## 六、规则文本使用

规则中推荐使用：

```text
打出1张游侠。
选择1名游侠。
该游侠参与接管权限节点。
将1名游侠移到另一座权限节点。
处理1个1星游侠。
吹回1个4星以下游侠。
打出1张越权模块。
选择1个可部署实体。
```

不再使用：

```text
执行者
队员
干员
探员
行动员
```
